
一夜宿醉之后，隔天起来，王惊蛰捂着“嗡嗡”疼的脑袋，回忆了半天才想起来自己人在哪，旁边的床上林汶骐也强支起眼皮，龇牙咧嘴的说道：“昨天这酒喝的太多，太多了，离断片不远了”<br/>　　<br/>　　“我感觉差点都要喝出人命了，唉，昨天最后是什么节奏你还记得不了？”<br/>　　<br/>　　“好像，令歌想当你姐夫来着吧？”王惊蛰顿时无语，搓着疲惫的脸蛋，感慨着说道：“有王冬至这个姐，是好事也是个坏事啊，她没出嫁之前，我可能是各路人马都想要的小舅子，但你说万一她哪天要是嫁人了，那我岂不是一下子也把各路人马都给得罪了，现在是个香饽饽，以后可能就是条咸鱼了，人人嫌弃啊”<br/>　　<br/>　　“你要这么说的话，我倒是有个办法可行”<br/>　　<br/>　　“啥啊？”<br/>　　<br/>　　“让王冬至跟各路人马只谈暧昧，无关婚姻……”<br/>　　<br/>　　王惊蛰抓起枕头就砸了过去，笑骂道：“滚犊子，这不是卖姐求荣了么”<br/>　　<br/>　　两人起来洗漱完后，从房间里出来，那个经理许叔就已经等在外面了，然后告诉他俩早餐已经准备好了，不过王令歌临时有事早走了一步，外面有车可以把他们送回去。<br/>　　<br/>　　林汶骐两手一摊，说道：“羞愧了！”<br/>　　<br/>　　王惊蛰咬牙说道：“这一茬，我早晚会找回来的”<br/>　　<br/>　　回到陈田村，王惊蛰没等进家，徐秀芳和吴国栋就把他给拽到了外面，两人一脸担忧的问道：“惊蛰啊，我们听说，那栋楼的大老板前天好像死了，从楼上掉下来的？”<br/>　　<br/>　　徐秀芳和吴国栋稍微想想，就能明白刘运辉的死，肯定跟他们的遭遇还有王惊蛰的出现有关，他俩现在倒是不担心自己的工作了，怕的是王惊蛰出事。<br/>　　<br/>　　“叔，婶，你俩什么都不用担心，安心工作就是了”王惊蛰平淡的说道：“他是怎么死的，你们也不用管，我人不是好好的站在这里呢么？”<br/>　　<br/>　　“可是，这……”<br/>　　<br/>　　“他死不死的，要是和我有关，警察不早上门来了么？放心吧，没事的”王惊蛰打了个哈欠，摆手说道：“昨晚喝多了，我回去睡会，你俩该上班上班，该干嘛干嘛，就当什么也不知道就行了”<br/>　　<br/>　　王惊蛰扔下一句话就走了，吴国栋和许秀琴对视一眼，吴国栋沉默半晌说道：“惊蛰这孩子，远没有我们看到的那么简单啊”<br/>　　<br/>　　一晃过去两三天。<br/>　　<br/>　　那天早上因为羞愧离去的王令歌给王惊蛰打了个电话，跟他说后天带他从海珠过去大澳，让他早上在家里等着，他过来接人。<br/>　　<br/>　　“我还以为你把这事都给忽略了，没胆给我打电话了呢”<br/>　　<br/>　　“哈哈，喝多了，喝多了……”王令歌打了个哈哈，就把话题给错过了：“去大澳，你跟着我去就行了，刘家那边我去谈，其他的都不用管，谈不上报复也说不上雪恨，屁事没有”<br/>　　<br/>　　“我无所谓，有你罩着就是了”<br/>　　<br/>　　“得，后天我过去接你……”<br/>　　<br/>　　再说另外一头，陇西。<br/>　　<br/>　　凉绣，小草这两位姑娘这几天经常泛舟于渭河上，钓鱼，谈天，说地，处的关系隐约有朝着闺蜜的方向发展了。<br/>　　<br/>　　用凉绣的话来讲就是，她这些年来都是走到哪里停到哪里的，碰见山清水秀对口味的地方，就暂时住下来一段时间休息休息，她觉得渭河风光不错，鲤鱼又很好吃，就在这里多停几天，只当是休养生息，整理下疲惫的身子了。<br/>　　<br/>　　小草也不以为意，只说你若喜欢就多呆几天，反正她每天的日子也基本都是钓钓鱼看看书过来的，所以几天过去之后，她俩俨然已经差不多要无话不谈了。<br/>　　<br/>　　凉绣和小草之间，总是有意无意的把话题牵扯到她的感情上去，并且聊的还挺起劲的，小草仿佛也很乐意谈论她的情感，基本上是做到了有问必答的状态。<br/>　　<br/>　　她每次说起的时候，脸上总是洋溢出满满的满足感。<br/>　　<br/>　　这天午后，两人撑着竹筏划到了河中心一带，今天天气不错，风和日丽，江水也比较平稳，没有一点风浪。<br/>　　<br/>　　站在竹筏一头，凉绣抻了抻小腰，拍着小嘴说道：“小草啊”<br/>　　<br/>　　“嗯？”钓鱼的小草抬头答应了一声。<br/>　　<br/>　　“咱们老是在这片河面上游荡，也太没意思了点，看得都腻了，我有点审美疲劳了”凉绣忽然转过来，蹲下身子，拉着小草的胳膊，弯着一双月牙眼说道：“下游那边你去过没？我看那里山峦叠起的，感觉应该不错哦”<br/>　　<br/>　　“以前偶尔倒是往那边划过几次，和这里的风景也有点区别，你想去看啊？”小草问道。<br/>　　<br/>　　凉绣晃荡着她胳膊说道：“我过两天就想走了，你带我去看看呗？”<br/>　　<br/>　　小草指着鱼竿说道：“今天的鱼还没有着落呢”<br/>　　<br/>　　“哎呀，吃了三四天的渭河鲤鱼，我已经彻底的领教了这鱼的味道和你的手法了，吃不吃的已经没什么大兴趣了，该让我饱饱眼福了”<br/>　　<br/>　　茅小草收起鱼竿说道：“行，但也只能去一段，再往下水流就该急了，竹筏受不住的”<br/>　　<br/>　　“行，行，一段就一段”凉绣点着脑袋说道。<br/>　　<br/>　　小草从竹椅上站了起来，拿起一根竹竿站到船头，伸到水里说道：“坐稳一点，开船了……”<br/>　　<br/>　　竹筏顺着水流而下，凉绣坐在后面看着前边撑船的小草。<br/>　　<br/>　　小草利索的撑着竹竿，嘴里没心没肺的哼着陇西小曲，曲调欢快，不知愁苦，很是自然。<br/>　　<br/>　　凉绣眯着双月牙眼，不知所想。<br/>　　<br/>　　竹筏往下游走，渐渐人烟稀少，渔船不见，此处周边并没有什么村落人家，所以水域显得稍微有点荒凉，很长一段河道都见不到人影。<br/>　　<br/>　　忽然间，前方下游的水面上，突然传来一阵马达的轰鸣声，三艘快艇离着挺远，就朝这边急速而来。<br/>　　<br/>　　小草停顿了下手里的竹竿，诧异的说道：“哪里来的？”<br/>　　<br/>　　凉绣在她身后，轻声说道：“我叫过来的……”<br/>　　<br/>　　(=)
