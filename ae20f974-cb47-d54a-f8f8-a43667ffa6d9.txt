连星儿说“很冷”的时候，郎安本能地解下自己身上那件军绿色的披风，往后一挥，双臂将连星儿圈在中间，利索地给她穿好扣上。
<br/>
<br/>        “如果大小姐手冷的话，大小姐可以把手插到我衣袖里来。”郎安接着说道。
<br/>
<br/>        而郎安这一本正经的样子，险些让连星儿哭笑不得。
<br/>
<br/>        她是心寒啊！
<br/>
<br/>        又不是身体冷。
<br/>
<br/>        连星儿微微仰头看着郎安，终于破涕为笑：“你真傻！”
<br/>
<br/>        看到连星儿脸上那浅浅的笑意，郎安也情不自禁地微微扬起嘴角，说道：“只要大小姐开心，怎么骂我都行。”
<br/>
<br/>        “好吧！我是感觉到很冷了，我们回酒店去。”连星儿接着说道，无奈地睐了郎安一眼，然后转身往酒店的方向走去。
<br/>
<br/>        郎安立马跟上去，却从来不会跟连星儿肩并肩的走在一排。
<br/>
<br/>        他每次都是跟在她身后，离她不远，也不会太近。
<br/>
<br/>        翌日。
<br/>
<br/>        连星儿还是决定去看比赛，毕竟“C”仍旧是她的师父，若不是贺泽川那个坏男人，她也不会纠结那么久，总是在疑惑为什么网络世界里的“师父”跟现实生活中的“师父”判若两人。
<br/>
<br/>        现在，她不纠结了。
<br/>
<br/>        只不过，她倒是可以进去观看比赛，郎安却不可以，因为她手里只有一张票。
<br/>
<br/>        连星儿此刻站在女生堆里，跟着大家一起排着长队，等待检票入场。
<br/>
<br/>        而郎安则站在不远处的一棵大树下，双手抱臂，倚着树干，默默地注视着连星儿那边。
<br/>
<br/>        “啊——是‘MS战队’的大巴车来了！”突然有女生疯狂地尖叫起来。
<br/>
<br/>        连星儿也跟着转过头去，只见路边有一辆大巴车缓缓地停稳。
<br/>
<br/>        下一秒，周遭的女生们不再排队，而是蜂拥而至地向那边跑去。
<br/>
<br/>        大巴门还未开，车门外便簇拥着一群“MS战队”的女粉丝。
<br/>
<br/>        大家手里还有“助威装备”，一个个都举得高高的，生怕自己的偶像看不到。
<br/>
<br/>        连星儿能看到的，大多数是“Cold”的助威灯牌。
<br/>
<br/>        她的师父到底是大神级别的人物，深受女生喜欢。
<br/>
<br/>        这一次，连星儿倒是安分了许多。
<br/>
<br/>        或许是因为失恋了的缘故，连带着她对追星也没兴趣了。
<br/>
<br/>        来看比赛，纯粹是抱着替师父捧场的心态。
<br/>
<br/>        因为其他女生都跑去“追星”的缘故，连星儿往前站了不少。
<br/>
<br/>        此时，“余寒犹厉”和他的四个战友还有教练从大巴车上下来后，径直往这边的入口走过来。
<br/>
<br/>        他们依旧一个个都戴着口罩和鸭舌帽，只有身上的战队服后的字幕，才能分辨出他们谁是谁。
<br/>
<br/>        连星儿不经意地一瞥，目光落在“C”的身上时，赫然发现，他就是昨天给她抓绵羊公仔的那个男生。
<br/>
<br/>        虽然看不全对方的样子，但是大概能从身形上去判断，他们俩是同一个人。
<br/>
<br/>        连星儿顿时开始仔细打量起向这边走过来的“余寒犹厉”。
<br/>
<br/>        越打量，她就越觉得“师父”像明司寒。
<br/>
<br/>        明司寒望向前方时，在人群里发现了连星儿后，见连星儿正直勾勾地盯着自己，于是抬手将头顶上的鸭舌帽又拉低了点。
<br/>
<br/>        连星儿微微拧起眉头，等着“师父”走向自己这边，但眼瞧着快要接近自己了，结果，身后那群跑回来排队的女生们又激动了。
<br/>
<br/>        一个个往她身旁这边凑，然后硬生生地将她从警戒线后挤了出去。
<br/>
<br/>        “师父”经过的时候，连星儿身前则隔着好几堵“肉墙”。
<br/>
<br/>        可惜了，她还是没看清楚“师父”的样子。
<br/>
<br/>        就连入场后，她坐在观众席上，依旧没有看清楚。
<br/>
<br/>        或许是她太思念明司寒了吧！
<br/>
<br/>        才会把“师父”幻想成明司寒的样子。
<br/>
<br/>        最后的总决赛，进行得很激烈。
<br/>
<br/>        连星儿却是心如止水地观看比赛，情绪上没有任何波澜。
<br/>
<br/>        哪怕“MS战队”进入了瓶颈状态，粉丝们一个个急得搔头挠耳，连星儿依旧淡定自若地坐在那儿。
<br/>
<br/>        但她的目光，从始至终，都未离开过自己的“师父”。
<br/>
<br/>        赛场上，明司寒快速操作着自己的电脑，黑色的屏幕上，白色的程序代码像刷白漆一样，快速地滚动。
<br/>
<br/>        明明没有任何空暇之余，他还是忍不住会抬眸望向观众席，一眼锁定连星儿所坐的位置。
<br/>
<br/>        看到她还在，他就会很安心。
<br/>
<br/>        收了收心思，明司寒继续防守，经过几个轮回的程序破解后，他开始主动攻击对方的战队的“主系统”，而他的队友则替他打掩护。
<br/>
<br/>        直到对方系统瘫痪后，主持人开始了倒计时。
<br/>
<br/>        明司寒自信地笑了笑，志在必得地放开了键盘。
<br/>
<br/>        可是，当他带着冠军非他们战队莫属的喜悦心情，下意识地抬头看向观众席时，口罩下，他嘴角的笑容瞬间全无，深邃的黑眸也忽然间黯淡了。
