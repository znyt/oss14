        朱小溪看到这一幕后，恍然大悟地瞪大了眼睛。
<br/>
<br/>        中年男子抬眸看了傅夜沉一眼，又畏畏缩缩地将目光挪到了一旁，就连傅夜沉伸过来的手，他也不敢去握。
<br/>
<br/>        只因这个男人有威慑四方的气场，令他不寒而栗。
<br/>
<br/>        直到这一刻，朱小溪才明白，连星儿真的不是她想惹就能惹得起的人。
<br/>
<br/>        连星儿回到学校后，并不知道自己的爹地是怎么去处理这件事情的。
<br/>
<br/>        总而言之，她在学校里安然无恙地度过了一天又一天，好像那天在咖啡馆里所发生的事情只是一场噩梦。
<br/>
<br/>        她不敢打电话去问爹地，有关于那件事情的结果，明明不是她的错，但她却像个犯了错的孩子一样郁郁寡欢。
<br/>
<br/>        心里有个时候还会想，如果她的爹地不是傅夜沉，那么发生这种事情后，她是不是也会被警察抓去坐牢？
<br/>
<br/>        “星儿、星儿、星儿！”耳畔忽然响起顾千凝小声的呼唤。
<br/>
<br/>        连星儿回过神来后，才发现自己竟然在课堂上走神了。
<br/>
<br/>        “你最近是不是有什么心事啊？”顾千凝问道。
<br/>
<br/>        连星儿微微摇了摇头。
<br/>
<br/>        顾千凝又担忧道：“可你脸色看起来不大好。”
<br/>
<br/>        “可能是我没睡好，最近这几天晚上睡觉，我都一直在做噩梦。”连星儿皱着眉头说道。
<br/>
<br/>        再加之，这段时间，她还是感觉有双眼睛在暗中盯着自己。
<br/>
<br/>        无论是她走在校道上，还是她去上洗手间。
<br/>
<br/>        好像有一双离她很近的眼睛，在监视着她的一切。
<br/>
<br/>        这种感觉，总会令她不寒而栗。
<br/>
<br/>        顾千凝忍不住抬起手来，一边抚平连星儿的眉心，一边笑着安慰道：“那我今晚跟你睡一张床如何？”
<br/>
<br/>        “好啊！”为了不让顾千凝替自己担心，连星儿强颜欢笑地点了点头。
<br/>
<br/>        下课后，连星儿回到寝室里因为无心做功课，索性打开了笔记本电脑打算玩一会儿游戏。
<br/>
<br/>        巧的是，她的师父也在线。
<br/>
<br/>        连星儿立马主动发了一条消息过去。
<br/>
<br/>        师父，来陪我双排！——Star大小姐。
<br/>
<br/>        我们认识这么久了，是不是应该正式见个面了？——余寒犹厉。
<br/>
<br/>        看到这句话后，连星儿身心一怔，突然莫名其妙地有点儿悸动。
<br/>
<br/>        怎么不说话呢？——余寒犹厉。
<br/>
<br/>        好！——Star大小姐。
<br/>
<br/>        那择日不如撞日，就今天。下午五点，临海大学正校门口，我来接你，邀请你一起共用晚餐——余寒犹厉。
<br/>
<br/>        嗯——Star大小姐。
<br/>
<br/>        那不见不散。这是我的手机号码：——余寒犹厉。
<br/>
<br/>        好的，不见不散。那师父，你现在还陪我双排吗？——Star大小姐。
<br/>
<br/>        不了，我还有事，下午见——余寒犹厉。
<br/>
<br/>        连星儿看完师父发过来的这条消息后，师父的头像便灰了。
<br/>
<br/>        不过，一想到今天下午就要跟师父见面，连星儿的心跳突然加快，“扑通扑通”地打起了小鼓。
<br/>
<br/>        那日在赛场上，她可是亲眼见证了她师父的厉害。
<br/>
<br/>        像她师父那样神级一样存在的男人，哪怕是戴着口罩，看不全他的真实容颜，女生们也都会因为他那一手超高的IT技术而怦然心动。
<br/>
<br/>        她连星儿也毫不例外。
<br/>
<br/>        想到这种感觉，连星儿下意识地捂着自己的脸颊，甜甜地笑了。
<br/>
<br/>        下午五点，连星儿特意精心打扮了一番，不仅穿着漂亮的淡紫色连衣裙，还画了淡妆，自己拿电夹板烫了微卷，如约来到正校门口。
<br/>
<br/>        正当她四下张望的时候，放在手提包里的手机却突如其来地震动起来。
<br/>
<br/>        连星儿从手提包里拿出手机，疑惑地盯着屏幕上这组陌生的电话号码，犹豫了一下，才想起是师父“余寒犹厉”的号码，立即按下接通键。
<br/>
<br/>        “师父？”连星儿亲昵地唤了声。
<br/>
<br/>        “徒儿，看你的左边那辆黑色的私家车。”手机彼端传来令连星儿身心一怔的低沉嗓音。
<br/>
<br/>        连星儿下意识地顺着对方的提示，向左边看去。
<br/>
<br/>        只见一辆黑色的豪华私家车驾驶座车门旁，倚着一个穿着一袭黑色衬衫和西裤，留着平头，五官立体的男人正拿着手机向她招手。
<br/>
<br/>        第一眼看过去的时候，连星儿感觉这个男人跟那天在比赛现场看到的师父给她的感觉不大一样。
<br/>
<br/>        这个男人看上去十分沉稳而低调，而她的师父在赛场上的那天是冷酷而高傲。
<br/>
<br/>        连星儿带着狐疑的心理，向那男人走了过去。
<br/>
<br/>        当她走到这男人的跟前，看清楚这个男人的面貌后，让她再喊一声“师父”，她却喊不出口了。
<br/>
<br/>        “你好，我叫贺泽川。”贺泽川看着连星儿，礼貌性地向她伸出了手。
