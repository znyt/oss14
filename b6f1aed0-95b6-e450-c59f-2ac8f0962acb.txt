　　在即将穿过空间门的时候，琥乾脚步一顿，提醒道：“千万不要触摸空间皱褶，否则，就算是我也救不了你们...”
<br/>　　全部顺利的通过空间门后，撕裂的空间也瞬间恢复，一扇极其严实的空间皱褶墙壁，将藏书阁护在其中。
<br/>　　视线跃过空间皱褶，再次看向藏书阁，楼阁下的灰袍老者已经再次融入空间中，在实力弱小的修炼者眼中，周围就是空空如也。
<br/>　　随着琥乾与学员的离去，空间秘境再度回复以往的平静，直到明年内院选拔赛结束，这里才会重新开启。
<br/>　　...
<br/>　　...
<br/>　　宽敞的书房中，琥乾望了一眼渐渐合拢，不留一丝缝隙的墙壁，转过身来，看向慕清霄几人。
<br/>　　“接下来的最后一轮选拔赛，与其说是选拔赛，倒不如说是考验你们的实战能力，与选拔比试截然不同，同时也关乎着在内院的发展。”琥乾的语气颇为慎重。
<br/>　　“这里可不是象牙塔，没有实力，你们的修为也会停滞不前，新人想在初进内院时，不被老生欺负，除非你们拳头够硬。”
<br/>　　慕清霄始终面无表情的站在一旁，而是想着，进入内院后，尽快前往练气塔，将陨落心炎炼化...
<br/>　　至于古薰儿与其他的学员，都各怀心思。
<br/>　　“所以，我奉劝你们，尽量放弃彼此间的间隙，彼此合作，特别是在初入内院的时候，虽然不至于被废，或者被杀，但后果依旧不是很好受。”
<br/>　　随后，琥乾的脸色恢复以往，拿起桌上的书籍，朝他们挥了挥手。
<br/>　　“至于别的信息，到时候你们自己就会清楚的，行了，没事的话，都回去吧，两天后来这里报道，到时候带你们进入内院，至于住宿方面，找你们各自的导师。”
<br/>　　其余五名男学员欲言又止，道过谢后，慕清霄与小医仙则径直离开，看都不看他们一眼。
<br/>　　彼此间的关系僵硬无比，至于相互合作，简直就是笑话。
<br/>　　“这些小家伙都是不撞墙不知道疼，等你们进入内院，便知道那里生存的艰难，那里可从来都不缺少天才啊。”
<br/>　　望着相继离去的几人，琥乾坐在书桌前，手指轻缓的敲击着桌面，语气平淡的说道。
<br/>　　......
<br/>　　......
<br/>　　离开藏书阁后，几人分道扬镳，由于霍德并没有回来的缘故，慕清霄和小医仙则找到了若琳导师。
<br/>　　在若琳的帮助下，两人暂时住在了外院学生宿舍中。
<br/>　　在这两天的时间里，慕清霄并没有修炼，而是带着小医仙在外院转悠了两天。
<br/>　　在阴阳玄龙丹的药效滋生后，伤势则迅速的恢复起来，原本需要半个月的恢复期，现在估计一周就能完全恢复。
<br/>　　而两天过去，慕清霄的实力就从元婴初期，恢复到了元婴巅峰。
<br/>　　竖日清晨，慕清霄睁双手缓缓结印，旋即一口黑色血液就从嘴中吐出，睁开双眼，脸色已经比昨日好上许多。
<br/>　　“仙儿，我们走吧，前往内院。”
<br/>　　当慕清霄与小医仙来到图书馆大楼前，只见门前的空地上，已经占满了不少人群。
<br/>　　人群大致分散在周围，各自低声谈笑着，而在这些群体的中间，古薰儿自然成为了一道靓丽的风景线。
<br/>　　虽然后者并不打算进入内院，但欢送仪式却依旧要参加的。
<br/>　　至于其他进入内院的学员们，相比起古薰儿，就变得有些默默无闻，站在一旁无人问津。
<br/>　　慕清霄与小医仙出现，使得热闹的空地上安静了一些，周围学员则纷纷投来复杂的目光。
<br/>　　“喂喂，你们听说没，他们两个好像就是特招生？”
<br/>　　“特招生？难道是传说中的关系户？”
<br/>　　“你们都小声点，人家可是关系户，小心给你穿小鞋，我们得罪不起的。”
<br/>　　无视周围的闲言碎语，慕清霄两人穿过人群，来到空地上，静静的站在此处等候。
<br/>　　约莫十几分钟后，图书馆大门终于开启，副院长和几名外院长老走了出来，整个空地上的窃窃私语也随之消失。
<br/>　　仅仅片刻，整个图书馆前的广场就变得鸦雀无声。
<br/>　　“各位同学，今日便是你们进入内院的时候，祝贺你们长久的努力得到回报。”琥乾目光扫过空地上五十名学员，见未有缺席后，上前一步，朗声说道。
<br/>　　“我相信，进入内院后，你们或许会不适应其中的修炼方式，不过有一点我很确定，内院能够最大程度激发你们的潜力，只要坚持下来，你们将脱胎换骨，你们当中想必有人知道，内院学生与外院学生的差距，是多么的巨大。”
<br/>　　听到琥乾的话语，在场不少学生微微点头，当初内院学生假期来到外院，他们也互相交流切磋过，差距简直太明显了。
<br/>　　而更多的人，则满脸期盼与兴奋，他们进入内院的目的，可不就是为了追求更强大的力量么？
<br/>　　他们坚信，不管内院的修炼是如何的苛刻与残酷，自己都能坚持下去，最后成为学员的骄傲。
<br/>　　能够从迦南学院中脱颖而出，这就证明了他们的天赋与努力，能够站在这里的人，都非常的清楚，想要得到力量，自己就得付出努力。
<br/>　　“内院，是迦南学院的核心地所，由于其保密性，大多数外院的学生包括导师，都不知道它的确切方位，因此，我们会将你们送达内院范围中。”
<br/>　　琥乾的话语声刚落，抬头望着远处的蔚蓝天空，只见十道黑影正飞掠而来，黑影逐渐变大，赫然是十只巨大的狮鹫兽。
<br/>　　十只狮鹫兽的阴影划过学院，最后在图书馆高空停了下来，双翅挥动间，狂风涌动，将下面一些学员身形都吹的有些站不稳脚跟。
<br/>　　琥乾冲着天空上的狮鹫兽挥了挥手，旋即阴影压地而来，最后在狂风涌动间，十只狮鹫兽已经停留在空地不远处。
<br/>　　而在狮鹫兽的背部，都有两名专门驾驭狮鹫兽的学员。
<br/>　　“时间也差不多了，各位同学出发吧。”
<br/>　　待所有学员都登上狮鹫兽，琥乾与几位长老相视一眼，身形一动，已经闪掠至上空，肩部一震，四双绚丽的斗气羽翼，便自背部伸展而出。
