　　不周神山，一处隐秘山洞之中。
<br/>　　随着十二声巨大的爆炸声响，震动山洞方圆数十万里，但这点震动相对于正在处于大战中的的洪荒，不值一提。
<br/>　　震动过后，十二道巨大身影突兀显现在山洞之中。
<br/>　　却是盘古所留的十二滴精血化形而出，他们就是十二祖巫。
<br/>　　十二祖巫形态各异，但每一个身上的气息都不弱于太乙金仙。
<br/>　　十二祖巫分别是：
<br/>　　空间祖巫帝江：状如黄囊，赤如丹火，六足四翼，浑敦无面目；
<br/>　　木之祖巫句芒：青若翠竹，鸟首人面，足乘两龙；
<br/>　　金之祖巫蓐收：人面虎身，身披金鳞，胛生双翼，左耳穿蛇，足乘两龙；
<br/>　　水之祖巫共工：蟒头人身，身披黑鳞，脚踏黑龙，手绕青蟒；
<br/>　　火之祖巫祝融：兽头人身，身披红鳞，耳穿火蛇，脚踏火龙；
<br/>　　风之祖巫天吴：八首人面，虎身十尾；
<br/>　　雷之祖巫强良：嘴里衔蛇，手中握蛇，虎头人身，四蹄足，长手肘；
<br/>　　电之祖巫弇兹：人面鸟身，耳挂青蛇，手拿红蛇；
<br/>　　时间祖巫烛九阴：人面蛇身，全身赤红；
<br/>　　天气祖巫奢比尸：人面兽神，双耳似犬，耳挂青蛇；
<br/>　　雨之祖巫玄冥：人面鸟身，全身骨刺，双耳各悬一青蛇，脚踏双头龙；
<br/>　　土之祖巫后土：人身蛇尾，背后七手，胸前双手，双手握腾蛇。
<br/>　　化形后的十二祖巫相互看了看对方，各自化为先天道体之象，帝江、句芒、共工等化为十个大汉，而后土、玄冥二人则化为两个端庄秀丽的女子。
<br/>　　“大哥！”
<br/>　　其余十一个祖巫都来到帝江面前，对他行礼道。
<br/>　　“不必多礼，如今洪荒正处大劫，我等修为还不够强大，还是就在这盘古殿中修炼，等大劫过后，再出去。”帝江对着其他十一个祖巫道。
<br/>　　“是，大哥。”
<br/>　　……
<br/>　　就在十二祖巫化形后百年，陡然，四海深处，一道龙吟传出，其声嘹亮，回荡天地八方三开，惊天动地。祖龙愤怒之声响彻云霄：
<br/>　　“凤祖，麒麟王！欺龙太甚！尔等要战，那便战！吾龙祖又何曾惧过谁？从今以后，洪荒先天百族，都必须以龙族为尊，尔等若是不服，尽管来战！”
<br/>　　“昂！昂！昂……”
<br/>　　龙祖声音过后，刹那间，四海之内，一道道龙吟声响起，响彻天地，一条条巨大真龙之身显现虚空，金光四射。
<br/>　　“战！战！战……！”
<br/>　　一股冲天战意惊动云霄，向着麒麟族、凤凰族碾压而去，久久不消。
<br/>　　要问龙祖为何如此愤怒，却是因为魔祖罗睺的算计，将龙祖的儿子大太子灭杀，嫁祸到麒麟族与凤凰族。大儿子都没了，龙祖不愤怒才怪。
<br/>　　在龙祖的带领之下，无数真龙破海而出，滚滚气势，直奔凤凰族、麒麟族老巢冲击而去，随之而起的，还有依附于龙族的各大先天百族，此刻也一个个出动。
<br/>　　“龙祖，你既然要称霸洪荒，那也要看我同不同意！战！”
<br/>　　不周山麒麟崖，麒麟王也在怒吼。
<br/>　　“龙祖，想要百族臣服，休想！”
<br/>　　南方不死火山，凤祖、凰祖也在喊道，这些年死在龙族手中的凤凰成员，亦不在少数。
<br/>　　一时之间，麒麟、凤凰两族也都纷纷冲天而起，凤祖和麒麟王打头阵，二人都气势滔天，带领着麾下各路强者，直奔龙族。
<br/>　　大战一触即发！
<br/>　　“哈哈哈，终于打起来了，本尊可是等了好久啊！这一天终于来了！哈哈哈！”
<br/>　　西方灵山魔窟之中，罗睺狂声大笑，周边，四柄凌厉的长剑散发着无边煞气，剑气冲霄，无边的煞气滚滚涌来，罗睺张嘴一吸，顿时，就将这些煞气吸入腹中，整个人变得魔威滔天。
<br/>　　“大战终于来了吗？也罢，贫道也要多做准备了，他们的大劫过后，就是贫道的大劫了，看来贫道还要去拜访几个道友，邀他们前来助阵。”
<br/>　　西昆仑玉京山上，鸿钧老祖睁开双眼，冒出金光，看向龙族所在之地，轻声呢喃道。收起头顶的造化玉蝶，他迈开脚步，向着山下走去。
<br/>　　“大劫已经开始了，不知道我等大劫过后还有几人能活着。”
<br/>　　其他各处名山，各大能也纷纷被这股气势惊醒，知道大劫已经开始了，各自都在纷纷闭关不出，躲避大劫，只是，躲避真的有用吗？
<br/>　　“嗯？大劫已经开始了吗？”
<br/>　　东海永恒岛上，玉虚宫中，正在闭关的元始亦被这股大劫的气息惊醒，微微一想，便知道是龙凤麒麟三族大劫已经开始了。不过，他面上并无半分波动，知道大劫来临后，便不再多管。
<br/>　　“这是怎么回事？想必就是大兄所说的大劫了？”太清岛上老子与五行岛上通天也感受到这股气势，纷纷猜测这其中的原由。
<br/>　　“天地无量大劫已至，你们最近都在洞府修炼，不要外出。”
<br/>　　就在这时，一道淡淡的声音传入他们耳中，让两人一惊，不过，听到这熟悉的声音后，便放松了下来，知道这是自家大兄的传音。
<br/>　　“是，大兄！”
<br/>　　……
<br/>　　“凤祖，麒麟王，尔等欺龙太甚，此战过后，龙族为天地之主，百族之首，战！”
<br/>　　龙祖大吼，卷起四海之水，带着一股不可撼动的遮天气势，轰然冲出，对上凤祖与麒麟王。其余龙族对上两族成员，地龙对上凰祖，大战一发不可收拾。
<br/>　　“龙祖，你别猖狂，吾来战你！”
<br/>　　麒麟王化作一身披五彩帝袍，头戴五彩帝冠的大汉，手拿麒麟杖，挥动着向龙族打去。
<br/>　　“龙祖，想要称霸洪荒，简直是痴人说梦。麒麟王，我来助你，战！”
<br/>　　凤祖化作一身穿金色凤袍的英俊男子，手持长剑，向着龙祖斩去。
<br/>　　“哈哈哈！来吧，吾何惧之！看我龙皇霸拳！”
<br/>　　龙祖见麒麟王与凤祖都想自己攻战而来，毫不畏惧，心中一阵豪情，挥动双拳，霸道拳意轰动天地，震得苍穹动荡，迎上麒麟王与凤祖。
<br/>　　三人顿时打得惊天动地，旁人根本插不进去。
<br/>　　一旁凰祖想要进去帮忙，但就在此时，她身前来了一人，将她拦住。
<br/>　　“想要上去帮忙，你还要问过我同不同意。”
<br/>　　“地龙，滚开！”凰祖望着眼前气势惊人的地龙，美目之中闪过一股凝重之色，怒气冲冲的娇声喝道。
<br/>　　凰祖伸出纤纤玉手，对着地龙就是惊天一掌，人虽美，但出手狠辣，毫不留情。
<br/>　　地龙没有说话，同样挥动手中拳头，迎上凰祖，两人顿时战在一起。
