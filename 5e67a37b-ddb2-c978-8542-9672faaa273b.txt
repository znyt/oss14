　　“额。”萧逸吃痛，手臂上那剧烈的疼痛，让他脚步微微停顿了一丝。
<br/>　　就是这一丝停顿，另一头荒鳄一尾巴从背后袭来，朝浅沫打去。
<br/>　　“阿。”
<br/>　　“别怕。”
<br/>　　萧逸反应极快，忍下手臂的疼痛，一个转身，抽出紫云精铁剑抵挡。
<br/>　　锵的一声，紫云精铁剑直接被抽飞，巨大的冲击力将萧逸一并击飞。
<br/>　　“噗。”萧逸猛地吐出一口鲜血。
<br/>　　“易霄，你没事吧。”浅沫着急地问道。
<br/>　　“不想我死就闭上你的嘴巴。”萧逸说了一声，忍痛再次跳跃了起来。
<br/>　　若是只有萧逸一人，这些荒鳄休想伤他分毫。
<br/>　　幸好，浅沫之后都没有再大呼小叫，而是静静地趴在萧逸背后，并用长生藤武魂为萧逸治疗手臂，不时偷看萧逸一眼。
<br/>　　在她眼中，这满脸鲜血的少年，脸庞上那坚毅的表情，让她不想挪开眼睛。
<br/>　　她甚至没有发现，自己的眼神竟变得温柔起来。
<br/>　　不多时，铁达大叔将荒鳄尽数击杀，萧逸才松了口气。
<br/>　　“易霄。”铁达大叔和林飞黄源看萧逸满身是血，立马过来询问。
<br/>　　不得不说，浅沫的长生藤武魂果真厉害，萧逸的手臂伤得算是比较严重的，此刻竟差不多痊愈了。
<br/>　　萧逸也吃了一惊，却在证实了他之前的猜测，长生藤果真不是寻常武魂。
<br/>　　铁达大叔忽然严肃地说道，“浅沫，你知不知道你差点害死易霄兄弟了。”
<br/>　　“我...”浅沫委屈难过地低下了头。
<br/>　　“我什么我。”铁达大叔教训道，“我们猎妖师是相当危险的职业，可不是你想象中的刺激冒险和玩乐，稍有不慎，会死人的。”
<br/>　　“幸好被咬的是手，那要是头，你的长生藤武魂能把头接上去吗？”
<br/>　　浅沫被教训得差点没哭出来。
<br/>　　“算了铁达大叔。”萧逸摆摆手臂，道，“我现在不是没事了嘛，浅沫毕竟是个小姑娘，没多大经验。”
<br/>　　经过这些时间的相处，萧逸发现，浅沫是个不谙世事的小姑娘，没有半点防人之心，更没有半点儿历练经验。
<br/>　　铁达大叔严肃道，“小姑娘？易霄兄弟你也没比她大几岁吧。哎，算了，既然易霄兄弟你不怪她，我也不多说什么了。”
<br/>　　“易霄，对不起。”浅沫满脸愧疚和难过。
<br/>　　萧逸笑道，“得了，我当年刚出道时，也是个没经验的菜鸟，经常犯错，经常大呼小叫的，久而久之就会成熟起来的。”
<br/>　　“噗哧。”浅沫忽然展露笑容，道，“还你当年呢，你不也是个新手炼药师吗？说话老气横秋的。”
<br/>　　萧逸笑笑，说道，“总之记住了，下次有危险，躲我背后就行。我既然把你背起来，自然不会让你受伤。”
<br/>　　萧逸可不想浅沫再添乱，这次还只是手臂受伤，下次可能就没那么幸运了。
<br/>　　“嗯，知道了。”浅沫低声说道。
<br/>　　“咦。”这时，杜新忽然惊叫一声。
<br/>　　在荒鳄都被杀光后，他就在房间内四处摸索，看有没有什么宝物。
<br/>　　“哇，我找到了一本黄阶巅峰武技。”另一边，传来了黄源激动的声音。
<br/>　　“这...这是破玄城失传了多年的御风灵弓。”林飞也是惊呼一声。
<br/>　　“嗯？”萧逸和浅沫站起来。
<br/>　　之前顾着战斗和抵抗荒鳄，现在才发现，这房间内竟有不少古朴的盒子。
<br/>　　林飞等人，早已将匣子开了个遍。
<br/>　　盒子中，有丹药，有兵器，有天材地宝，还有武技。
<br/>　　萧逸看了几眼，没多大兴趣。
<br/>　　丹药和天材地宝都是二品以下，武技书籍最厉害的不过是黄阶巅峰，兵器也只是普通兵器。
<br/>　　这里最好的东西，应该是林飞找到的御风灵弓。
<br/>　　御风灵弓，相传曾经是破玄城一位有名的箭武魂武者的武器，比不上灵器，但也算威力不凡。
<br/>　　萧逸奇怪地看了浅沫一眼，发现她同样没多大兴趣，只是呆呆地站在自己旁边，不时傻笑。
<br/>　　“你笑什么，有很多宝贝呢，去挑一些吧。”萧逸道。
<br/>　　“阿...”浅沫反应过来，低声道，“不用了，那些宝贝给队长他们吧，我在这陪你。”
<br/>　　她的声音说到最后有些低。
<br/>　　萧逸听不太清楚，但也懒得问。
<br/>　　不多时，铁达大叔拿着满满一堆盒子过来。
<br/>　　“易霄兄弟，你是炼药师，这里面全是天材地宝，便都归你了吧。”铁达大叔道。
<br/>　　“都归我？”萧逸一愣，说实话，他对这些低品材料没兴趣，“还是分给其他人吧。”
<br/>　　“就是。”这时，一旁的杜新拿着一堆盒子过来，不满道，“这么多天材地宝，凭什么都给易霄。”
<br/>　　林飞和黄源这时也回来了，道，“这些天材地宝我们也没用，还不如给易霄兄弟呢。这一路上，易霄兄弟给我们的丹药，都抵得上这里所有天材地宝的价值了。”
<br/>　　“哎呀，拿着吧。”铁达大叔将盒子全放入乾坤袋，全给了萧逸，连乾坤袋也送了。
<br/>　　“这...”萧逸刚想推辞。
<br/>　　铁达大叔道，“别推辞了，我们个个都有好东西呢，你看，我拿了一把大刀，也是上等的武器，还有一本武技呢，可比你的材料值钱得多。”
<br/>　　萧逸笑笑，道，“好吧。”
<br/>　　“哼。”一旁的杜新满脸不忿埋怨。
<br/>　　半晌，当众人将房间翻了个底朝天后，发现再没有其他宝物了，才算罢手。
<br/>　　而后众人又把荒鳄的尸体搜集起来。
<br/>　　“队长，你说这里怎么会有这么多荒鳄，而且实力还这么强。”林飞一边收拾妖兽尸体，一边问道。
<br/>　　萧逸眼尖，忽然道，“你们看，这些荒鳄果然不对劲。它们的头上有只尖角，外头的荒鳄可没有。”
<br/>　　铁达大叔思索了一下，缓缓说道，“但凡强者的洞府，必要妖兽守护。”
<br/>　　“如果我没有猜错的话，这些荒鳄，应该就是狂血玄君当年禁锢在这里的，为他看守洞府。”
<br/>　　“这么多年来，这些荒鳄被困在这里，可能产生了变异吧。”
<br/>　　众人点点头。
<br/>　　房间的另一头，有一门，和进来的门遥遥相对。”
<br/>　　铁达大叔说道，“那门，应该是通向更深处的。”
<br/>　　杜新兴奋道，“这才是刚进来宫殿，已经有这么多宝物了。要是再深入，说不定会出现更好的宝物。玄阶武技、地阶武技，甚至是灵器。”
<br/>　　这些宝物，他们平时可不敢想。但现在，似乎有希望得到。
<br/>　　林飞忽然说道，“没想到宫殿内竟然是一个房间。而且，还有其他猎妖队也进入了这个宫殿的，但走的不是同一个门，竟然就到不同房间了。”
<br/>　　杜新急道，“废什么话，其他猎妖队肯定也在其它房间找到宝物了，甚至已经更加深入了，我们也快走吧。”
<br/>　　铁达大叔点点头，一想到可能会有地阶武技，甚至是灵器，任何武者都无法抵抗这种诱惑。
<br/>　　铁达大叔推开了门，然而，门的外面，却有一层屏障。
<br/>　　众人穿过屏障，顿时一惊。
<br/>　　因为，这里竟然是一条宽阔的长廊。
<br/>　　长廊内，起码有上百猎妖师，以及无数的荒鳄。
<br/>　　激烈的战斗，在这长廊中不停发生着。
<br/>　　地面上，尸体遍地，有人类武者的，也有荒鳄的。
<br/>　　“又是荒鳄，而且它们头上都有尖角，该死，又是那些麻烦的妖兽。”众人皱起了眉头。
<br/>　　嗖，数头荒鳄已经朝他们攻来。
<br/>　　“先退回去。”萧逸喝了一声。
<br/>　　然而，当众人转过头时，那扇门已经消失。
<br/>　　而且，一股虚无的力量生生地阻挡了他们。
<br/>　　“该死，回不去了。”铁达大叔咬了咬牙，一刀将那荒鳄劈成两半。
<br/>　　“只能一战了，小心。”萧逸大喝一声，一剑将一头准备袭击的荒鳄拦下。
<br/>　　“你们贴着墙壁走，跟紧在我身后。”铁达大叔吩咐一声，再次拔刀而出。
<br/>　　长廊不同于房间内。
<br/>　　铁达大叔挡在前方，可抵挡大部分荒鳄的攻击。
<br/>　　只会不时有一两头荒鳄能偷袭到萧逸等人。
<br/>　　林飞三人在房间内得到的武器让他们实力大增，已然不惧这一两头荒鳄的袭击。
<br/>　　林飞一箭射出，在御风灵弓的增幅下，将荒鳄射伤。
<br/>　　黄源杜新则联手将其斩杀。
<br/>　　如无意外，拼着受伤的代价，众人能安全地渡过这条长廊。
<br/>　　可是，恰在此时，长廊的前方，一群猎妖师忽然回头，个个手拿巨斧，凶神恶煞。
<br/>　　其中一个为首的中年人，更是手持一把威力惊人的巨斧。一斧头劈下，数头荒鳄被砸成烂泥。
<br/>　　“遭了，是巨斧猎妖队的人，该死，怎么偏偏在这个时候遇到他们。”铁达大叔顿时脸色难看。
<br/>　　“铁达，好巧阿。”中年人阴恻恻地走了过来。
<br/>　　话音刚落，十多个猎妖师纷纷围了上来。
<br/>　　“巨斧，你想做什么？”铁达大叔冷冷地问道。
<br/>　　“你说呢，呵呵。”巨斧冷笑道，“不干事的都给我滚吧。”
<br/>　　巨斧的眼光，不屑地瞥了萧逸等人一眼。
<br/>　　“易霄兄弟，还有林飞你们先走。”铁达大叔凝重地说了一声。
<br/>　　“好嘞，我们....”杜新一副如蒙大赦的表情，刚想说些什么。
<br/>　　林飞和黄源抢先说道，“队长，我们一起来的，自然要一起走。”
<br/>　　萧逸笑道，“铁达大叔，没了你，我们可没法安全离开这长廊，所以还是一起走吧。”
<br/>　　“你们...这是我和巨斧的仇怨，与你们无关。”铁达大叔呵斥道。
<br/>　　“一群废物。”巨斧已经不耐烦了，“要不是我要忙着杀荒鳄，还轮得到你们说话？想活命的立刻滚，别自己找死。”
<br/>　　“呵。”萧逸不屑一笑，猛地站前一步，冷冷地盯着巨斧。
<br/>　　“区区一个先天四重，我还不放在眼里。我来这里是杀妖兽的，别逼我杀人。”
