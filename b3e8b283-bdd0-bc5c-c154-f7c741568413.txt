        &lt;h3&gt;第三卷第八十九章  纠结的冯睿&lt;/h3&gt;
<br/>
<br/>        冯睿被打了，她当晚就回了娘家，在娘家呆了一个多月。
<br/>
<br/>        可是陈子建也不去接她回家，她的心里就七上八下的，她上班的地方其实离家里很近，但是她就是不想回去，他陈子建必须给她认错。
<br/>
<br/>        他认错了她才回去。
<br/>
<br/>        她恨陈家的那些个穷酸相，她好心好意的请他们吃饭，他们竟然不懂恩情，陈子建竟然为了一只手镯打她，难道她在他心里比不上一只破手镯吗？
<br/>
<br/>        如果他不认错，她就和他离婚，谁离不开谁？
<br/>
<br/>        她坐在沙发上看电视，电视的内容她一点也没有看进去。
<br/>
<br/>        她的母亲说：“陈子建今天还没有给你打电话？”
<br/>
<br/>        “没有。”冯睿不耐烦的回答。
<br/>
<br/>        “那个时候就不让你跟他，你偏偏一根筋，好了，现在看着你年龄大了开始动手打你。”母亲絮絮叨叨的。
<br/>
<br/>        “他从来都没有打过我。”冯睿辩解。
<br/>
<br/>        “呵呵，有第一次就会有第二次，你等着吧。”母亲冷冷的笑着。
<br/>
<br/>        “妈，你说什么话呢？”冯睿心里更是难过，就开始怼她妈妈了。
<br/>
<br/>        “我说的是实话，你在家里呆了这么久，他陈子建都不来接你回去，你以为你们的感情有多深厚？”
<br/>
<br/>        “他是生气了。”
<br/>
<br/>        “他为什么生气啊？不是还是爱的不够深吗？要是爱得深，别说是一个破手镯，就是一片江山都会给你。”
<br/>
<br/>        “你能不能不要说了？”冯睿将手中的遥控器摔到地上，遥控器摔成两半。
<br/>
<br/>        “你冲我发什么火啊？自己的男人管不住，给我摔脸子？我吃你的了，还是喝你的了。”冯老太太将电视遥控器拿起来心疼的看了又看。
<br/>
<br/>        “你......”冯睿气急，她怎么连一个遥控器都不值了？
<br/>
<br/>        “妈，这几年家里吃的喝的不是我的？陈子建的工资都给了家里了，包括冯盺他们都是我给的生活费。”
<br/>
<br/>        “你们原来住的房子不是我们掏钱买的？就陈子建那一点工资顶什么用？我将你养大为什么？”
<br/>
<br/>        “妈，你原来不是这样的。我们原来住的房子是陈子建的嫁妆。”
<br/>
<br/>        “我怎么样的了？你弟弟托你办的事情你没有办，现在你弟弟失业在家，你满意了？”冯老太太冰冷的话砸在冯睿的心口。
<br/>
<br/>        在母亲的心中，弟弟一直是她的心头肉，无论她怎么做，都达不到母亲的满意。
<br/>
<br/>        她这一生一直努力，努力的给家里赚钱，努力的养家糊口，对于陈子建家里的她从来都没有给过钱，除了过年过节和陈子建一起回去买点东西之外，钱是一分钱都不会给的。
<br/>
<br/>        弟弟没有工作，家里的生活费都是她承担的，就是因为她没有来得及给陈子建说弟弟的要求，母亲就这样。
<br/>
<br/>        冯睿拿起包，回家了。
<br/>
<br/>        陈子建和冯睿当着弟弟妹妹的面吵了一架，大家都走了，就剩了他一个人，面对着空荡荡的家，陈子建心里一阵阵的愧疚和失落，作为家里男孩子中的老大，他从来都不曾为家里做过任何一件事情。
<br/>
<br/>        他的工资交给了冯睿，每个月冯睿只给他极少的生活费，感觉这一生他是为冯家而活的，他每次想要给家里贴补些家用的时候，冯睿不是哭闹就是撒娇，弄得他完全的没有脾气。
<br/>
<br/>        让他不能理解的是，冯睿还插手他的工作，不可原谅的是，冯睿将父亲给他用来寻找亲人的手镯送人了，连招呼也不给他打一个，这让他如何对父亲交代。
<br/>
<br/>        他记得有个朋友送了一个翡翠烟斗给他，虽然他很少抽烟，但是他觉得那个烟斗很不错，他还是接受了，他在家里找了找，看看冯睿是不是也送人了，书房里没有，他打开了储物间，冯睿曾经让人将储物间重新装修过，但是他都没有去找过东西。
<br/>
<br/>        当他打开储物间的时候，他傻眼了。
<br/>
<br/>        储物间里摆放着的都是名贵的字画和古玩，他细细的拿出来看了看，每一件价值都不菲。
<br/>
<br/>        原来外间传闻他酷爱收藏是真的，只不过不是他本人，而是他的好妻子冯睿。
<br/>
<br/>        陈子建一夜无眠，他不知道冯睿何时变成了这样的。
<br/>
<br/>        但是这些东西是万万不能放在家里了。
<br/>
<br/>        他到单位，写了一封长长的检讨，自检了自己工作上的不足，打电话给潘青松，找相关部门将这些东西上缴了。
<br/>
<br/>        潘青松看着那个曾经意气风发的二哥，心里一阵难受；“二哥，东西你已经上缴了，也就没有必要自责，你能发现问题，及时改正就是好的，以后让二嫂不要再拿人家的东西了。”
<br/>
<br/>        “青松，我悔啊。我决定调换一下工作岗位，休上一段时间假，回去陪陪父亲和母亲。”
<br/>
<br/>        “二哥，其实你完全不必要这样做。”
<br/>
<br/>        “青松，冯睿的心已经被腐蚀了，如果我还在这个位置上，她怕是收不了手的。在哪里都是工作，为了有个安稳的晚年，我还是要调换工作岗位的。”
<br/>
<br/>        陈子建休了长假，回去陪他的父亲和母亲了。
<br/>
<br/>        当冯睿回家的时候，家里已经落了厚厚的一层灰，储物间里什么都没有，陈子建也辞了保姆。
<br/>
<br/>        看着空空的储物间，冯睿心里大惊，这些东西都去哪里了？
<br/>
<br/>        “陈子建，你在哪里？”冯睿拨通了陈子建的电话。
<br/>
<br/>        “有事情吗？”
<br/>
<br/>        “我储物间的东西去哪里了？”
<br/>
<br/>        “我上缴了。”
<br/>
<br/>        “陈子建，你混蛋，那是我收藏了很多年的宝贝。”
<br/>
<br/>        “冯睿，那些东西是怎么来的，你心里很清楚吧？我这样做也是为了你好，你好自为之吧。”
<br/>
<br/>        “陈子建，我要和你离婚。”
<br/>
<br/>        “如果你想离，你写好东西，我签字就是。”陈子建沉默了一会。
<br/>
<br/>        “陈子建，我不知道我当初是怎么看上你这个土鳖的，你们家都是土鳖，一屋子土鳖。”冯睿恶狠狠的骂着。
<br/>
<br/>        陈子建听着冯睿的骂声，看着远处的大山，不知道在想什么。
<br/>
<br/>        冯睿拿着电话一直在叫骂，见对方没有动静了，觉得没有意思，便挂了电话。
<br/>
<br/>        挂完电话，冯睿痴痴的站在客厅里，当突然安静下来之后，她感到了一阵恐慌，陈子建去哪里了？家里这个样子看来陈子建是很久没有在家里呆了。
<br/>
<br/>        她打电话给陈子建单位的小陈，小陈因为上次被陈子建批评，又因为陈子建已经调换了单位，对她的态度不太好。
<br/>
<br/>        “嫂子，陈哥已经不是领导了，你也没有必要再对我呼来喝去的了，我不是给你们家打工的。”
<br/>
<br/>        “你说什么？”冯睿不相信小陈的话。
<br/>
<br/>        “你不知道吗？陈哥已经离开单位十来天了。”
<br/>
<br/>        冯睿又给她儿子打电话，儿子已经结婚另住，儿子也不知道陈子建去了哪里。
<br/>
<br/>        当她将电话再次给陈子建打过去的时候，陈子建的电话已经关机了。
<br/>
<br/>        “呸，陈子建，当初真是瞎眼了，放着好好的领导你不当，呸呸、土得掉渣，哼，谁离不开谁？”冯睿生气的坐在沙发上。
