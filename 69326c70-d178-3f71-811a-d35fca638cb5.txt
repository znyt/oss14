　　第191章
<br/>　　方伯山和杨龙停听到的两个『大事不好』，是同一件消息：北蛮部大军放过蛤蟆寨不管，戎族大王子领兵八万，直扑幽州而来。
<br/>　　先攻落叶城，落叶城无人守，破城，屠杀一日一夜。
<br/>　　再打赤沙城，以难民冲城，守军惊慌失措，被戎族大军冲到城门下，破城，赤沙城城主带着亲随从小门逃跑。
<br/>　　屠城。
<br/>　　又打铁河城，铁河城坚守不出战，戎族留下两万人困城，声明降者免死，不降屠城。
<br/>　　分出六万人攻打幽州城，杨龙停亲自上阵指挥，与士兵共存亡，士气大振，戎族自知不敌，攻北镇城。
<br/>　　投石机将北镇城城墙攻破，士兵奋死力战，生在危机关头修好城墙。
<br/>　　话说青州牧万古风，打探得知蛤蟆寨的山贼都挡了戎族两月，便觉戎族也不过尔尔，打着驱除蛮夷，解救苍生的旗号领着五万大军浩浩进入幽州。
<br/>　　与戎族遭遇，在戎族铁骑的冲锋下，几个回合便将万古风的大军冲散，死伤无数。万古风带着一万残兵被困长风谷。
<br/>　　戎族围而不杀，万古风几个儿子知道父亲被困的消息后，领大军来助阵，但在戎族铁骑之下根本不堪一击。
<br/>　　标准的围点打援战术，最后至少消灭十万青州兵，万古风冒死突围，带着残兵败将狼狈逃回青州。
<br/>　　此时，被困的铁河城自知不敌，城主出门投降，只求戎族莫要祸害城中百姓。戎族大摇大摆进城。
<br/>　　屠城。
<br/>　　戎族八万大军在幽州境横行一月，正应了程大雷推测的那八个字：生灵涂炭，伏尸遍野。
<br/>　　幽州三五百万人口，骤降一半。除了幽州、北镇二城未破以外，其他三城都遭遇了戎族的血洗。
<br/>　　城中的居民至少还有些希望，但城外的百姓却连半分希望也无。戎族骑兵横行在帝国的土地上，以射杀百姓为乐，看着他们狼狈逃命的模样，骑在马上哈哈大笑。
<br/>　　这一战，本该让帝国明白，戎族并非不可战胜。幽州城坚固自不必说，如北镇城只是一座小城，却也在戎族强攻下坚守不破。野战，的确帝国暂无和戎族骑兵抗衡的办法，但攻城一事，戎族并不擅长。
<br/>　　只要将士一心，存了破釜沉舟之意，那么戎族除了围城，并没有更好的破城办法。
<br/>　　可多年的失败，早已让太多人换上恐戎症，戎族还未打来，便做鸟兽散。
<br/>　　幽州城下，难民如蚁。
<br/>　　“放我进去，放我进去，求求你们了，让我进去！”
<br/>　　无数人挤在城墙下，他们都是从各地逃过来的平民，家园被破，他们已无家可归，如今幽州城就是他们唯一的希望。此时有人哀嚎，有人痛哭，有人手脚并用向城墙上攀爬。
<br/>　　“杀！”
<br/>　　杨龙停干净利落的吐出一字，所谓慈不带兵，义不行贾，这些人中间未必没有混着戎族，一旦被他们混进城中，里应外合，幽州城必定要被血屠。杨龙停是一员老将，半生行伍，知道军人只看重结果，人命只是数字。他从不和下层士兵有过多交往，因为只要是人，便必有感情，当你和他们有感情后，便不再忍心将他们送死。
<br/>　　杨龙停也知道城下难民的心情，他同样不忍心看着他们惨死。
<br/>　　所以他选择不看。
<br/>　　箭如雨般射下，有人死在弓箭下，临死前却一声惨叫都来不及发出。有人逆着箭雨冲锋，也有人只能无奈的离开城墙范围。
<br/>　　“你们这样贵人，高高在上，不敢去杀戎族，却只敢向我们放箭，你们……会有天来罚你们。”
<br/>　　北镇城，又是另外一番景象。
<br/>　　又是一番攻城和守城的对决，城主白沙鱼已经一日一夜未眠，城中每一块砖头，每一个人都被他发挥出作用。他吃住都在城墙上，与士兵共眠共战。
<br/>　　“我是北镇城主，脚下便是我所守之土，我在城在，誓和此城共存亡。”
<br/>　　“杀！”
<br/>　　疲惫的将士发出一声闷吼，早已准备好的滚木、檑石，金汤等物向城下攻城的戎族砸了下去。
<br/>　　戎族难以攻破。
<br/>　　不过，北镇城只是一个小小的意外，对戎族来说，最近的战场还是进行得很顺利。
<br/>　　连番破城，连番屠城，铁蹄所到之处，杀人如割草。近些日子，许多人刀上的血都来不及擦干，只是不停的杀人，杀人，杀人。
<br/>　　长久在青牛山受的憋屈，顿时一扫而空，他们又回到自己熟悉的领域。长刀所指，杀人如麻，铁蹄到处，望风而逃。他们是受长生天眷顾的儿郎，天生马背上的战士，他们要向天下人证明：戎族骑兵，天下无双并不是一句虚言。
<br/>　　那个讨厌的程大雷。
<br/>　　和程大雷相比，这些势力简直像乖巧的兔宝宝啊。
<br/>　　在烧杀抢掠一个月后，戎族放下北镇和幽州两城不管，收兵领大军直扑青牛山。
<br/>　　现在，该是去收拾那只讨厌的癞蛤蟆了。
<br/>　　……
<br/>　　蛤蟆寨。
<br/>　　程大雷终于修好了第一道山梯，接着绳索助力，他和山寨各家首领率先爬上了山头。
<br/>　　由上往下望，山谷内的情形一览无遗，程大雷忽然一怔，恍然想起些什么。
<br/>　　“XXX”程大雷愣了一会儿，破口大骂道：“特么的，我终于明白他们为何要死缠着我不放了。”
<br/>　　很久很久以前，程大雷从黄三元那里得到了一份藏宝图，据说这份藏宝图是三十年前，戎族侵略帝国时，所抢的财物来不及带走，所以就地埋了起来。
<br/>　　不过戎族连文字都没有，这藏宝图画得就格外粗燥了些，想要按图索骥寻找到藏宝地，几乎是不能做到的事情。
<br/>　　程大雷根本就没把这件事情放在心上，而此时此刻，站在程大雷的位置往下看，正和地图上描绘的一模一样。
<br/>　　戎族三十年前在帝国大肆劫掠，所抢到的金银珠宝有多少，根本难以想象。而这些财宝，现在就在山谷中，甚至程大雷直接可以确认它们的位置。
<br/>　　本来想到这里程大雷还是很激动的，但他愣了一会儿，轻轻叹了口气。
<br/>　　“可惜这次没时间把它们取出来了。”
