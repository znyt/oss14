　　第252章
<br/>　　程大雷拎着一柄大斧从天而降，弓步，弯腰，倾身，眼睛里像是藏着暴怒的狮子。
<br/>　　没有人怀疑，任何敢靠近的人都会被他一斧砍成两段。
<br/>　　气场是个很玄的东西，千军万马中一名小卒无足轻重，可有时候一个人就是千军万马。
<br/>　　乱军被压了压，主要是大家不知道他是干嘛的。马匪觉得程大雷比马匪还马匪，而程大雷这不要命的架势，又让鱼龙卫觉得程大雷是自己人。
<br/>　　于是乱战出现了一刻的安静，这安静或许只有一瞬，或许只有一息，但这一息一瞬给了程大雷机会。
<br/>　　他没有任何犹豫，从天而降，踹翻罗酒，紧跟着越上马车，一拉缰绳，重重喝了一声。
<br/>　　“驾！”
<br/>　　骏马扬蹄，不顾一切往前闯，程大雷一手拉着缰绳，一手斧头劈砍，臂展之内无人能靠近。
<br/>　　终于，双方都明白了程大雷的来意。
<br/>　　今日究竟是瓮中捉鳖还是守点待援，不，都不是，今天明明是单骑闯关。在虎和狼的对峙中，突然冒出一只蛤蟆，想要把虎狼口中的肉抢走。
<br/>　　“不要让他抢走公主！他是程大雷。”鱼龙卫中已经有人认出了程大雷。
<br/>　　“杀了他。”马匪头目大喝。
<br/>　　双方依旧在乱战，可也可各分出人马去阻击程大雷。
<br/>　　罗酒去的最快，他刚才将程大雷当作自己人，还特意去打招呼，可没想到却被程大雷一脚踹翻。
<br/>　　拎一条铁鞭跃到马车前，照着程大雷脑袋就劈了下来。
<br/>　　“狗贼，去死！”
<br/>　　程大雷架起斧头抡砸向对方，斧头和铁鞭在空中碰撞，各自手臂都是一震。
<br/>　　“此人好大的气力。”罗酒倒抽一口凉气。
<br/>　　突然，一名马匪用马刀攻向罗酒，罗酒一个俯身避过，紧跟着一鞭将对方从马上砸下来，立刻对方脑浆迸裂，便再也没有站起来。
<br/>　　三方乱战，鱼龙卫要保护公主撤离，马匪要抢走公主，双方都不想程大雷抢走公主。鱼龙卫和马匪的混战为程大雷分担了压力，不过他面前还是有重重精兵。
<br/>　　弯刀，长矛，兵器在黄日下闪着杀人的寒光。罗酒越过来，没有攻击程大雷，而是砸在马首上。
<br/>　　射人先射马，擒贼先擒王。
<br/>　　一匹枣红马吃痛，头骨已经被砸裂，它用最后的力气狂奔，冲锋的势头越来越弱。终于，前腿一软，如一座大山倒塌。
<br/>　　程大雷已经将李婉儿背在身上，在马车塌架的时刻，身子跃起，脚尖在某人头顶一点，轻盈飞起。
<br/>　　那一刻，众人都看见一支高高飞起的……癞蛤蟆。
<br/>　　跃出重围，程大雷脚尖刚落地便向前狂奔，此时的他已经辨不得东西，分不清左右，只是一力向前狂奔而已。
<br/>　　“追，把公主夺回来！”
<br/>　　宫符大喊，冷不防一柄刀向他脑袋砍来。罗酒出现，帮他挡住钢刀，宫符一身冷汗。
<br/>　　“抢公主，抢公主！”
<br/>　　马匪大吼，可鱼龙卫并不好惹，双方纠缠在一起，一时打得难解难分。
<br/>　　都想去追程大雷，可却又都腾不出手来，双方难解难分，一时都抽不出站团。所以只有乱战外围的一批人去追劫走公主的程大雷。
<br/>　　草原上一望无垠，极适合驰马狂奔，程大雷虽然仗着『热血』技能提供体力，强撑着冲出重围。但是，一旦时间拉长，程大雷这两条腿绝对跑不过四条腿。
<br/>　　程大雷心里想着，要不稍微放慢脚步，从身后追兵里抢一匹马来。
<br/>　　正此时，前方出现一个娃娃脸少年，他没有骑马，骑的是一头黑驴。
<br/>　　浑身油黑，只是头顶有一撮白毛，看上去俊逸非常。
<br/>　　他不是突然出现，他早已等在这里，似乎早已算定程大雷会从这里经过一般。
<br/>　　见到程大雷时，他手搁在剑柄上，嘴角依然带着笑意，可在衣服下，整个人已经是一张拉满的弓，只等放出那必杀一剑。
<br/>　　马贼鬼见愁，号称草原第一快剑，能切山崖上掉下的落石。
<br/>　　他候在这里，正琢磨着台词，是说一句：嘿，纵然你有飞天之能，也难逃我的手掌心。还是笑笑说：你好，多谢你把公主给我送来。
<br/>　　眼看着程大雷已经近了，他脸上的笑容更加灿烂。
<br/>　　“你好，……”
<br/>　　程大雷一边跑一边回头，正想着如何夺一匹马，猛抬头看向前方，面前正好有头毛驴，只是毛驴上有个少年。
<br/>　　程大雷拔剑，收剑，将李婉儿搁在驴背上，自己也翻身跃了上去，重重一拍驴脑袋。
<br/>　　“这驴真不错！”
<br/>　　“你怎么把人杀了，或者他根本不是马匪，就是个普通人。”李婉儿道。
<br/>　　“呃，我刚才杀了人么。”程大雷道：“刚才只注意这头毛驴了，不好意思，不好意思。”
<br/>　　李婉儿叹口气，虽然滥杀无辜终究不好，可是在逃命关头，也顾不得这么多了。只是刚才那少年笑时还蛮好看的，如此死了委实可惜。
<br/>　　娃娃脸少年倒在草地上，喉咙一道红线，然后血涌出，笑容永远僵硬在脸上。
<br/>　　马匪大军明显怔了怔，感觉就像是一道奔流而下的泥石流突然被冰冻住。
<br/>　　然后一片呼声响起，这呼声听上去既像是欢呼，也像是恐惧。
<br/>　　“大当家死了，大当家死了！”
<br/>　　“大当家死了，大当家死了！”
<br/>　　呼声铺天盖地，连绵成一片。
<br/>　　程大雷骑着毛驴，摸摸脑袋，嘴里道：“总感觉有什么大事在刚才发生，可却没有一点头绪。”
<br/>　　“现在他们都在追杀我们，这还不是大事么。”李婉儿在程大雷怀里道。
<br/>　　“喔，对对对，咱们快逃命去吧。”程大雷恍然大悟，却也不再想这件事。
<br/>　　鬼见愁，边城十三家马贼之首，仗着一手快剑和阴狠性格在草原上横行许久。凡是被他劫掠的商队，男人都惨遭屠杀，女人则无一例外的被折磨得精神崩溃。可是，怕他也没想到，最后自己会这么稀里糊涂的死。只能说，人生的大起大落实在太刺激了。
<br/>　　嘟，恭喜完成隐藏人物，诛杀马贼鬼见愁，奖励抽奖次数1 。
<br/>　　咦，逃命的程大雷发出一声惊叹：我的系统又活了。
