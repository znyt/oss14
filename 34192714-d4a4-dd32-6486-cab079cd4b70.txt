        “小沉不愿意查你身上的事情，但我想要一个真正的答案。你变得和婴宁如此相像，绝非是偶然。你身上一定有什么我们不知道的秘密。小沉想装糊涂，但是我可不想就这么糊涂下去。婴宁的事情，一直是我心里的一个结，就像小沉他父亲冷卫峰的死，是他的结一样。”冷逸峰铿锵有力地说。
<br/>
<br/>        童筱颖似懂非懂地反问：“所以，你这是要找出杀害婴宁的真凶，给婴宁报仇？”
<br/>
<br/>        “婴宁是我的妻子，我不能让她死得不明不白。”冷逸峰微眯起黑眸，语气沉重地说道。
<br/>
<br/>        童筱颖却怒意横生：“你要报仇，那是你的事情，你为什么要拖我和大哥下水？”
<br/>
<br/>        “我的能力有限，而你是小沉的软肋！”冷逸峰勾了勾唇。
<br/>
<br/>        童筱颖只能懊恼地抿着嘴，不想再跟这个男人说一句话。
<br/>
<br/>        傍晚时分。
<br/>
<br/>        季雅珊见冷逸峰一个人回来了，于是关心地问道：“筱颖呢？”
<br/>
<br/>        “被我关起来了。”冷逸峰面无表情地回答。
<br/>
<br/>        季雅珊顿时惊愕道：“关、关起来呢？为什么啊？你为什么要关她？”
<br/>
<br/>        “你打电话通知傅夜沉，让他来一趟这里。”冷逸峰冷冷地命令道。
<br/>
<br/>        季雅珊试图在冷逸峰的身上探求原因：“你是打算让筱颖当婴宁的替身吗？”
<br/>
<br/>        “你觉得，我是那种乱伦的人？”冷逸峰眼神冷淡地瞥了一眼站在一旁的季雅珊。
<br/>
<br/>        再怎么说，童筱颖是傅夜沉的合法妻子，也就是他冷逸峰的堂侄媳，他不至于要去跟自己的堂侄抢女人。
<br/>
<br/>        “好，你说什么就是什么。我都听你的。”季雅珊不再追问，而是聋拉着脑袋，黯然神伤地说道。
<br/>
<br/>        这世上，最安全的地方，就是最危险的地方。
<br/>
<br/>        童筱颖也没想到，冷逸峰竟然会把她交给大哥傅夜沉的弟弟傅昼景。
<br/>
<br/>        滨江城的某高档别墅小区里。
<br/>
<br/>        童筱颖被人摘下眼罩后，看到面前的这个男人，有些傻眼。
<br/>
<br/>        傅昼景则十分淡定地她松绑，并给她端了来一杯水，安慰道：“喝口水。”
<br/>
<br/>        “你应该帮大哥才对，怎么就帮冷逸峰了？”童筱颖没好气地说。
<br/>
<br/>        傅家的这些人际关系还真是复杂。
<br/>
<br/>        傅昼景搬了条椅子，在童筱颖的面前坐下，咧嘴道：“我和大哥的关系不就是因为你才破裂的吗？”
<br/>
<br/>        童筱颖一脸无奈道：“我和你之间的事情，大哥全都告诉我了。但是，在我看来，那都是过去的事情了。更何况，我都忘了。所以，你放我走，可以吗？”
<br/>
<br/>        “我们曾经那么相爱，我不相信，你说忘就忘了。”傅昼景不以为然地回答。
<br/>
<br/>        童筱颖懒得跟他理论，直接起身往大门口走去。
<br/>
<br/>        当她打开大门，正准备踏出大门一步时，忽然“嗖”的一声，不知从哪儿冒出来一颗子弹，直接打在了她脚边的地面上，子弹壳弹出去了好几米远。
<br/>
<br/>        “这是给你的警告。”傅昼景淡定地从椅子上起身，转身看向童筱颖，接着说道，“别墅四周，十爷安排了狙击手。只要你敢踏出这里一步，他们便会从你的腿开始射击。你若不想死，就安安分分地待在这里。”
<br/>
<br/>        “……”童筱颖无语。
<br/>
<br/>        不过，傅昼景的话，却成功逼退了童筱颖想要出去的想法。
<br/>
<br/>        “明明是你负我在先，为什么你不知反省，却一而再再而三地不肯放过我？”童筱颖返回到傅昼景的面前，对上傅昼景的目光，冷冷地质问。
<br/>
<br/>        傅昼景顿时一手握拳，振振有词地反驳：“没错！我是对不起你！但是，我也只是身体上对不起你而已。我的一颗心，一直都在你这里。而你呢？你身心都给了我大哥，到底是谁负了谁？”
<br/>
<br/>        “你负我在先，我和你已经离婚，我再转嫁给谁，跟谁在一起，都是我的自由，跟你一点关系也没有。此外，我觉得，我跟你的思想观念真的不在同一个世界里。”童筱颖有些不耐烦地皱起了眉头。
<br/>
<br/>        这世上，怎么会有这样的不可理喻的男人呢？
<br/>
<br/>        出轨了还有理？！
<br/>
<br/>        “筱颍，我可以发誓，我至今都还爱着你。那么你呢？”傅昼景伸手将童筱颖拉到自己的跟前，信誓旦旦地说道。
<br/>
<br/>        童筱颖顿时觉得他的这句话极其的可笑：“你现在对我来说，只是我夫家的亲戚而已。”
<br/>
<br/>        “夫家的亲戚？呵！你还真当自己是傅家大少奶奶了吗？如果，我现在把你办了，然后再拍几张照片公布出去。你说，大哥会怎么看待你？”傅昼景的脸上，突然露出了阴冷又阴森的笑容。
