　　其实，当慕清霄听到‘阴阳’两字的时候，心中就已经掀起一阵惊涛骇浪，但表面上却不动声色。
<br/>　　他不清楚，自己身上的秘密，究竟让对方看出了多少。
<br/>　　“生与死之间，它会对你有所帮助。”沙哑的身影再度响起，没有丝毫的感情波动。
<br/>　　莫名其妙的话语，令慕清霄举棋不定。
<br/>　　眼前有一个素不相识的人，给一块奇怪的令牌，说了一堆莫名其妙的话，慕清霄不知道该不该收下令牌。
<br/>　　任谁遇到这种问题，估计都会再三斟酌。
<br/>　　最让慕清霄无奈的是，空间被封锁，他想走的走不了，似乎是后者故意为之，为的就是让他将令牌收下。
<br/>　　接过令牌，慕清霄只是稍微打量了一眼就收回目光，令牌巴掌大小，看不出是何材质，上面纹路交错，有些神秘。
<br/>　　见慕清霄接过令牌，被封锁的空间在下一秒就解开。
<br/>　　瞥了一眼黑袍人，慕清霄欲言又止，最终将到嘴的话语咽了下去，将令牌收进储存空间后，便转身登岸。
<br/>　　来到陆地上，天鬼皇就走了上来，眉宇皱在一起，眼中满是不可思议：“你与引路人说话了？”
<br/>　　“有些古怪，但愿不会惹祸上身吧。”
<br/>　　想起引路人刚才说的几句莫名其妙的话语，慕清霄不禁摇了摇头，回头一看，哪里还有引路人的影子。
<br/>　　“走吧，带我去极乐世界入口。”
<br/>　　天鬼皇不再多想，严肃的点头道：“本座不能现身，只能带你到入口，聚时就在这岸边等你，火鬼王就只能由阁下自己对付了。”
<br/>　　“无妨。”
<br/>　　说罢，慕清霄瞥了眼冥河，道：“引路人走了，我们如何回去？”
<br/>　　“阁下不需要担心，只要有人想要渡河，引路人就会自动驾船出现，所以不需要担心无法回去的问题。”
<br/>　　慕清霄瞥了眼平静的冥河，若有所思，处处透着神秘的引路人，难道与冥河教祖有关？
<br/>　　将如此不切实际的想法抛之脑后，仙剑三位面虽是高级仙侠位面，但体系依旧不算高，又怎会出现那种层次的人物。
<br/>　　不再胡思乱想，周围空间轻微波动了一下，慕清霄的身影就消失不见，天鬼皇紧随其后。
<br/>　　在慕清霄与天鬼皇离开不久，不知何时，原本消失的冥舟与黑袍人却再度出现，沙哑的声音自黑袍中传出。
<br/>　　“唯一的契机，没想到会在此处等到，难道真是冥冥中自有定数？”
<br/>　　……
<br/>　　……
<br/>　　与此同时，慕清霄与天鬼皇的身影已经出现在一座偌大的火山前，山体中央印着四个古篆字体‘极乐世界’。
<br/>　　天鬼皇的身影，则隐匿于虚空当中。
<br/>　　“火山口便是极乐世界的入口，本座只能送阁下到这里，岩浆屏障本座也过不去，在进一步，可能就会被火鬼王察觉到。”
<br/>　　说罢，天鬼皇就悄无声息的退去。
<br/>　　等天鬼皇离开极乐世界范围，慕清霄视线落在火山上，他本就没指望天鬼皇进去。
<br/>　　而且，区区火鬼王，他根本就没放在眼里。
<br/>　　身躯一动，慕清霄已经出现在火山口。
<br/>　　从上往下望去，能看到火山中热浪滚滚，巨大的气泡自岩浆中冒出，最后炸裂开来，一股炙热的气浪伴随着白烟升腾而起。
<br/>　　慕清霄已经不是第一次遇到这种情况。
<br/>　　不管是在风云位面中的凌云窟，雷峰塔，或者是在斗破苍穹位面中的迦南学院里，他都遇见过相似的情况。
<br/>　　岩浆的温度很高，极乐世界在岩浆屏障的后面，其中肯定深浅难测，否则鬼王们不至于无法入侵。
<br/>　　慕清霄脸色轻松，纵身一跃，便掠入火山中，伴随着噗通一声落水声，映入眼帘的就已经是一片赤红色的世界。
<br/>　　感受到周围粘稠的岩浆，慕清霄面色一变。
<br/>　　好高的温度，好浓郁的火属性能量！
<br/>　　刚进入岩浆中，慕清霄就能感觉出，岩浆中蕴含的火属性能量，简直高的没边，根本就不是风云位面，斗破位面的岩浆领域能比的。
<br/>　　恐怖的温度，堪比异火！
<br/>　　“难怪了，看来火鬼王也不笨，知道将火灵珠中的火属性能量引入岩浆中，形成天然的屏障。”
<br/>　　若非慕清霄实力已经是大乘期，甚至黑凤血脉加持，不惧怕火属性的东西，恐怕对这岩浆领域还真没办法。
<br/>　　除非，将岩浆中的精纯能量全部吸收，让沸腾的岩浆平息下来。
<br/>　　岩浆中的能量，倒是可以修炼，但慕清霄不准备现在就动手，现在最主要的还是获得火灵珠。
<br/>　　岩浆世界中的火属性能量，相比火灵珠中庞大且精纯的火属性能量，简直是小巫见大巫，根本没有可比性……
<br/>　　深入塔底后，映入眼帘就是一片赤红的岩浆世界，其中充斥着死寂，没有一丝的生命痕迹，无边无际，一眼望不到尽头。
<br/>　　“咕噜……”
<br/>　　死寂的岩浆世界，空间轻微一阵波动，岩浆气泡升腾而起，慕清霄的身影已经不知所踪。
<br/>　　“真是够深的，已经有六千多米了吧，难怪除了火鬼，其他鬼族都讨厌这里。”
<br/>　　顷刻间，慕清霄就出现在地底六千多米的地方，打量了一眼四周，由此可见，火鬼界估计是建立在了地底很深的地方。
<br/>　　慕清霄的速度很快，不多时，身影就在岩浆中顿住。
<br/>　　在他的前面依旧是赤红的岩浆，不过，这只是表面现象：“看来，通过空间结界，就能抵达火鬼界了。”
<br/>　　伸手触碰了一下前面的岩浆，果不其然，前面的空间如同水面一样，荡起一阵阵涟漪。
<br/>　　火鬼王的修为相当于合道期的鬼修，实力可能不如天鬼皇等鬼王，但手中有火灵珠，加以善用，确实是棘手的东西。
<br/>　　慕清霄随手一挥，岩浆世界中就出现一条空间裂缝。
<br/>　　双手往两边拨开，裂缝就渐渐扩大，当裂缝能容纳一人通过的时候，慕清霄就已经掠入其中，转眼就消失不见。
<br/>　　与此同时，在空间结界的内部，空间裂缝再现，慕清霄自其中踏出，感受到实质性的地面，脸上牵起一抹弧度。
<br/>　　踏出空间门，慕清霄视线在周围扫视一圈，眼前的场景与岩浆世界截然不同。
<br/>　　……
