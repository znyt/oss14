　　第422章
<br/>　　林少羽真切的感受到『气数』这个词的含义。
<br/>　　帝国气数已尽，林少羽真感觉天都在帮自己。从幽州一路打到长安，几乎没有遇到什么挑战，甚至攻破天下第一城的长安，都算不上什么挑战。
<br/>　　那是因为天命已经抛弃了帝国，可在攻破长安后，天命也继而抛弃了正义教。
<br/>　　长安城一战，正义教损失了太多人手。事后，郭烦人的尸体在皇宫发现，而小道士红尘了却神秘失踪。至于他去了哪里，其实无人知道，很大可能他也死在了混战中。
<br/>　　攻破长安城后，正义教在城内烧杀抢掠，所犯下的罪行与戎族屠城相比，有过之而无不及。对于这种事，林少羽不是不想管，而是管不了。
<br/>　　正义教的组成，大多是帝国最底层的一拨人。他们终日劳作，以期待换来微薄的衣食，在潜意识里，他们觉得整个世界都欠了他们。
<br/>　　一旦有机会，他们便拼命索偿。
<br/>　　财主的金库被砸开，女人在暗室被糟蹋，幼童被杀戮在长街，正义教占领长安城的日子里，长安成了一座炼狱。
<br/>　　而于此同时，原先选择袖手旁观的各路诸侯，在正义教攻破长安城后，瞬间变得激动起来。各路诸侯联手攻打正义教，在这种攻势下，正义教连十天都未撑过。林少羽带着正义教的残余人手，从北门逃走，打算逃回幽州大本营。
<br/>　　林少羽杀死明帝后，帝国就陷入混乱之中。天下诸侯都像是被打了鸡血一样，全部亢奋激动起来。
<br/>　　国不可一日无君，事实上，帝国不是没有皇帝，而是有很多皇帝。
<br/>　　在这段时间内，有太多人宣布称帝。他们占据着穷乡僻壤，手下大概有几千人，就宣布登基称帝。具体数目不详，反正至少有几十人。
<br/>　　他们今天称帝，明天可能就被灭国，在生命最后时间里，算是过了皇帝的瘾。
<br/>　　真正的过把瘾就死。
<br/>　　枪打出头鸟，这些人终究是乱世之火腾上空中的余烬，风一吹火就灭了。而真正的聪明人，却都明白闷头发大财的道理，就算想要称帝，也得找个名正言顺的理由。
<br/>　　扬州牧李星打出了替皇兄复仇的理由，手下聚拢了一帮门客。
<br/>　　荆州、益州、交州……帝国十三州一百单八城，各路诸侯割据，天下烽烟四起。
<br/>　　情况最复杂的还要数长安附近区域。长安从古至今就是天子脚下，这里聚集了大量财阀、名门、世家。在这个乱世，他们展现了庞大的家族底蕴，招兵买马，蓄积势力。而明帝留下的皇子们，成了这些人哄抢的香饽饽。
<br/>　　李乐天在大将军尉迟离的护送下逃离长安，后在并州落脚。等正义教被赶跑后，他们又返回长安，持天子剑，要继承帝位。尉迟离想要得到士族的支持。然而，对持着天子剑的李乐天，大家都说再议，再议……
<br/>　　接着，长安城附近便开始有关于李乐天的流言。说他守城无力，是个酒囊饭袋，长安城被攻破，他难辞其咎。甚至，明帝的死，他也脱不了干系。为何他不知保护好明帝，为何他要让正义教的逆匪打破长安城。
<br/>　　民怨总是要有发泄口，短时间内，李乐天的名声就臭了大街。
<br/>　　也就在这种情况下，李行哉赶到了长安城，而此时，长安城觊觎那张龙椅的人，已经不下七位。
<br/>　　……
<br/>　　凉州，蛤蟆城。
<br/>　　“在今天这个重要的日子，大家聚集在这里，为了我们共同的理想。希望在今天，大家拿出干劲，拿出勇气，秉持友谊第一，比赛第二的精神，追求更高、更快、更强……”
<br/>　　程大雷站在点将台上，口若悬河，而在点将台下挤满了人。今天的蛤蟆城很热闹，锣鼓喧天，鞭炮齐鸣，城外的平民也涌入城内，蛤蟆城在这些天人山人海。
<br/>　　天下大乱，帝国九万里江山刀兵四起，在大家打得如火如荼的情况下，程大雷主持召开了蛤蟆城第一届比武大会。
<br/>　　说是比武大会，但这次比赛比的不仅仅是擂台上的比武较量，全部比赛有二十几个项目。包括：骑术、射箭、游泳、划船、围棋、术算等等，甚至还有耕田。比单位时间内，谁耕的田最多。
<br/>　　程大雷搞出的旗号是，人人参与，人人有奖。哪怕你只是翻跟头翻得好，也有可能得个翻跟头第一名。
<br/>　　这个冬天还未过去，尚属于农闲的时候。程大雷之所以兴师动众，大张旗鼓搞这场比武大会，并不是没有目的。
<br/>　　如今帝国到处都在打仗，今天你打了我，明天我再打过去，长安城的情形更是严峻。在如此情况下，蛤蟆城一帮大老粗也是人人思战。
<br/>　　不用提其他人，就连程大雷自己，在夜深人静时也是碾转反侧，心潮澎湃。
<br/>　　天下英雄齐逐鹿，有志男儿谁又只愿当个旁观者。
<br/>　　然而，程大雷又清醒的明白。如今的情况，一动不如一静。现在只是乱世的初级阶段，小鱼吃虾米，虾米吃污泥，吃到最后，小鱼成了大鱼，然后鱼跃龙门，蜕壳化龙。
<br/>　　这场逐鹿比赛，是场零和游戏，赢家拿走全部，输家丢弃一切，包括生命。
<br/>　　蛤蟆城这三万兵是程大雷辛苦拉起来的，把这三万兵丢进这场游戏中，估计一个水花都砸不起来。况且，程大雷也不愿意拿别人的命，成全自己的野心。
<br/>　　所以，程大雷搞了这场比武大会，在蛤蟆城搞得如火如荼。
<br/>　　最后，樊梨花得了骑术第一，柳芷得了绣花第一，白元飞得了赛跑第一，秦蛮得了擂台赛第一。令程大雷没有想到的是，李婉儿竟然击败和珅，得了术算第一。
<br/>　　程大雷对每个项目的前三名都颁发了奖品，不得不说，程大雷这么搞还是蛮有作用的。大家都很激动，一腔热血浇在比赛上，暂时把其他事都忘了。这段日子，蛤蟆城上下讨论的都是比赛的事情。
<br/>　　然而，程大雷浇得灭其他人的热血，却熄灭不了自己心头的火。
<br/>　　眼望东南，东南是长安。
<br/>　　他也想去长安，斩十万匹马呐。
