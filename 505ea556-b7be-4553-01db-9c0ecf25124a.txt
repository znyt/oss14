　　第112章
<br/>　　渔阳楼，据说是千舟城最好的客栈。
<br/>　　南来北往的客商在千舟城做生意，往往都选择在这里落脚。一来是这里房间舒适，饭菜可口，服务也是极周到。客人旅途寂寞，想要城内青楼女子陪宿，只要银子给到了，客栈也能满足；二来，这里聚集了太多商人，有北地过来运皮货的，也有南地的茶商，甚至也有飘扬过海，将异域货物贩卖到此地的南洋客商。
<br/>　　另外，住在这里也是某种身份的象征，生意人总要讲究个体面。
<br/>　　掌柜姓牛，名唤牛本善，是千舟城的场面人物，和许多达官贵人都有来往。
<br/>　　今日黄昏时分，牛本善迎来一拨客人，听口音是北地来的。为首的派头大，脾气也很大。穿着是有些不伦不类，不过却贴身带着两个小妾。虽然都是男装打扮，但牛本善迎来送往，练就一双好眼力，自然一眼就看穿了。这种事情也常见，来往的客商都带着中意的妻妾，有些在扬州做生意的几天，也会雇某位青楼女子陪侍，甚至在本地都有外室。
<br/>　　令牛本善吃惊的是这两个小妾的模样，即便放在美女如云的扬州城也是鹤立鸡群，牛本善觉得，就算在宫里陪皇帝的女人也不过如此了。
<br/>　　看一个男人的本事，就看他身边的女人，这句话无论在那个朝代，总是有几分道理的。
<br/>　　如此，牛本善对几人说话也恭敬起来。
<br/>　　“诸位客官一路辛苦，欢迎来本店歇脚，本店有七处小院，目前丁字号还空着，诸位今晚就在丁字号歇息怎样？”
<br/>　　“让你们伺候的人都出去，饭菜每天送来就好。”
<br/>　　“自然，自然，他们笨手笨脚怎么照顾得好几位尊贵的客人。”牛本善满脸堆笑，忽然想起一事：“诸位，今晚甲字号的客人会办一场辩宝大会，只要是本店入住的客人都可以参加，几位若有兴趣，晚上可以去看看。”
<br/>　　『辩宝大会』在渔阳楼倒是经常出现，东南西北四方的客人聚集在此地，兼之这个时代信息不通畅，有些货物无人识得的情况也是有的。所谓的辩宝大会，便是某个有财力，有面子的商人出面，把大家聚集在一起，品茶论道。聊聊年景如何，各自都带着什么货物。
<br/>　　也是一个交流信息，扩展人脉的场所。
<br/>　　会上各人的地位自然就是有所带货物的贵贱决定。
<br/>　　『啊，阁下带着十斗夜明珠，啧啧，我观赵兄的模样与过世的家父有八九分相似，今日我就斗胆，认你当干爹了』
<br/>　　『来呀把那个卖炊饼的赶出去，等等，他有个妻子姓潘，等等，这件事还能再聊』
<br/>　　徐神机几人都将目光看向程大雷，程大雷略沉吟片刻，道：“辩宝大会么，有时间我会去看看的，不过今晚估计会很忙，看时间吧。”
<br/>　　说着，有伙计带着他们来到丁字号，这是一座独立的院落，和前生的五星级酒店自然不能相提并论，胜在地方大，特别大。
<br/>　　程大雷一行人住进去，也是半点问题没有。
<br/>　　刚歇脚，程大雷就把徐神机和赵子龙发出去打探消息。渔阳楼住着不少客商，打探消息也灵便，现在程大雷对千舟城的情况是两眼一摸黑，急需知道外面的情况。张肥，高飞豹几个不甘寂寞，也想出去逛逛。
<br/>　　对于他们程大雷是不太放心的，口中道：“你们出去是出去，但不能离开客栈，别在外面生事。”
<br/>　　“我们再生事，难道还有大当家能惹事么……”高飞豹。
<br/>　　“……”程大雷。
<br/>　　最后，张肥、高飞豹、小白狼兴致勃勃的出去了，院子里只留有程大雷、苏樱、柳芷三人。
<br/>　　程大雷进了屋子，北地睡炕，南地卧床，两地有很大差异。
<br/>　　程大雷走进屋子后，长长松了口气，把脖子上的貂绒解了，老实说一直戴着这玩意，还是蛮热的。
<br/>　　“爷，您先洗洗脸吧，我伺候着您歇一会。”柳芷用铜盆端着井水进来。
<br/>　　程大雷的确有些累了，这种累不仅是身体上的，更多是精神方面的压力。
<br/>　　“你这一路肯定累坏了，我刚打的水……”
<br/>　　正这个时候，苏樱用铜盆端着井水进来，见到柳芷拿着毛巾立在程大雷身边，双方目光都愣住了。
<br/>　　“怎么了？”程大雷眨眨眼睛问：“你打水干什么？”
<br/>　　“没什么，我喝。”
<br/>　　“这么大水盆，你肚量好大。”
<br/>　　……
<br/>　　渔阳楼的晚上总是很热闹，来此地做生意的客商聚在一起，交流生意上的消息，也有文人骚客聚在一起，以酒助兴，谈些诗词歌赋。
<br/>　　渔阳楼二楼，今日正有一帮年轻人之间的雅集，都是二三十岁的年纪，风华正茂，有正值妙龄的青楼女子，捧着琵琶，瑶琴，浅吟低唱。
<br/>　　风流才子，风流才子，不风流能叫才子么。酒过三巡，菜过五味，几人都齐了诗兴。
<br/>　　“妙，婉儿小姐一舞倾城，不才刚才偶得一首，为几位助兴。”
<br/>　　“此时正值冬日，不如我们就以冬为题……”
<br/>　　所谓文人雅集，无非就是一些文人骚客聚在一起，聊聊茶道，书画，围棋，诗词歌赋……
<br/>　　此时一行人正在尽兴，就见楼梯口鬼鬼祟祟冒出两个脑袋。此二人不是旁个，正是张肥和高飞豹两个。
<br/>　　所谓千古文人侠客梦，放在张肥身上却正好相反，他糙汉的外表下，有颗文艺青年的心。高飞豹的情形也差不多，山贼的职业，体内也住着个湿漉漉的骚年。
<br/>　　二人碰在一起，可以说是臭味相投，不过，以文化程度上来说，高飞豹是不如张三爷的。
<br/>　　二人在客栈转了一圈，慢慢就混到二楼。从二人内心出发，对这种文人雅集还是很向往的。
<br/>　　“什么人！”名叫唐如才的年轻人发现了两人，道：“你们是干嘛的？”
<br/>　　被发现之后，张肥和高飞豹都有些尴尬。张肥整整衣衫，近前一步道：“兄台勿怪，我们也是本店的客人，听得你们在这里聚会，特来拜访，大家以诗会友。”
<br/>　　“你长这个模样……”唐如才道：“你说这个话自己信么？”
<br/>　　
<br/>(本章完)
