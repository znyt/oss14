　　时间一晃到了五天之后，试射场还是原来的试射场，只不过这一次来观看试验的多了几个老头子，比如李二陛下，比如秦琼，比如老尉迟。
<br/>　　李承乾陪在几个老头儿的身边，唐善识亦如此，只不过相比于前几日，这次的小唐显的有忐忑，生怕再出现上一次的问题。
<br/>　　“太子，怎么还不开始啊？”李二陛下等了好久都不见外面有动静，便舍了正在与自己聊天的秦琼，对李承乾问道。
<br/>　　“父皇还请稍等，很快外面就布置好了。”李承乾微微弯腰，俯身在老头子耳朵说道。
<br/>　　“嗯，快一些。”老头子点点头。
<br/>　　这一次的试验听说有好几种新式武器，还有能打出六千米的贞观炮，虽然老头子对此表示怀疑，但最终还是按捺不住好奇心，亲自跑来观看试验了。
<br/>　　又过得看刻，就在老头子等的有些不耐，准备再次发问的时候，混疑土防炮洞的大门被人敲响，一个声音在外面喊道：“报告，试射准备完毕！”
<br/>　　“让他们开始吧。”李承乾等的就是这句话，给小唐使了个眼色之后说道。
<br/>　　……
<br/>　　不得不说，此时的大唐军队已经与以前完全不同了，一身接近现代的迷彩军服将一个个士兵武装的英武非常，虽然身上没有护甲，但依旧让人感受到那一身凌厉的煞气。
<br/>　　李二陛下和几个老头子每人占据一个观察孔，向外面看着，口中时不时的发出一声评论，使李承乾的脸上不自觉得露出那么一丝傲然的微笑。
<br/>　　虽然每一个款式的军服都是他从后世抄来的，但在大唐谁又能知道呢，左右就当自己才是原创好了。
<br/>　　“哎哎，来了来了，你们看那个小圆筒，那个就是迫击炮。”唐俭这老货现在可是得意非常，毕竟这里所有的武器都是出自他儿子的手笔，若说这老头儿不高兴那才是假的。
<br/>　　什么？你说老唐的儿子成了工匠头子，老唐怎么可能高兴？
<br/>　　这可就大错特错了，要知道大唐现在的工匠和以前可是完全比不得的，工业化发展到了今天，可以说什么事情都离不开工匠。
<br/>　　而且工匠也要分干什么的工匠，铁匠什么的自然不用说，还是几样，地位有所提高也没有提高太多。
<br/>　　但是像小唐这样的，据说可以用工程师来称呼，也就是说小唐算是工匠中的领导者，已经脱离了低级工匠的范围，上升到了老师的程度。
<br/>　　更何况小唐主抓的工程现在可都是大唐最先进的东西，像什么铁甲舰，火炮之类。
<br/>　　就凭借这些东西，小唐想要混个公爵什么的一点都不困难。
<br/>　　另外就是小唐与李承乾的关系从眼下来看完全就是这位太子殿下嫡系中的嫡系，也就是说现在就算是老唐立马死了，也不用担心唐家的未来。
<br/>　　所以老唐此时可是开心的不得了，有段时间甚至逮谁跟谁提五儿子怎么怎么有出息，听的一帮老家伙耳朵都快要出茧子了。
<br/>　　所以老唐尽管显的很兴奋，但是却没有人理他，听到他说话的众人无不对其投以鄙视的目光，包括李二在内，弄的唐善识都跟着尴尬不已。
<br/>　　……
<br/>　　迫击炮很快就在射手的组装下弄好了，一组十发的炮弹箱也被摆到了炮位上。
<br/>　　接着也不见什么准备不准备，一个炮手直接从箱子里拿出一颗样子十分奇怪，带有尾翼的炮弹就塞进了炮筒里面。
<br/>　　手一松，“嗵”的一声，炮弹便飞了出去。
<br/>　　只是与上一次不同的是，这次炮弹飞出去之后，一阵尖啸声立刻传进了众人的耳中。
<br/>　　“轰……”啸音渐渐远去，远处的靶场后部升起一团烟雾。
<br/>　　“打歪了？！”刚刚吹过牛的老唐飞快的看了小唐一眼。
<br/>　　“呃，父亲，那是试射，用来调整角度的。”小唐连忙解释。
<br/>　　这个时候如果影响了老头子炫耀的心情，就算是已经结了婚，儿子都已经三岁，小唐也依旧逃不了被抽的命运。
<br/>　　“哼！下次打准点。”果然，老唐的脸色有些不好看，但看在小唐的解释还算过得去，却也没有多说什么。
<br/>　　“快点看，又一发，这速度还真快。”就在唐家父子对话的空当，老唐身边的一个老家伙捅了他一下。
<br/>　　而等唐俭再把眼睛凑到望远镜上的时候，耳中已经再次听到了‘嗵’的一声，接着便又是一阵刺耳尖啸声。
<br/>　　“轰……”这一次，炮弹不偏不倚的正好砸到了它应该去的位置。
<br/>　　不过尽管如此，却没有任何一个老家伙开口。
<br/>　　从侧面看去，所有的老头子都把望远镜死死的顶在自己眼睛上面，嘴巴张的老大，尤其是老尉迟，甚至连口水都流出来了。
<br/>　　良久之后，终于有老家伙回过神来，喃喃说道：“这，这威力也太大了吧？这，这到底是什么东西？”
<br/>　　正所谓不看不知道，一看吓一跳，靶场上那些按照以往大唐军阵摆放的草人已经在爆炸过后缺了好大一片，目测大概有四、五十个草人被炸碎。
<br/>　　这已经不是老家伙们概念中的炮了，似乎除了神罚之外，老头子们想不到还有什么比这更加有威力的攻击。
<br/>　　所以接下来的八次试射老头子们几乎都没怎么看，因为他们不用看也知道，那些草人一定逃不过被炸成碎片的下场。
<br/>　　……
<br/>　　但是，留给老头子们的惊讶显然不止于此。
<br/>　　当能最大射程达到六千步的火炮被拖上来之后，老家伙们的心中只有一个念头。
<br/>　　不是自己疯了，就是设计这个炮的人疯了。
<br/>　　刚刚小臂粗半尺长的炮弹已经可以炸飞几十个草人，现在这个比大腿还粗，比大腿还长的炮弹这特么是要干什么？这是用来打人的么？什么人能够得了这东西！
<br/>　　就算是这东西不会爆炸，特么落到人的头上也能把人砸成肉泥吧？
<br/>　　而且最最重要的是，这炮的发射程序竟然是特么用大锤来砸的，就跟以前的投石机一样，这也太奇葩了吧？与那个炮身的高大上完全不配套啊。
