        连星儿却毫不在意地向童筱颖挥手告别，说了句“妈咪，拜拜”后，便转身出了门。
<br/>
<br/>        当她赶到网咖的时候，明司寒看着她的样子，有些忍俊不禁。
<br/>
<br/>        这丫头今天是中邪了吗？
<br/>
<br/>        连星儿的这张脸本来就长得稚嫩，可偏偏她今天却化了个浓妆。
<br/>
<br/>        她的眼睛本来就大，睫毛原本就长，竟然还要打眼影带假睫毛。
<br/>
<br/>        不仅如此，连星儿双颊的腮红和唇瓣上的蜜膏，看上去虽然很妩媚诱人，但是跟她这张透着青春气息的甜美脸蛋极其不搭。
<br/>
<br/>        “你……这是来才参加选秀呢？”明司寒忍不住调侃道，真不知道这丫头在想什么。
<br/>
<br/>        连星儿一听他这话，先是懵了一下，反应过来后，拎起手上的手提包直接往明司寒胸膛上一捶：“你才是来选秀的了！”
<br/>
<br/>        “那你没事化什么妆？还化得这么浓！丑死了！”明司寒一边笑着，一边伸手捏了捏连星儿的脸颊。
<br/>
<br/>        好在，她用的化妆品都是顶级的。
<br/>
<br/>        他捏完她的脸后，指腹上干干净净，她脸上的妆也没花掉。
<br/>
<br/>        连星儿却一脸不高兴地瞪着明司寒：“你才丑！哼！”
<br/>
<br/>        她说完，转身就走，明司寒见状立即上前，抓住了她的臂弯，将她给拉了回来。
<br/>
<br/>        “你去哪儿？”明司寒问道。
<br/>
<br/>        连星儿没好气地回答：“我就去对面那家美容店，把妆卸了。”
<br/>
<br/>        “你还真卸妆啊！我跟你开玩笑的！”明司寒笑道，“好啦！不逗你了！我们去打游戏吧！”
<br/>
<br/>        “可我突然觉得，脸上化着妆不舒服了。”连星儿实话实说，早知如此，她就不化妆了。
<br/>
<br/>        明司寒便不再阻拦，并松了手：“我陪你一起去。”
<br/>
<br/>        连星儿不禁对他翻了个白眼：“你一个大男生，进什么美容店？”
<br/>
<br/>        她说完，便只身离开了网咖，过了马路后，直接进了对面那家美容店。
<br/>
<br/>        明司寒站在网咖门口，一直看着她安然无恙地进了美容店后，才转身去了自己的包厢里等她回来。
<br/>
<br/>        连星儿刚进美容店时，里面的店员将她上上下下打量了一番后，女店长立马过来大献殷勤。
<br/>
<br/>        当得知是进来卸妆的后，女店长这变脸的速度比翻书还快，于是随手招来一个女服务员过来替她卸妆。
<br/>
<br/>        连星儿卸了妆后，女服务员推荐的产品她一样也没买，而是直接去了前台结账。
<br/>
<br/>        就在她刚推开美容店的玻璃门，走下台阶时，“嘭”地一声，枪声响起，路边倏然一阵混乱不堪。
<br/>
<br/>        “嘭——”
<br/>
<br/>        又是一声枪声。
<br/>
<br/>        街上的行人纷纷抱头乱窜，时不时地传来女人的尖叫。
<br/>
<br/>        连星儿惊怔地扫视着乱窜的人群，目光的焦点终于落在了左边一个在人群里奔跑的戴着口罩，留着平头，身着灰色格子衫，一手抱着一个黑皮箱子，一手持着手枪在胡乱开枪的男子身上。
<br/>
<br/>        人群里的尖叫声不绝于耳，再这么下去，一定会有更多的人受伤。
<br/>
<br/>        眼瞧着那个男子向自己这边跑过来，连星儿蓄势待发，找准机会后，待那男子从她身前经过时，一脚踢了上去，直接踢中了他的胳膊。
<br/>
<br/>        那把黑色的手枪从那男子手中腾空飞出，在空中划出一个抛物线后，“啪嗒”一声落地滑行了一段距离后，被一只穿着黑色马丁靴的脚给踩住。
<br/>
<br/>        男子被连星儿这行为惹恼后，二话不说便向连星儿冲了过来。
<br/>
<br/>        连星儿本能地后退，岂料脚跟撞到台阶上，打了个趔趄。
<br/>
<br/>        男子疯了似的，扑上来后，直接伸手掐住了连星儿的脖子，他那布满血丝的眼睛，恶狠狠地瞪着连星儿，嘴里咆哮般地呵斥道：“你找死是不是？好！老子现在就掐死你！”
<br/>
<br/>        连星儿只觉自己快要被这男子掐得窒息的时候，耳畔再次响起一道枪声。
<br/>
<br/>        顿时，鲜血溅了她一脸。
<br/>
<br/>        “啊——”男子痛得松了手。
<br/>
<br/>        连星儿怔愣地看着男子，只见他刚刚掐住她的那只手的臂弯处正鲜血直流。
<br/>
<br/>        而此刻，男子却捂着臂弯上的伤口，惊悚地侧过头去，眼珠子都快要瞪了出来。
<br/>
<br/>        连星儿顺着男子的目光，讷讷地偏过头去。
<br/>
<br/>        只见一个身段挺拔，脸上戴着半边银色恶魔面具的少年，手上戴着黑色手套持着黑色的手枪，脚下生风，正向他们踱步过来。
<br/>
<br/>        他穿着黑色大风衣，整个脑袋隐匿在风衣的风帽之中，一双恍若深不见底的金眸忽然间变成了黑色，像千年寒冰那般冷冽，摄人心魂。
<br/>
<br/>        男子浑身一个哆嗦，吓得腿软跪在了地上，有液体从他的膝盖上流下，侵染了地面。
