“…”夏夕绾就知道的，不可能随便拿，从她踏进这里的那一刻开始，她已经参与了这场游戏。
<br/>
<br/>        夏夕绾澄亮的眸子看着那个侍者，“为什么不是女为主，男为仆，你们这里搞性别歧视？”
<br/>
<br/>        侍者一愣，这里的酒会办了这么多场了，还没有哪个女人敢说出女为主，男为仆的话来。
<br/>
<br/>        来参加这个酒会的男人都非富即贵，不缺钱，这些来参加酒会的女孩子个个打扮的花枝招展，她们默认了这种游戏规则，说白了就是用自己的年轻美貌来勾搭金主的，成为他们的玩物。
<br/>
<br/>        “美女，这是我们这里的丛林法则，弱肉强食，如果你不想玩也可以，请立刻离开。”侍者道。
<br/>
<br/>        夏夕绾垂下纤长的羽捷，伸出纤白的小手开始挑选面具，“我玩，我选这个面具。”
<br/>
<br/>        侍者看了一下，夏夕绾挑了一个…小猫面具，很冷门的面具。
<br/>
<br/>        夏夕绾觉得这只小猫咪跟小圆圆挺像的，虽然没有小圆圆可爱，就这个吧，她将小猫面具戴在了脸上。
<br/>
<br/>        夏夕绾来到了酒会的入场口，她想进去，但是被人给拦下了。
<br/>
<br/>        “小姐，不好意思，请你先出示邀请函。”
<br/>
<br/>        “邀请函？我没有。”
<br/>
<br/>        “小姐，那你不能进去，除非有人愿意带你进去。”
<br/>
<br/>        “怎么带？”

<br/>        “哎呀，好疼啊。”那个女孩子娇嗲嗲的叫了一声。
<br/>
<br/>        “小美人，哪里疼了，我看看。”那个老总色眯眯的笑道。
<br/>
<br/>        “讨厌，这里人好多，等进去了我再给你看。”
<br/>
<br/>        “好啊，走，我带你进去。”
<br/>
<br/>        那个大腹便便的老总一把将那个女孩子给搂在了怀里，手脚很不老实的将那个女孩子带了进去。
<br/>
<br/>        这样也可以？
<br/>

<br/>
<br/>        这时又一个大腹便便的老总戴着面具走了过来，一个人来的，这种酒会的男人都不会带女伴的。
<br/>
<br/>        夏夕绾迅速走上前，不过她没有撞上去，而是假装崴了脚，“哎哟”了一声。
<br/>
<br/>        那个老总的目光当即被吸引住了，夏夕绾虽然戴着面具，但是她的身段纤柔玲珑，气质清纯绝丽，很经典的女大学生那一挂的，也是最受这些企业老总欢迎的。
<br/>
<br/>        “小美人，你怎么了，崴到脚了吗？”那个老总迅速上前。
<br/>
<br/>        夏夕绾抬头，这也不是很难嘛，她一双澄亮而黑漉的翦瞳望向了那个老总，嗓音里带着几分楚楚动人，“我脚好痛。”
<br/>
<br/>        话音刚落下，一辆加长版的劳斯莱斯商务豪车疾驰而来，停了下来，后车门打开，一道高大英挺的身影跃入视线。
