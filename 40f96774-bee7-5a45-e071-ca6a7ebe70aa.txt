“那我换个说法，你还没有放弃陆寒霆的病情，你依然想治愈他，这也是你想要带走曼陀罗花的原因，是么？”
<br/>
<br/>        夏夕绾转身，看着陆子羡，“这是我自己的事情，现在已经很晚了，我想要休息了。”
<br/>
<br/>        她这是下了逐客令。
<br/>
<br/>        陆子羡看了她一会儿，“以你现在的身体绝对不能再碰这个曼陀罗花，除非你不要命了，虽然我不知道你想干什么，但是为了安全起见，明天我跟你同一架飞机飞帝都城，我在机场里等你，好好休息，明天见。”
<br/>
<br/>        陆子羡开门离开。
<br/>
<br/>        夏夕绾拧起了秀眉，他这是什么意思，监视她？
<br/>
<br/>        夏夕绾坐在了椅子上，又拿出了那本医典，翻到了最后一页。
<br/>
<br/>        ……
<br/>
<br/>        陆子羡出了房间门，然后拿出自己的手机，拨出去一个电话，“从现在开始派人盯着夏夕绾，明天就飞帝都城了，我不希望离开的时候发生任何意外。”
<br/>
<br/>        ……
<br/>
<br/>        翌日清晨，夏夕绾睡到了一个自然醒，洗漱好的时候都已经九点多钟了，她打开房门走了出去。
<br/>
<br/>        走到回廊里，她站着等电梯。
<br/>
<br/>        这时耳畔传来了一阵沉稳的脚步声，她侧眸一看，只见前方走来了一道高大英挺的身躯，是陆寒霆。
<br/>
<br/>        他竟然也在这里。
<br/>
<br/>        夏夕绾没有想到在这里碰到他，这么猝不及防的，毫无心理准备，这还是两个人离婚后第一次碰面。
<br/>
<br/>        她迅速移开了目光，看着电梯跳跃的红色数字键。
<br/>
<br/>        这时陆寒霆走了过来，站在她的身边，他嗓音有些暗沉沙哑，“最近…过得好吗？”
<br/>
<br/>        夏夕绾没有看他，轻轻的点了一下头，“恩，挺好的，你也住在这家酒店？”
<br/>
<br/>        “工作需要。”他简单的答了四个字。
<br/>
<br/>        “哦。”夏夕绾没有再说话，她努力让自己轻松自然一点。
<br/>
<br/>        这时耳畔又传来他暗哑的嗓音，“昨晚，我看到陆子羡进你房间了，你们发展好像挺快的，进行到哪一步了？”
<br/>
<br/>        夏夕绾一滞，迅速扭头看他，他也在看着她，他狭长的眼梢里全是颓废疲倦的红血丝，眼下一片乌青，看着精神状态很不好，整个人萦绕着一种病态的阴郁和戾气。
<br/>
<br/>        “我听不懂你在说什么。”电梯来了，夏夕绾丢下一句话直接进电梯。
<br/>
<br/>        但是下一秒她纤细的皓腕就被一把扣住了，男人骨节分明的手指探过来用力一扯，她被他推抵到了墙壁上。
<br/>
<br/>        纤柔的后背贴着墙壁，前面是他精健的胸膛，她被他给堵在怀里了，夏夕绾拧眉，“陆总，有话好好说，别动手动脚。”
<br/>
<br/>        这么近的距离足以让他嗅到了她身上那股清甜的少女香气，是他迷恋和喜欢的，“你怕是对动手动脚有什么误解，在我真的动手动脚之前好好回答我的问题，昨晚你跟陆子羡在房间里都干了些什么？”
<br/>
<br/>        夏夕绾看着他血红的眼眶，两只小手抵上他精硕的胸膛想将他推开，“这是我的私事，不要你管，陆寒霆，你好讨厌！”
