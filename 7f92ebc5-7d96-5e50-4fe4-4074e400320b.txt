　　老者和煦的笑道：“两位初来天涯城就遇到如此令人不悦的事情，为何不多留两日，让老夫等人好生招待。”
<br/>　　“天涯城的景色已经看过，不必在久留。”慕清霄平静的语气中充满不容置疑。
<br/>　　若是平时，方圆千里内鲜有人敢如此跟他们说话，可想到被斗尊轰杀的罗家老祖，两名老者却是敬畏如虎。
<br/>　　他们知道，暗中肯定有斗尊至强者在保护两人。
<br/>　　“既然如此，那就由老夫两人开启虫洞，让公子与姑娘前往中州。”
<br/>　　说话的同时，两位老祖走在前面，随后盘坐于中央石台上的空间虫洞两边，双手结印，周围空间微微扭曲。
<br/>　　随后一股空间之力自两名老者身上涌现而出，旋即冲进空间虫洞中。
<br/>　　随着空间之力的涌进，空间虫洞中的呼啸声越来越强，其中弥漫出的吸扯力也逐渐递增。
<br/>　　见状，两名老者不敢犹豫，毫不犹豫的将浑身空间之力爆发而出，低喝声中，斗气弥漫，将空间虫洞包裹，同时也将吸扯力隔绝掉。
<br/>　　他们已经不是第一次开启空间虫洞，对此非常了解，若是不慎被吸入虫洞中，活下来的可能性很小。
<br/>　　除非，像慕清霄这种有斗尊保驾护航的存在，方可平安无事。
<br/>　　慕清霄平静的站在一旁，肩上的彩鳞与左侧的小医仙都是饶有兴致，美眸都是一眨不眨的盯着空间虫洞。
<br/>　　开启空间虫洞耗费的时间并不长，只是需要空间之力而已，
<br/>　　不多时，两名老者徐徐睁开双眼，满脸皱纹的脸颊上泛起一丝苍白，可见由两人开启空间虫洞也是颇为吃力的事情。
<br/>　　随着两人双眼睁开，只见旋转中的空间虫洞轻轻颤了一下，旋即一丝银色光芒缭绕在虫洞四周。
<br/>　　站在下面看去，宛如照亮虫洞的繁星，并且从其中弥漫出来的吸扯力也渐渐消散。
<br/>　　“不负所托，就算没有罗老鬼，老夫两人一样能开启虫洞。”
<br/>　　感受到空间虫洞在正常运行，两名老者也是相视一笑，眼中皎洁如老狐狸一样，谁也猜不出他们心中在想什么。
<br/>　　微微点头，慕清霄道：“仙儿，彩鳞，我们该前往中州了。”
<br/>　　闻言，小医仙轻轻颔首，彩鳞金色蛇瞳中则闪过一抹精芒，对于中州已经有些期待起来。
<br/>　　见状，邢家老祖心中有些失望，自纳戒中取出一个巴掌大小的木船递给慕清霄，木船上隐隐透着一股空间之力。
<br/>　　本想尝试抱住慕清霄的大腿，就算扯住一根腿毛也好，现在看来完全没希望，后者根本不接他们的话茬。
<br/>　　“告辞。”
<br/>　　慕清霄知晓空间船的妙用，牵起小医仙的玉手，直接跨入空间虫洞中。
<br/>　　在两人身影跨入空间虫洞后，便瞬间感应不到气息，一股银芒扩散而出，最后闪掠至漆黑的空间之中。
<br/>　　望着慕清霄与小医仙离去，邢家老者道：“真是飞来横祸，也怪罗老鬼不管束孙女，惯的嚣张跋扈，以后得让后辈把眼神放亮一点，免得惹来祸事。”
<br/>　　“……罗家倒台，咱们合力压制天涯城其他势力，等事情过去，在将罗家的产业全部均分了，你看如何？”
<br/>　　两人谁不是人老成精，现在最重要的不是瓜分资源，而是压制其他势力，免得引起众怒。
<br/>　　等事情平息过去，资源还不是他们两家的。
<br/>　　“也好，既然那位公子对罗家与虫洞不感兴趣，就由我们瓜分了吧。”
<br/>　　“有斗尊在暗中保护，岂会在意区区一个虫洞，老夫猜他肯定是中州某个恐怖势力的公子，比起中州，还是天涯城比较安定啊。”
<br/>　　说罢，两人相视一眼，皆是叹息一声，转身朝台下走去，六名斗皇紧随其后。
<br/>　　……
<br/>　　……
<br/>　　与此同时，空间虫洞中。
<br/>　　慕清霄打量着空间虫洞的内部空间，若有所思，这与他的神通《空间门》有些相同。
<br/>　　眼前是一个看不见尽头的空间通道，通道两旁，附着若有若无的淡银色空间壁障，这是斗宗独有的空间之力。
<br/>　　壁障中的空间很宽阔，足足有十丈左右，通道的尽头是一片深邃的黑暗，脚下也是令人余悸的黑色。
<br/>　　隐隐中有若有若无的空间波动自其中渗入，整个通道死寂一片。
<br/>　　慕清霄一眼就能看出，空间虫洞有不稳定的因素，空间波动能渗入其中，便是说明可能会出现问题。
<br/>　　而他自己施展的空间门极其稳定，根本不会出现这种情况。
<br/>　　想到此处，慕清霄取出空间船，犹如法宝一样，眨眼间就化作几丈宽的船只，船只上有银芒流转。
<br/>　　慕清霄搂着小医仙跃上船只，暗自称其，倒是与他在鬼界冥河上的经历有些相似。
<br/>　　此时，小医仙则抱着他的胳膊，与彩鳞聊天的同时也在打量着空间虫洞内的景象，有慕清霄在，她们根本不需要担心安全问题。
<br/>　　慕清霄轻轻坐下，将能量灌输到空间船中，下一刻船只就行驶起来。
<br/>　　“在虫洞中的时间很枯燥，你们可以放心修炼。”
<br/>　　话语刚落，彩鳞的美眸眨了眨，见识过空间虫洞中的风景后，她就对此没有了兴趣，卷缩在他的肩上，开始转换能量。
<br/>　　修炼过心法后，彩鳞是真正的感觉到了心法的恐怖，完全超出了她的认知。
<br/>　　她此刻的能量才转换到八星斗皇级别，但是那种独特，浑厚并且至寒的灵气，足以支撑她匹敌斗宗强者。
<br/>　　要知道，她此刻只是转化到了斗皇，若是转化成斗宗，绝对能对战比她高出数个等级的强者。
<br/>　　这种不靠秘法，只靠自身强大的道路简直令人心惊。
<br/>　　她心里非常清楚，若是这种心法流露出去，不知道会造成多么恐怖的声势，就算远古八族也会动容吧。
<br/>　　而且，她隐隐觉得，自己的血脉似乎在蜕变。
<br/>　　小医仙转化能量后，灵气中继承了厄难毒体的性质，而她转化能量的同时，也察觉到自己的血脉力量似乎有所转变。
<br/>　　虽然不知道朝哪个方向转变，但她清楚这是自己的机缘，有利无害。
<br/>　　……
