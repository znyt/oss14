<br/>				目光带着愕然的落在前面杂草丛中那伸着手脚动着的小婴儿身上。<br/>　　<br/>　　这地方怎么会有婴儿？她心下怪异，迈着步伐走上前，见果然是一个小婴儿，只不过，这个小婴儿是没睁开眼睛的，但他的手脚却在舞动着，嘴里发出啊啊的叫声。<br/>　　<br/>　　看着那浑身赤果着的小婴儿在那里舞动着手脚，她神识释放而出，朝周围扫去，却发现这周围除了她之外，没有别的人了。<br/>　　<br/>　　然而，就在她正在收回神识时，一阵轻风吹拂过来，却闻到一股淡淡的血腥味。<br/>　　<br/>　　“血腥味？”<br/>　　<br/>　　她挑了挑眉，看向林中的某一处，又看了脚下边的婴儿一眼，这才弯下腰将杂草丛中的婴儿抱了起来，当将孩子抱起时，手一湿，低头一看，只见孩子的后背处有着一处伤口正流着血。<br/>　　<br/>　　“受伤了？”<br/>　　<br/>　　她微讶，刚才只闻到远处传来的一阵血腥味，倒没闻到这婴儿身上的血腥味，压下心头的怪异，她腾出一手，正想着从空间中取出件衣袍来包住这婴儿，可就在她这一腾出间，那被她拥在怀里的婴儿突然间睁开双眼，那眼睛一睁开，根本不是婴儿的眼睛，而像是兽类的眼睛一般。<br/>　　<br/>　　“嘶！”<br/>　　<br/>　　突然间怀中的婴儿整个一变，竟如一妖兽一般从她怀中窜起，那嘴巴嘶的一声大张而开，露出锋利尖锐的獠牙，那如爪子般的双手一手抓住她的肩膀，一手抓着她的衣襟，透着嗜血气息的大嘴猛然凑上前便冲着她的脖子咬下。<br/>　　<br/>　　突如其来的变故让凤九来不及避开，她只能伸出没被抓住的手狠狠一掌拍向那妖兽的头，手掌间一股凌厉气息划过，只听砰的一声，那妖兽的整个头瞬间飞了出去，但身体仍紧紧的粘在她的身上还在动着。<br/>　　<br/>　　她手一甩，试图将那妖兽的身体甩出，只是，当身体灵力涌动，强大气息将之弹开的同时，她被那妖兽抓住的肩膀处也被撕划出几道深可见骨的血痕。<br/>　　<br/>　　眉头微微一拧，闷哼了一声，她眸光泛着寒意的看着那尸首分家却还在动着的头和身体，手掌一翻，两道火焰呼的一声飞出，瞬间将那妖兽的头颅和身体烧了个干净。<br/>　　<br/>　　“凤丫头！”<br/>　　<br/>　　捡好树枝却未等到凤九归来的老头因有些不放心，就一直往凤九走去的方向寻着，当走出一段距离后，却突然听见前面的动静，神识当即释放而出，迅速朝那方向掠去。<br/>　　<br/>　　凤九的手无力的垂落着，青衣染血，胆战心惊，她的手臂因那深可见骨的伤口而不自觉的在微微颤抖着，她垂眸瞥了一眼，听着前面仿佛被吵醒而发出的婴儿哭喊声，目光微闪，迈着脚步往前走去。<br/>　　<br/>　　“凤丫头！”<br/>　　<br/>　　混元子的身影飞掠而来，几个呼吸之间已经来到凤九的身边，当看到她那胆战心惊的伤口时，不由的倒抽了一口冷气。<br/>　　<br/>　　“嘶！你、你怎么伤成这样了！伤口还在流血，赶紧包扎一下止血啊！”老头看到她这模样，也是生生吓了一跳。<br/>
