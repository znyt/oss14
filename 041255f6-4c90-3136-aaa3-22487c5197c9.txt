　　第419章
<br/>　　一道人影飞扑而来，生生横搁在明帝与铁枪之间。
<br/>　　在关键时候，刘阿吉爆发了体内最后力量，挡住了林少羽的绝命一枪。
<br/>　　随之的心脏被铁枪洞穿，一张脸由白转红，噗地喷出一口血，一命呜呼。
<br/>　　林少羽怔了怔，但明帝的反应比他更快。明帝恍若没有看见刘阿吉的尸体，右手抬枪，扎向林少羽。
<br/>　　这种反应，当真不是林少羽能有的。等他醒悟过来，铁枪已穿过他的肩膀。随之，林少羽拔出铁枪，向明帝扎来。
<br/>　　明帝退，一退再退，直到退无可退，面前是林少羽血红的枪锋。
<br/>　　临事方知一死难，明帝眼底闪过一抹惧意，不，自己还没有输，自己还有援手。
<br/>　　林少羽似乎知道对方在想什么，嘴角浮现一抹冰冷的笑。
<br/>　　“你等的人不会来了。”
<br/>　　随即，手中的铁枪决绝扎下。
<br/>　　皇宫外，浮云寺。
<br/>　　一人推开寺门，快步而出，他一跃攀上屋顶，疾步而行，各处宫墙下都有争斗，但他视若无睹，直扑金殿而来。
<br/>　　忽地，前方屋顶上出现两个人影，一老一少，少年手中举着一个白幡，上书四个大字。
<br/>　　『看破人间』
<br/>　　老僧的脚步停住，双方隔着二十步对视。周围繁杂声响，厮杀、惨叫、长空赫赫风声好像都已被隔开，这大殿屋顶，只有三个人。
<br/>　　半晌，老僧双手合十，微微垂首。
<br/>　　“师兄。”
<br/>　　老道手捻胡须，轻轻点头。
<br/>　　“师弟。”
<br/>　　一僧一道，都是年迈苍苍，却以师兄弟相称。这令旁观这一幕的小道士大呼不可思议。
<br/>　　青叶佛孟知了，浮云寺主持，在长安城有活佛之称。人赞他精通佛理，口绽金花，每次开坛讲佛时，浮云寺总是挤满了人。
<br/>　　云游道人郭烦人，号北斗先生，举着『看破人间』的牌子，十分猖狂欠揍。但在如今的正义教，他是当之无愧的第一军师，而在正义教信徒眼中，他简直就是活神仙降世。
<br/>　　但事实上，孟知了并不是和尚，郭烦人也并非道士，在很久很久以前，他们都有另外的身份。
<br/>　　郭烦人，出身江南，是帝国立国以来，第七位状元，曾官拜帝国太学教授。三十年前，与戎族签订丧权辱国合约的那一次，便是他亲自主持的和谈会议。也因为那次的事情，郭烦人被整个帝国戳脊梁骨，背了太多骂名，于是辞官不做，选择浪迹江湖。
<br/>　　孟知了，青州人士，出身青州孟家，而孟家是跟着帝国开国之君一起打江山的人物。刚出生，孟知了身上便背着爵位，在很年轻时，世上红尘滚滚，他便都享受过。
<br/>　　二人一个寒门贵子，一个风流公子，看似两条路上的人，但却有一个共同的老师：大儒伯阳子。
<br/>　　二人学礼义廉耻，诗书乐经，从儒家的无上经典中，一个悟出了佛理，一个悟出了道法，然后走上了截然不同的两条路。
<br/>　　恰又在今日重逢。
<br/>　　二人相隔二十步而立，目光是同样的平静，但身上却流露出罕见的锐气。老人在年轻人面前总会有些嫉妒的情绪，也只有在同龄人前才会找回些自信。
<br/>　　大家都是老狐狸，互相都知道彼此的底细，谁也不用去装大尾巴狼。
<br/>　　远处的打斗声传到这里，孟知了内心不像外表一样平静，自己此刻赶过去，还有可能挽回局势，时间拖得越久，对己方越是不利。
<br/>　　“师兄，可否让一让？”
<br/>　　“师弟，可否留步。”
<br/>　　孟知了的呼吸变得越来越缓，最后甚至接近消失。无非就是打一场而已，打得过对方才能通过这里。
<br/>　　“师兄，请了。”
<br/>　　“了儿。”
<br/>　　郭烦人摊开手，一柄剑递到他掌心。他摘掉剑鞘，锋刃搁在面前。
<br/>　　青叶佛悍然出手，摘下身上的袈裟当武器，直如棍，曲如勾，能甩能抽，一块袈裟在他手中成了最致命的武器。
<br/>　　郭烦人以剑相格，连续数招，安邦、定国、引山河！
<br/>　　他防得滴水不漏，青叶佛找不到破绽。红尘了呆呆的看着这一幕，有些失神。与师父相处太久，平常并未见过他出手，以为他只会坑蒙拐骗，却不知他还有如此犀利的剑法。
<br/>　　“浩然剑！”青叶佛冷笑：“引贼入京，以乱天下，你还有资格用浩然正气剑么？”
<br/>　　“帝国气数已尽，凡事不破不立，我问心无愧，还能养我浩然正气。”
<br/>　　“这浩然正气，需要太多血来养了。”
<br/>　　随着一声尖喝，青叶佛再次出手，袈裟卷成一条布棍，向郭烦人拍来。
<br/>　　知止、修身、己所不欲、逝者如斯、当头棒喝……
<br/>　　儒家三十六棍，棍棍诛心，打的就是天下不听话的人。
<br/>　　二人一个和尚一个道士，一个用剑，一个用棍，但事实上，却是师出同门，剑法棍法里面的道理都是相通的。
<br/>　　“师弟何必执迷不悟，定要做帝国的孤臣逆子。”
<br/>　　“你有你浩然正气，但我虽千万人吾往矣。”
<br/>　　这个时代的儒家还不想程大雷所了解的那样死气沉沉，『他』还很年轻，充满改变天下的锐气。
<br/>　　“帝国这口大船已无可救药，聪明人早已下船。”郭烦人刺出一剑，这是浩然剑中最普通的一式：横平竖直。
<br/>　　“师兄忘了，我从来算不上聪明，要做这帝国的傻瓜。”孟知了手中的袈裟转个圈，是三十六棍的基础招式：捺如刀。
<br/>　　横平竖直，是写字的开始，一撇一捺是做人的开始。
<br/>　　二人的本事都到返璞归真的境地，招式越来越简练，威力却越来越强。
<br/>　　“但已晚了。”
<br/>　　郭烦人爆喝，剑锋忽然变得犀利起来，浩然正气剑中的杀招。
<br/>　　引山河。
<br/>　　孟知了突然变招，是佛门武学中的菩萨低眉。
<br/>　　不怕金刚怒目，只怕菩萨低眉，菩萨低眉是收起慈悲，杀机已动。
<br/>　　郭烦人没有算到他的突然变招，猛醒悟过来，孟知了已欺近身边，布棍甩在他胸口。
<br/>　　君子可以欺以方。
<br/>　　郭烦人哇呀吐出一口黑血，一张脸由白砖红，由红变青。
<br/>　　他抬起头：“师弟……”
<br/>　　“我赢了。”
<br/>　　“已经晚了。”
<br/>
