　　第696章
<br/>　　“孙小毛在何处？”歪坐在椅上的程大雷问道。
<br/>　　吴用回过神来，躬身道：“还押在大牢之中。”
<br/>　　程大雷长身而起：“带我去见他。”
<br/>　　吴用引着程大雷下了大牢，牢中还颇关着一些人。有些是趁着这次破城闹事的兵将，他们犯得事并不大，不至于死罪，暂且关在牢中。有些是凉州城的降兵降将，此刻见到程大雷进来，胆子大的破口骂娘。
<br/>　　一路走进去，来到一间牢房前。里面关着一个面白无须的年轻人，看模样竟然有几分清秀，见到程大雷后，顿时跪倒，以头触地不肯起身。
<br/>　　程大雷隔着牢栅看着他，道：“抬起头看我。”
<br/>　　孙小毛这才敢抬起头，道：“大当家，属下死罪。”
<br/>　　程大雷没有说话，细细端详着他的面貌。不过是二十岁出头的年轻人，虽然是战场中的悍夫，但眼神中还带着几分稚气。一股火热的生气，还想着好好活些年头，耗尽一身血肉方才罢休。
<br/>　　“孙小毛，你怕死么？”
<br/>　　孙小毛坚定的摇摇头，未发一言。
<br/>　　程大雷长身而起，迈步离开，再没有回头看一眼。
<br/>　　一直走出牢房，外头阳光炙热，程大雷长长吐出一口浊气。
<br/>　　“大当家，这孙小毛该如何处置。”
<br/>　　“三日后午时，斩。”
<br/>　　程大雷在大牢外说出口的这句话，没用多少时间便传遍全城，街头巷尾都在议论着这件事。
<br/>　　甚至，就连在城主府足不出户的崔白玉几个也听见了。
<br/>　　“小姐，那程……大当家，真会斩了那人么？”正在书房内忙碌的小鹿，见左右无人低声问道。
<br/>　　“杀一儆百，收拢人心，有什么值得大惊小怪的。”崔白玉淡淡道。出身在相府，这种事她都看得厌了。莫说一个默默无闻的小兵，便是生死弟兄杀了也就杀了。
<br/>　　正这时候，程大雷进来，四人同时逼上嘴巴。
<br/>　　“事情处理得如何了？”程大雷坐下后问。
<br/>　　崔白玉递过来一份自己整理的材料，程大雷摊开后，先见一手漂亮的蝇头小楷便忍不住赞了一声。
<br/>　　不过短短时间，崔白玉就大概有了眉目，她将城中的大户根据家财多少分成四等，依次规定税收。动刀子动得恰到好处，当然，程大雷现在还不能完全信任她，不过，她工作到这种程度，还真是值得程大雷拍手叫好。
<br/>　　“汝不愧是我的女诸葛。”
<br/>　　“诸葛是谁？”崔白玉皱眉问道。
<br/>　　程大雷撇撇嘴，转移话题道：“好了，这等无关紧要的小节，就没必要在意了。你快快完成这件事，你做得好，本当家都记在心上，日后不会委屈你的。”
<br/>　　崔白玉沉默的点点头，委身于贼实在难令她开心起来。不过，这也是绝路中没有办法的办法，至少、情况没有比最糟的糟。
<br/>　　程大雷目光在崔白玉身上打转，她如今又罩上黑纱，这倒是为程大雷减轻不小压力。
<br/>　　“来，有件事我还要向崔姑娘请教。”
<br/>　　“程当家有话直讲，请教二字不敢当。”
<br/>　　程大雷清了清嗓子，道：“城中发生什么事，相信你也知道了。我想问一下，若是崔相大人会如何处置这件事？”
<br/>　　崔白玉颦眉想了想，道：“须看人也得看事，如这次的事，怕父亲大人会和程当家做得一样。”
<br/>　　“原来崔相也会这么做。”程大雷轻轻点头。
<br/>　　崔白玉心头想得明白，那孙小毛在军中无亲无故，正是杀一儆百的上佳人选，杀了他收获民心何乐而不为。程大雷也不知在为难什么。
<br/>　　程大雷招招手，香珠递过来一杯香茶。程大雷手托着茶盏，却也不饮，思考了半晌道：“令尊……他是个什么人？”
<br/>　　崔白玉一怔：“程当家为何要问这个？”
<br/>　　“随便聊聊么，忽然想了解一下。”
<br/>　　崔白玉认真想了一会儿道：“如白玉平生所遇，有两个人我最看不穿，其中一个便是家父。”
<br/>　　“那另外一个是？”程大雷被激起好奇心。
<br/>　　“另一个便是程当家。”
<br/>　　程大雷摸了摸鼻子，道：“我怎么能和崔相相提并论，不提我了，聊聊令尊，老人家究竟什么路数？”
<br/>　　崔白玉道：“若在外人看来，家父不过是个普通老叟，能一路走到今天，一是肯去做事，二是肯把事情做尽。”
<br/>　　程大雷点点头：“敢为人先，持之以恒，的确是成大事者必备的素质，还有其他呢，令尊有没有什么朋友？”
<br/>　　崔白玉苦笑着摇摇头：“天下之大，谁有资格与家父做朋友。”
<br/>　　程大雷也不知心中想着什么，随口道：“那想必令尊此生应是很孤单吧。”
<br/>　　一句话触动了崔白玉的心思，她试图在脑海中回忆父亲的印象。的确是个普通老叟，可不会有人小看他，老人家一举一动便影响帝国的气数。于无声处落子，却能运筹帷幄千里之外。
<br/>　　崔白玉一直是很怕崔相的，在她的印象中，崔相代表的是智慧、强大、沉默。却是没想到，程大雷从父亲身上捕捉的第一个词汇竟然是孤单。
<br/>　　老人家妻子早亡后便没有续弦，家中的妾室是多的，还有各处孝敬过来的女人。崔白玉其实就是庶出，如果不是仗着美貌与智慧，在家中的地位一定不高。
<br/>　　父亲一世有过许多女人，但若谁能与他交心，怕是一个也没有。至于儿女孙辈，老人家也是运筹帷幄，不能让任何一人坐大。
<br/>　　对自己的子孙尚且如此，何况是归附的世家、高高在上的李氏皇族。
<br/>　　记忆中的父亲一直坐在那间昏暗的书房中，驭下之策，识人之术，驱狼吞虎……这些门道父亲都玩得很通透。天下之大，估计没有一人知道老人家心中想什么。
<br/>　　可若不是程大雷一句话点头，崔白玉怕是一生一世都不会想到。那坐在书房中的智慧老者，的确是有些孤单呐。
<br/>　　崔相一人之下，万万人之上，膝前跪伏着几多人。这样一个老者，自然享尽人间富贵。但普通人在田间酒楼，与老友吹牛打屁的乐趣，他怕是一生都无法享受。
<br/>(本章完)
