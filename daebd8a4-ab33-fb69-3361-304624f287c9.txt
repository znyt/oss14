　　“天道有缺，众生皆苦。吾乃盘古氏血脉，巫族祖巫后土，今有感天地不全，轮回有缺，为天下苍生之大计，愿以身演化天地轮回，补全天地！”
<br/>　　陡然间，巨大的身影开口说话，声若雷霆，传遍洪荒，轰动九霄，震动乾坤，四海八荒为之震动，天地间所有生灵都能听到。
<br/>　　后土的声音之中，带着一丝留恋，一丝解脱。这一刻的她，神圣，高贵，慈悲，宛如神话中唯一的神女一般，高不可攀，贵不可言。
<br/>　　“什么，后土妹妹，不可！”
<br/>　　“不可能，怎么如此？”
<br/>　　“后土妹子，你怎么那么傻？”
<br/>　　“……”
<br/>　　所有巫族祖巫，都面露悲伤，纷纷大叫着，急忙身形急飞，向幽冥血海飙去。
<br/>　　“后土娘娘慈悲！”
<br/>　　“娘娘大德！”
<br/>　　这一刻，天下所有生灵，无论巫妖人，俱皆朝着血海方向朝拜而去，泪流满面，悲痛之意表露无疑。
<br/>　　“匡！”
<br/>　　万寿山，五庄观。
<br/>　　“娘娘大慈悲，我辈不如也！”
<br/>　　镇元子带领几十位弟子踏出庄门，向着血海方向躬身行礼，肃然道。
<br/>　　天庭之上，太阳神宫。
<br/>　　“后土祖巫，果然不凡！”
<br/>　　帝俊与太一脸色肃然，朝着血海方向躬身一拜。尽管他们妖族与巫族一直都是宿敌，但对于后土祖巫如今的做法，他们还是非常尊敬的。
<br/>　　此刻，天地间无数大神通者，纷纷起身行礼，以示尊敬！
<br/>　　昆仑山，元始三圣脸色肃然，对着后土持半礼。
<br/>　　西方灵山，接引与准提两位圣人感受到天地之间的大悲之意，远远超过他们佛家的寂灭意境。这才是真正的大慈悲，而不是他们那种只是悟道出来的慈悲。
<br/>　　佛家的道是一种假慈悲，为了修佛成道才有了慈悲，意图度化苍生，自然远远比不上后土祖巫这种发自内心的怜爱世人、怜悯苍生的大慈悲。
<br/>　　佛家慈悲少了一种发自内心之意，却又多了一种虚伪的慈悲在其中。而这一刻的后土，却是发自内心的，真情实意，是圣贤，布德于天下！
<br/>　　“圣贤者，先天下忧而忧，必以造福后世、天下苍生为己任！”
<br/>　　这一刻，西方二圣，面色肃穆，对着血海，行半礼。
<br/>　　混沌天外，娲皇宫中，女娲面带震惊，对着后土，行了半礼，道：
<br/>　　“后土祖巫，吾不如也！”
<br/>　　天地之间，除了道祖鸿钧，谁还能得圣人一拜？
<br/>　　血海深处，后土法天象地，她面容坚毅，虚空之中，一股无形的力量加持在她身上，好似在助她悟道，玄之又玄，同时，也有一股无形的力量正在拉扯着她的身体，向着幽冥界方向而去，令她不由自主地发出悲鸣声。
<br/>　　三生道人站在一旁，看得很清楚，前一股无形的力量是天道之力，后一股力量是幽冥界的力量。天道为何会出手，他也很清楚，洪荒世界不完美，天道想要进化，自然是想要向着完美的方向进化，只有天地完美了，天道才能更强。
<br/>　　天地大道相助，助她悟道，先前那些不明之意，此刻已经十分清晰的展现在她眼前，有感之下，后土苦笑一声，道：
<br/>　　“天道法则亦来助我提升功力，看来我后土身化六道轮回已成定局了。哈哈！还真是天意弄人！”
<br/>　　后土的声音之中，带有悲戚之意，三生道人听后，满心不是滋味。
<br/>　　“也罢，得道，即是失道。我心之所向，为天地万物众生，轮回大道，纵消散于天地间，亦万死不悔！”
<br/>　　“以我祖巫之力，开幽冥之界！”
<br/>　　血海上空一阵涟漪，虚空陡然间现出冥冥一界，乌黑的玄冥之光，幽幽闪烁，阴森森的气息，弥漫整个世界。后土一步踏出，走进幽冥界，她声音不断，继续喊道：
<br/>　　“以我祖巫之躯，现六道轮回！”
<br/>　　幽冥界深处，一阵晃动，虚空显出六道巨大好黑洞，尽皆透出玄奥的气息，六道上接洪荒大地，下连幽冥血海。
<br/>　　“以我祖巫之灵，合六道轮回！”
<br/>　　六道巨大的玄妙的黑洞缓缓出现，幽冥玄光闪动之间，六道之中传来恐怖的吸引力，将天地间那些游离的孤魂野鬼吸引过来，向着六道而去。
<br/>　　后土祷告苍天九幽，震动天地，赫见六道渐渐清晰，上部隐没洪荒不见，下部隐入幽冥血海不见，六道之洞齐齐闪动着玄妙的光芒，就要把后土吸引过去。
<br/>　　“巫族，我的族人，洪荒，我的家园，永别了……”
<br/>　　来自六道轮回的吸引力越来越大，后土距离六道孔洞也越来越近，望着离自己越来越远的洪荒大地，后土默默地说了一声，语气之中，更多的是留恋，是不舍，同时还有感慨，感慨大道无形，天道强大，感叹圣人的无为，自己的无力。
<br/>　　六道轮回建立的这一刻，天地轰动，乾坤倒转，风云倒卷，洪荒生灵俱皆朝着幽冥界跪拜，口呼后土娘娘功德无量。
<br/>　　亲眼见证了这开辟幽冥，建立六道这样的旷世奇观，三生道人却愣在了原地，他心中不由得为之一叹，不仅为后土的大德慈悲而感叹，还为自己的无力而感慨。
<br/>　　迈步走进幽冥界中，只见偌大的地府，刚刚开辟，却已有洪荒的三分之二般大小，但此刻里面却空荡荡的，没有一丝生气，冷清无比。不过，三生道人对此也早已习惯了，毕竟他之前就一直住在这幽冥界中，此刻的情况比之前好了不少，以后的幽冥界也会越来越热闹。
<br/>　　不过，不知为何，他的内心并没有高兴，有的只是悲哀。
<br/>　　“唉……”
<br/>　　“嗡！”
<br/>　　陡然间，在洪荒众生那不可置信的目光中，一道有些虚幻的元神从六道轮回之中走出，她柔弱无比，形单影只，好似随时要随风飘散一般。
<br/>　　后土娘娘，她不是已经死了吗？
<br/>　　眼前这是怎么回事？难道我眼花了？
<br/>　　这怎么可能？
<br/>　　同时，一旁的三生道人好似对此早有所料，并没有表现出太过吃惊的表情。
<br/>　　“嗡！”
<br/>　　这时，一滴淡青色的散发着浓浓威压的血滴从虚空中出现，并快速没入后土那虚弱的元神之中，让后土那虚弱的的元神慢慢变得凝实起来，同时，全新的血肉从头部开始，慢慢向下延伸，直至完全覆盖后土的元神，一具全新的肉体出现在后土身上。
<br/>　　虽然这具肉体还很弱，但潜力十足。只因为那滴青色的血滴，乃是盘古神尊精髓所化，是当初后土从女娲手中交易到的，只是这些年一直没有能够炼化，谁又曾会想到它会在此刻出现，并且帮后土重新凝聚出一具肉体？
<br/>　　与此同时，九天之上凝聚的无量功德金光在此刻仿佛找到了主人一般，向着后土飞来，其中半成落入冥河手中，一成落入三生道人手中，余下的八成半全部落入后土元神之中。
<br/>　　“轰隆隆！”
