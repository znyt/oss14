<br/>    沙蝎王庭。
<br/>    
<br/>        沙蝎大妖王站在黄沙之前，左手一个，右手一个，各自抱着一个衣衫半露的妖女，这里并不是沙海的范围，而是沙海之外蔓延过来的一片沙漠，而他，便是这里的主人。
<br/>    
<br/>        阴月天狼在他的背后缓缓落下，“沙蝎，有活来了。”
<br/>    
<br/>        沙蝎不经意地点点头，伸手将两个妖女推开，“下去。”
<br/>    
<br/>        沙蝎回头，问道：“天狼大统领，这次有多少货？”
<br/>    
<br/>        阴月天狼道：“这一次，货的数量有点多，估摸不下百万丹药，所以，你最好倾巢而出，以免被别人抢了。”
<br/>    
<br/>        沙蝎惊道：“这么多？”
<br/>    
<br/>        即便是作为一方大妖王，一次听到这么多数量的丹药，还是惊得说不出话来。
<br/>    
<br/>        阴月天狼点头道：“不错，这一次，还是按照之前的规矩，你抢到之后，大君拿一半，你们拿一半。”
<br/>    
<br/>        沙蝎道：“我明白了，我会保证万无一失的，这一次，我亲自上场！”
<br/>    
<br/>        阴月天狼笑了笑，道：“那我就放心了。”
<br/>    
<br/>        看着沙蝎已经在调兵遣将，阴月天狼笑了笑，“真是不知道燕山到底哪里来的这么多的丹药，地里长出来的吗？不过再多又有什么用呢，还不是只能落入大君的手中，这个燕山，你们根本出不来！”
<br/>    
<br/>        拜月看了一眼天空中的月色，坐在商队的马车之上饮了一壶酒，有些无聊，也不知道对方什么时候来截杀，看了一眼背后在沙漠中行进的商队，燕山镖师为了隐藏自己的修为，在地面上行走，他终于是冷静了下来，无论怎么样，有这些燕山镖师在，应该是安全的。
<br/>    
<br/>        在前方， 沙漠一场颤抖，狂暴的风沙起卷，如同要将所有人吞噬，拜月看向前方暴动的风沙，来了！
<br/>    
<br/>        不过来的，似乎有点眼熟，最前方的，是沙蝎！
<br/>    
<br/>        拜月无论如何也没有想到，来的居然是一位大妖王，原来一直在截杀他们商队的，是沙蝎！
<br/>    
<br/>        本来是信心满满的拜月，现在可是没有了任何战斗的心思，沙蝎亲至，背后至少是上万的兵马，兵力差距不大，但是对方，可是有着一位大妖王，拜月见过大妖王之间的战斗，那已经可以称之为毁灭一切了。
<br/>    
<br/>        拜月看向身旁坐着的镖师大统领，银翼，燕山镖师一般都不会大举行动，所以就交给银翼了，这一次，银翼也是因为燕山镖局没人了，决定出来混一圈，“银翼大人，我们撤走吧。”
<br/>    
<br/>        银翼正想鼓掌说好，实际上打也不是不能打，但是就是有点心虚，但是一想到自己上次为了接手燕山镖局，可是在陈明的面前立下了军令状的。
<br/>    
<br/>        银翼只能摇摇头，拜月道：“银翼大人，我们决定撤销雇佣，让他们抢走这些丹药，我们还可以继续炼制！”
<br/>    
<br/>        一年以来，镖局也帮了他们不少，他们也是不想看到燕山镖局因为这一战解散。
<br/>    
<br/>        银翼摇摇头，然后道：“我不想和你说那些什么他们这一次抢了，下一次还会来抢的大道理，我只知道燕山镖局有一句话：燕山镖局，镖镖必达，镖在人在，镖失人亡！”
<br/>    
<br/>        银翼道：“虽然听山主说起来很装逼的样子，这一次我说出来，好像也蛮装逼的样子啊！不说那么多了，我要准备作战了。”
<br/>    
<br/>        银翼起身，挥手道：“所有人，披甲，带剑，整军！”
<br/>    
<br/>        本是押送商队货物的镖师齐齐打开马车上的箱子，一件件血衣剑卫换下来的灵甲披身，一柄柄最初系列的杀王剑落于手中。
<br/>    
<br/>        银翼看了一眼，这些灵甲和杀王剑，都是血衣剑卫已经淘汰了的灵兵灵甲，现在归燕山镖师穿着了，虽然实力比不上现在的血衣剑卫，但是和一年多前的血衣剑卫相比，已经是强了不少。
<br/>    
<br/>        银翼看向沙蝎，大喝道：“沙蝎，你想与我燕山为敌不成？”
<br/>    
<br/>        沙蝎哈哈大笑道：“你他娘的不是废话啊，不是来抢你们燕山的，老子带着这么多人来这儿喝西北风不成？这么多的丹药，别说是一位大妖王了，就是两位，三位，老子都抢了！”
<br/>    
<br/>        银翼提着一柄银色的长剑，将最高大的一座马车车顶当做指挥台，一跃冲了上去，面对着漫天的黄沙，银翼举剑道：“我们镖局的使命是什么？”
<br/>    
<br/>        “燕山镖局，镖镖必达，镖在人在，镖失人亡！”
<br/>    
<br/>        银翼大笑道：“好，我们是燕山第一任镖师，可不能让这个名头现在就没了！以五百人为军团，全军冲击！”
<br/>    
<br/>        银翼一声令下，愣在原地，指挥？不存在的，他除了这一句以五百人为军团，全军冲击之外，就不知道了，反正见黑虎上战场，是以三百人为军团，燕山镖师差很多，就以五百人为军团，应该问题不大。
<br/>    
<br/>        不过一瞬之间，二十八个军团在战场上出现，鲜红色的血煞之气弥天，所有的风沙都被阻挡在外，整个天地之间，血煞之气和风沙相撞，各自平分战场。
<br/>    
<br/>        沙蝎仔细看了一眼，也是有些震动，“这是些什么玩意儿？”
<br/>    
<br/>        不过沙蝎也没怎么在意，虽然对方有着上万人，但是压根没有王级军团，也没有大妖王在场，陈明又不在，对于他对战场的认知来说，这tm还用指挥？
<br/>    
<br/>        平推过去不就完了吗？
<br/>    
<br/>        只要自己牵制住对方的妖王，自己的妖王杀过去，妖王骑脸，怎么输？
<br/>    
<br/>        沙蝎背后的披风在狂风中猎猎作响，大手一挥，瞬间下达了指令，“小的们，给我杀过去！”
<br/>    
<br/>        两军正式相遇，黄沙与血煞之气汇于一处，战场之上，各处都是响彻着惊雷般的轰隆巨响，银翼看着两军交战，现在心里也是没底，燕山镖师的实力他知道，都是黑虎黑蟒挑剩下的，根据最近一次和血衣剑卫演练的结果来看，三百镖师只能和一百血衣剑卫打平。
<br/>    
<br/>        现在，也不知道是什么结果，银翼忽然听得到后面有响动，原来是拜月带着三十多位妖王过来了，他们也是带着杀王剑，穿戴着灵甲，杀王剑和灵甲，对于他们来说是限购的，只能买一套自己用，现在战局不明朗，他们作为妖王，燕山镖师为了自己的丹药在搏命，他们也不能干看着，“银翼大人，带领我们杀过去吧！”
<br/>    
<br/>        银翼琢磨了一番，若是没法限制沙蝎，那几乎就是不可能赢，大妖王在这种战场之上，堪称绝对的主宰，“我对于山主对于练兵所施展的杀王剑阵也不清楚，但是还记得山主吩咐的一点，那就是围杀强者的时候，绝对不要和对方正面硬拼，只管在远处，从四面八方用杀王剑招呼就行了，你们既然已经买了一套杀王剑，便随我杀过去试试看！”
