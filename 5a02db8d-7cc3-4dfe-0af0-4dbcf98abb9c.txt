        龙之卫垂涎那小丫头片子手里的烤鱼，只得乖乖照她所说的话去做。
<br/>
<br/>        火熄灭后，诺筱颖将手里的其中两条烤鱼分享给了龙之卫。
<br/>
<br/>        龙之卫小小地咬了一口，发现这鱼烤得特好吃后，才大口大口地吃了起来。
<br/>
<br/>        他真的发现，诺筱颖这小丫头和城里的女孩子不一样。
<br/>
<br/>        她会的东西很多，而且，还是个名副其实的小吃货。
<br/>
<br/>        经过这一段时间的相处，晚上睡觉的时候，龙之卫不再摆“三八线”，而是在不知不觉中，和诺筱颖抱到了一块睡觉。
<br/>
<br/>        旧得发灰的补丁蚊帐里，外婆手里摇着蒲扇，给这两个孩子扇风。
<br/>
<br/>        慢慢地，龙之卫不再抗拒与诺筱颖一起洗澡。
<br/>
<br/>        外婆烧好的艾叶洗澡水，他两总能在木盆里玩上一会儿才肯出来。
<br/>
<br/>        龙之卫以前总觉得自己很孤单，自从来到这里后，他脸上的笑容多了起来。
<br/>
<br/>        这世上，最好吃的饭菜，是外婆做的，最好看的衣服和鞋子，也是外婆做的。
<br/>
<br/>        在诺筱颖的眼里，外婆就是她的超级大英雄。
<br/>
<br/>        而这天下，终究是没有什么不散的宴席。
<br/>
<br/>        龙之卫的妈妈回来了，要带龙之卫回城里去。
<br/>
<br/>        那天，二姨回来时，外婆外出就诊。
<br/>
<br/>        家里就只剩下诺筱颖和龙之卫。
<br/>
<br/>        诺筱颖看着龙之卫的妈妈，也就是二姨在外婆的卧室里翻箱倒柜的找着什么东西。
<br/>
<br/>        直到二姨翻出一本土黄色的小册子，翻看了几眼后，慌忙塞进了自己的布包中。
<br/>
<br/>        二姨随后替龙之卫收拾了衣服，看了一眼守在门口的诺筱颖后，于是朝诺筱颖走过去，蹲下身，拉着她的小手，温柔地说道：“筱颍，等外婆回来后，你跟外婆说，二姨把小卫接走了！”
<br/>
<br/>        诺筱颖乖巧地点了点头。
<br/>
<br/>        二姨微微一笑，抬起手来，揉了揉诺筱颖的小脑袋。
<br/>
<br/>        见这小丫头只是扎着一个马尾辫，二姨又拿来了梳子，给诺筱颖梳了一个漂亮的公主头。
<br/>
<br/>        二姨还从她那布包里，“变”出了一件粉色的小连衣裙。
<br/>
<br/>        她给诺筱颖换上后，拉着诺筱颖的小手，来到坐在门外台阶上的龙之卫的面前，问道：“小卫，妹妹穿这件粉色裙子，好看吗？”
<br/>
<br/>        “嗯，好看！就像小公主一样好看！”龙之卫点了点头，又看向妈妈，接着说道，“妈妈，可不可以带妹妹一起走？”
<br/>
<br/>        “小卫乖，我们不能带妹妹走！妹妹有外婆，还有自己的爸爸妈妈。”二姨安慰着龙之卫。
<br/>
<br/>        龙之卫又看了一眼诺筱颖，从台阶上起身，走到了诺筱颖的面前，掏出裤袋里的小口琴，递给了诺筱颖。
<br/>
<br/>        “等我长大以后，我来娶你，带你离开这座大山，去见见我们那儿的大城市，可好？”龙之卫信誓旦旦地说。
<br/>
<br/>        诺筱颖接过小口琴，微微点了点头。
<br/>
<br/>        二姨只是忍俊不禁地笑着，然后拉起龙之卫的手，和诺筱颖道别后，走出了篱笆院墙。
<br/>
<br/>        诺筱颖并不知道何为“嫁娶”，也并不知道，这一别，又是何时再能相见。
<br/>
<br/>        她只记得，小卫离开的时候，篱笆上的五颜六色的牵牛花开得正艳。
<br/>
<br/>        有两只白色的小蝴蝶，在篱笆院墙上翩翩起舞，不知不觉中，飞向了蔚蓝的苍穹中。
<br/>
<br/>        童筱颖睁开眼睛的时候，屋内灰蒙蒙的一片。
<br/>
<br/>        窗户那边，有一到光线，从窗帘的缝隙间透了进来。
<br/>
<br/>        几点钟了？
<br/>
<br/>        童筱颖下意识地坐起身，从床头柜上摸到手机，看了看手机上的时间。
<br/>
<br/>        凌晨五点半……
<br/>
<br/>        原来，她做了一个真实的回忆梦。
<br/>
<br/>        小时候的事情，她忘却了很多，甚至还忘记了那个，发誓长大以后要娶她，带她离开那座大山的龙之卫。
<br/>
<br/>        也直到这一刻，童筱颖才记起，自己还有个二姨，二姨还有个生父不详的儿子。
<br/>
<br/>        揉了揉惺忪的睡眼，童筱颖再也睡不着了，便从床上下来，去了内卫洗漱。
<br/>
<br/>        她还是没能想起外婆的那本“蛊札”的去向。
<br/>
<br/>        或许，真的是因为那时候她太小了，所以记不起来了吧！
<br/>
<br/>        童筱颖为自己未能回忆起过去，而不能帮到连医生而感到愧疚。
<br/>
<br/>        人命关天，而她却无能为力。
<br/>
<br/>        早早地起床后，童筱颖在私宅的后花园里散步，却没想到会遇见大哥傅夜沉。
<br/>
<br/>        他穿着运动鞋，白色背心，白色七分裤，双手缠着白色绷带，在园子里对着木桩练搏击。
<br/>
<br/>        傅夜沉有着魔鬼般的肌肉，身段高大却不魁梧，棱角分明而修长，让他帅气英俊。
<br/>
<br/>        童筱颖渐渐地看出了神，看着傅夜沉的那张脸，恍惚间，她好像看到了傅昼景。
