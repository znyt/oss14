　　第423章
<br/>　　程大雷在蛤蟆城晃了一圈，最后拎着东西去找李婉儿了。
<br/>　　今天的颁奖仪式，李婉儿没有出现。程大雷要亲自把奖品交给她，事实上，这些日子程大雷尽量和李婉儿多接触，但在心底，程大雷是不愿和李婉儿见面的。
<br/>　　长安和琴川关相隔很远，消息传递到这里有一定的时间差。程大雷虽然不会卷入这场浑水中，但一直在收集长安方面的消息。
<br/>　　这几天，长安城那边的消息已传到这里，情况不是很好。至于正义教在长安城烧杀抢掠的事情，程大雷甚至不感到意外。在这个吃饭都成问题的冷兵器时代，程大雷不能期望每个人都有较高的道德素质。打仗为了什么，不就是为了抢钱抢粮抢女人么。如果不是吃不饱饭，谁会去信什么蛤蟆大神。
<br/>　　但消息对于李婉儿来说，简直糟糕透了。她的父亲明帝被杀，她的姊妹被人糟蹋，她的亲人成了刀下亡魂……程大雷自己都不知道，假若将自己换到李婉儿位置，自己会怎么做。
<br/>　　如今这些事还瞒着李婉儿，但以李婉儿的聪明，程大雷不相信她会一点察觉都没有。程大雷想多和李婉儿聊聊天，宽慰她的心情，但心里隐瞒这些事情，程大雷毕竟有几分心虚。
<br/>　　“哈，忙着呢！”
<br/>　　程大雷走进院子的时候，看到李婉儿坐在庭前读书，翻一本《山河堪舆考》。程大雷觉得读读书总是好事，心思放在书里，对于远在长安的事情就不关心了。
<br/>　　李婉儿抬头看了程大雷一眼，嘴角挂着一抹笑容。不知怎地，李婉儿表现得越平静，程大雷又心虚。她要是苦一阵闹一阵，程大雷反而能放松些。
<br/>　　“你今天怎么没去呐，我告诉你，今天可热闹了。秦蛮把关鱼都给击败了，关鱼不服，还要约时间再战。”程大雷把东西放下：“喏，悄悄这个，江南到的绸缎，在现在，搞到这样的货可不容易。”
<br/>　　在粮食满足后，程大雷开始发展轻工业，如今纸张、麻木蛤蟆城已能自己生产。
<br/>　　李婉儿手扶过绸缎，表达了感谢和赞美。不过，程大雷总感觉对方在捧着自己聊，身为明帝最宠溺的公主，李婉儿什么没有见过，这种绸缎在程大雷眼里是宝，恐怕在人家眼里，铺狗窝都嫌次。
<br/>　　然后，李婉儿抬起头问：“有长安城的消息没？”
<br/>　　“长安那边情况还好，反正就是……”程大雷注意到李婉儿的眼神，他本打算像以前一样，随便说些话蒙混过关，但此刻他不想再骗下去。
<br/>　　毕竟，迟早要面对的。
<br/>　　“长安已被攻破。”程大雷。
<br/>　　“喔，早该想到的。”
<br/>　　出乎程大雷预料，李婉儿的眼神很平静。甚至没有去询问攻城之后的事情，怕那些也是早能想到的。
<br/>　　程大雷准备说些安慰的话，却又觉得，说什么都是多余的。遭遇这种事，也不可能是凭几句话就能安慰的。
<br/>　　最后，程大雷什么话都没说，黯然离开了院子。某些事，终究要靠李婉儿自己面对，程大雷又不能以身替之。
<br/>　　好在李婉儿表现得很平静，若是从前的李婉儿，怕早已掀翻了天。不过程大雷倒是有些怀念以前闹腾的李婉儿。李行哉说帝国明玉公主，在草原上已经死了，怕只怕，在决定和亲的那一刻，曾经的李婉儿已经被她自己杀死了。
<br/>　　程大雷走在街道上，一个人心里想着心事。
<br/>　　“大当家，大当家……”
<br/>　　高飞豹从对面快步奔跑过来，面带焦急之色。
<br/>　　“怎么了？”程大雷有些心不在焉。
<br/>　　高飞豹最近这些日子有些憋屈，比骑术比不过樊梨花，单打独斗山寨又有一帮人压着他，他本想在这次比武大会上得个名次，但最后什么都没捞着，连带他的豹字军也有些抬不起头来。
<br/>　　他这么急忙忙过来，难道是要同自己说道说道这件事，可程大雷现在的心情，哪里有时间关这些。
<br/>　　“大当家，大事不好，我大哥来了！”
<br/>　　程大雷眼睛一亮，这才有了些精神。他瞥了对方一眼：“你大哥来了就来了，什么叫大事不好，现在人在哪儿呢？”
<br/>　　曾经并肩作战，感情自然和旁人不同，对于高飞虎赶到，程大雷心情也很高兴。
<br/>　　可高飞豹却苦着一张脸，道：“我大哥被堵到半道上，现在过不来了。”
<br/>　　“呃……被堵到哪里了？”
<br/>　　“凉州，牵牛谷。”
<br/>　　“走，去城主府谈，把人叫齐。”
<br/>　　程大雷出现在城主府的大厅，他自己坐在帅案后，在他左手边是刘悲、关鱼、张飞、赵子龙、樊梨花、高飞豹。在他右手边是徐神机、和珅、吴用等。
<br/>　　最近山寨没什么大事，程大雷又绝不是一个事必亲为之人，山寨还是很少有人凑得这么齐。
<br/>　　在程大雷面前，立着一个人，身上挂着伤。他名叫徐松，是高飞虎收的手下，这次是从包围圈中冲出来报信的。
<br/>　　“说吧，具体什么情况？”程大雷问。
<br/>　　徐松没见过程大雷，是高飞虎后来收的手下。当然，对程大雷的名字他并不陌生，知道这是自己老大的老大。
<br/>　　原来，如今天下大乱，江南也不安生。李星拥兵自重，想要割据称帝，而高飞虎这拨人马，也就成了他们的眼中钉肉中刺。
<br/>　　如今，高飞虎手下也有一万人马，算是江南一伙不小的势力。不知多少人盯着这拨人马眼馋，一天几波人过来当说客，想要把高飞虎拉拢到自己麾下。
<br/>　　高飞虎也看清楚局势不妙，要么给人当够，要么怕就是要被人当猎物吞下。
<br/>　　索性，高飞虎舍了江南的基业，来凉州投奔程大雷。
<br/>　　程大雷虽然足不出户，但天下也有他的名声，大家都知道他在凉州建了一座城池，成了琴川之主，凉州枭雄。
<br/>　　高飞虎这一路走得不算顺利，跋山涉水，终于抵达凉州。恰就在凉州边境被人拦住了，和凉州城的守军打了一仗，不敌，躲入了凉州牵牛谷。
<br/>　　所以才有徐松突破重围求援，要程大雷去解围。
<br/>　　
<br/>(本章完)
