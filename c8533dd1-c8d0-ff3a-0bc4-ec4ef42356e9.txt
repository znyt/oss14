　　主府之外。
<br/>　　一众武者，在西门孤夜的冰冷气势下，瞬间止步。
<br/>　　西门孤夜，可是十八府天骄中排名第三的强者。
<br/>　　一身圣王境九重修为，足以傲视在场任何武者。
<br/>　　一身实力，更是足以达到圣皇境以上。
<br/>　　这些普通的圣王境武者，自是不敢再靠近主府半分。
<br/>　　“萧逸，你呢？”西门孤夜，冷眼看着主府内的萧逸。
<br/>　　虽语气不如对普通武者时的强硬，但同样带着冰冷。
<br/>　　唤作‘月姐姐’的女子，亦眼眸冰冷，“萧逸公子，我劝你不要触及我天星府与天煞府的底线。”
<br/>　　萧逸充耳不闻，也懒得理会，他现在正皱眉着。
<br/>　　因为，他的手掌搭在石坛上，竟连半分星光力量都吸摄不了，更别说修炼了。
<br/>　　“怎么回事？”萧逸皱了皱眉。
<br/>　　主府门外，公孙火舞冷笑一声，“萧逸小贼，你打的倒是好算盘。”
<br/>　　“一直蛰伏尾随，特地在主府出现后才暴起，并拿出遗失的天星盏。”
<br/>　　“可你根本控住不了天星石坛的禁制，也吸收不了星光力量，任你百般算计，又有何用？”
<br/>　　“除了天星府嫡系族人的鲜血，谁也操控不了天星石坛…”
<br/>　　公孙火舞得意地说着，但话还未说完，已被两双冰冷的目光止下。
<br/>　　“愚蠢。”西门孤夜冷喝一声。
<br/>　　“笨蛋。”月姐姐剐了公孙火舞一眼。
<br/>　　“糟。”公孙火舞连忙捂住了嘴巴，脸色一变。
<br/>　　“呵呵。”萧逸玩味一笑，“原来如此吗。”
<br/>　　萧逸话音落下的瞬间，顾怜星心头闪过一丝不安的预感。
<br/>　　而这抹预感，仅在一秒后便实现了。
<br/>　　萧逸一把捉住了顾怜星的手臂。
<br/>　　“有劳怜星姑娘了。”萧逸轻笑一声，指尖一道锋锐剑气凝聚。
<br/>　　剑气的速度极快，快得远远超过了在场任何人的反应。
<br/>　　但众人反应过来时，顾怜星那纤纤素手上，已被划出一条血痕。
<br/>　　血痕之内，鲜血缓缓而流。
<br/>　　“你…”西门孤夜瞬间暴走。
<br/>　　“怜星。”公孙火舞惊呼一声。
<br/>　　两人，瞬间出手。
<br/>　　“好猖狂的小贼，修得伤我家少府主。”一众天星府武者，同样暴起。
<br/>　　萧逸眼眸一冷，“叨叨个没完，聒噪。”
<br/>　　“滚出去。”
<br/>　　萧逸大手一挥，如舞剑气顷刻打出。
<br/>　　剑气之下，无一合之敌。
<br/>　　西门孤夜、公孙火舞以及天星府武者，瞬间被轰出主府。
<br/>　　轰…
<br/>　　主府大门上，一块巨石，轰然落下，封住了大门。
<br/>　　主府之外。
<br/>　　众人看着忽然的变故，霎时愣住。
<br/>　　西门孤夜、公孙火舞等人，面露无比忧色。
<br/>　　“该死，主府入口被封住了。”西门孤夜面若寒霜。
<br/>　　“混蛋，萧逸小贼，快把怜星放出来。”公孙火舞疯了般冲向巨石。
<br/>　　手中鞭索，化作一道道红色飞舞，连绵不断地轰击着巨石。
<br/>　　不过，巨石乃是主府的一体；以她实力，怎可能撼动分毫。
<br/>　　“混蛋。”公孙火舞脸色的怒色，头一次飙升到极点。
<br/>　　“萧逸小贼，若怜星有半分损伤，我公孙火舞发誓，天索府必与你不死不休。”
<br/>　　一众天星府武者，面露愧色。
<br/>　　“都怪我们修为低下，保护不了少府主。”
<br/>　　“若少府主有半分闪失，我们万死难辞其咎。”
<br/>　　月姐姐摇了摇头，面容冰冷，“怜星太善良了，之前就不该信这萧逸小贼的。”
<br/>　　“若之前就不让这小贼跟来，便不会有现今之事。”
<br/>　　“天星禁地内的浑厚星光，乃是她的续命之物，她自己不谨慎些，倒害了自己。”
<br/>　　西门孤夜咬了咬牙，“浑厚星光还是小事。”
<br/>　　“那萧逸，本就臭名昭著，劣迹斑斑。”
<br/>　　“怜星一人孤身在主府内，就怕这萧逸贼人做出些什么不轨之事。”
<br/>　　西门孤夜脸上，尽是担忧之色。
<br/>　　公孙火舞面庞涨得通红，一双拳头握得极紧，指甲陷入皮肉中，却丝毫未觉。
<br/>　　……
<br/>　　主府内。
<br/>　　顾怜星对于忽然的变故，同样愣了愣。
<br/>　　只是，一张苍白的面庞上，无丝毫惧怕之色，只是皱眉看着萧逸。
<br/>　　萧逸注意到这抹目光，皱眉道，“怎么？”
<br/>　　顾怜星微微蹙额，“怜星只是不相信会看错人。”
<br/>　　“呵。”萧逸笑笑，再次捉过顾怜星的手臂。
<br/>　　那只如凝脂般白嫩的手臂，很美。
<br/>　　“你想做什么？”顾怜星这时才露出一丝慌色。
<br/>　　“你说呢？”萧逸反问一声，指尖轻动。
<br/>　　一道元力，从指尖迸发，随后一滑。
<br/>　　顾怜星手臂上的血痕，瞬间消失。
<br/>　　下一秒，萧逸捉着顾怜星的手，搭在了石坛上，随后松手。
<br/>　　石坛上，还留着顾怜星之前手臂上滴落的几滴鲜血。
<br/>　　萧逸看了一眼，走到另一边，将手搭在石坛上，随后闭目，开始了修炼。
<br/>　　早在顾怜星手臂上的血液滴落石坛时，萧逸就已经能通过这几滴鲜血操控这主府内的禁制。
<br/>　　不过操控的程度并不高。
<br/>　　如刚才那落下的石门，是主府的保护禁制之一。
<br/>　　至于其他的，萧逸则操控不了。
<br/>　　如果没猜错的话，应该是那第18盏天星盏，乃是天星府的先辈所拿，并烙印了气息。
<br/>　　故天星府的后人，能通过血脉力量，操控一丝禁制。
<br/>　　而主府的这整个禁制，乃是由18盏上品圣器级别的天星盏所加持。
<br/>　　若想掌控整个禁制，需得掌控18盏天星盏。
<br/>　　而想掌控18盏天星盏，又需得破掉禁制。
<br/>　　这根本是一个死循环，只有远远高过这个禁制的绝对实力，才能无视这一切。
<br/>　　而现今，能掌控的天星盏，只有一盏；自然的，能操控的禁制程度，也只有一丝。
<br/>　　不过够了，能启动修炼的效果就行。
<br/>　　另一边，顾怜星手掌搭在石坛上，感受着源源不断涌入体内的星光力量，亦惊疑地看着萧逸。
<br/>　　当看到萧逸闭目修炼，并未理会她时，她松了口气，亦会心一笑。
<br/>　　随后，亦闭目，开始了修炼。
<br/>　　萧逸这边，同样感受到了那惊人的浑厚星光力量。
<br/>　　不过，这些浑厚的星光力量，他并未用作吸收，未用作提升修为。
<br/>　　而是用作…感悟星幻剑道。
