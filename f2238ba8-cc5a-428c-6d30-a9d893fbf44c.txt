　　萧逸伸出手，拿过那枚戒指。
<br/>　　戒指入手，包裹着它的光芒，瞬间涌入萧逸脑海。
<br/>　　那是留在禁制中的气息，也是裂天剑魔留下的信息。
<br/>　　如无意外，包裹这枚戒指的光芒。
<br/>　　以及那本书籍的光芒。
<br/>　　是最后的两道信息了。
<br/>　　待得那道光芒蕴含的信息被萧逸消化。
<br/>　　萧逸顿时面露惊骇，随即便是狂喜。
<br/>　　对，狂喜。
<br/>　　比之刚才成就地元金丹，突破地元二重，更为惊喜。
<br/>　　因为，这枚戒指，竟是元器，而且位列极品。
<br/>　　名为怒炎戒。
<br/>　　正是炎武王国历史记载上的，唯一一件，并已失传的极品元器。
<br/>　　据裂天剑魔留下的信息所讲。
<br/>　　怒炎戒，是他在某处远古遗迹中找到。
<br/>　　得到以来，从未用过。
<br/>　　实在是没机会用。
<br/>　　他刚得到，便已经准备前往火焰圣山闭生死关了。
<br/>　　所以，世人并不知这唯一一件极品元器在他手上。
<br/>　　另外，怒炎戒，虽是历史上有记载。
<br/>　　但，也仅仅是记载了名字。
<br/>　　至于其它信息，则毫无记载。
<br/>　　实在是怒炎戒，出现的年代太过久远了。
<br/>　　久得，只剩下名字还遗留在历史长河中。
<br/>　　萧逸喜悦地将其待于手指上。
<br/>　　随后，满意地点了点头。
<br/>　　怒炎戒，表面看起来，除了图案和形状有些霸气之外。
<br/>　　便只有平平无奇。
<br/>　　戴于手中，外人顶多会以为是一个装饰。
<br/>　　自己倒是可以随便戴着。
<br/>　　至于其功效，便是增幅火焰威力。
<br/>　　任何火焰，受它增幅而出，都会化为怒焰滔天，威力剧增。
<br/>　　简单来说，它与烈焰手套，功效一样。
<br/>　　只不过，怒炎戒，是最强大的极品元器。
<br/>　　烈焰手套，则只是下品灵器。
<br/>　　两者的效果，一个天，一个地。
<br/>　　火焰的增幅威力，更是云泥之别。
<br/>　　当初，萧逸还打算将烈焰手套，锻造为极品灵器。
<br/>　　没想到，现在得到了远远凌驾于极品灵器之上的极品元器怒炎戒。
<br/>　　而除了增幅火焰威力外。
<br/>　　怒炎戒，还有一个功效。
<br/>　　如乾坤戒一样，可储存物品。
<br/>　　萧逸释放感知入内，看了一眼，顿时倒吸一口凉气。
<br/>　　里头的空间，竟大得骇人。
<br/>　　怕是一座巨大高山，都能装进去。
<br/>　　极品元器，已经是天地间最珍贵，最强大的武器了。
<br/>　　其效果，自然是堪称恐怖。
<br/>　　顺带一提，怒炎斩与怒炎戒，没有丝毫关系。
<br/>　　只是裂天剑魔将自己创造的武技，起了个与怒炎戒一样的名字罢了。
<br/>　　戴好怒炎戒。
<br/>　　萧逸将手伸向第二道光芒，那本书籍。
<br/>　　光芒，化作信息，传入脑海。
<br/>　　待得完全消化了这道气息，萧逸的表情，陡然间变得惊骇无比。
<br/>　　甚至比刚才得到怒炎戒更为惊骇。
<br/>　　书籍的表面，写着五个大字，九阳轮回诀。
<br/>　　毫无疑问，这是一门功法。
<br/>　　当年，裂天剑魔之所以选择从火之一道突破。
<br/>　　很大程度是因为这门功法。
<br/>　　这门功法，同样是裂天剑魔在那远古遗迹中得到。
<br/>　　书籍上，并没有标明功法的品阶。
<br/>　　裂天剑魔也不知。
<br/>　　只是知道，这门功法，极其强大，但也无比玄奥。
<br/>　　极难修炼。
<br/>　　连裂天剑魔这等惊才绝艳之辈，也无法修炼成功。
<br/>　　其功效，乃是将世间一些无比强悍的火焰，融合起来。
<br/>　　九，为数之极。
<br/>　　九种世间最强火焰，一旦集齐，便成轮回。
<br/>　　届时，威力滔天，甚至可焚毁天地。
<br/>　　让世间万物，重归轮回。
<br/>　　“嘶。”萧逸倒吸一口冷气。
<br/>　　他无法真正感受到这门功法的强大，他的修为和境界，还达不到那个层次。
<br/>　　但，功法的寥寥只言片语描述。
<br/>　　却已经足够证明这门功法的恐怖与惊人。
<br/>　　难怪当年裂天剑魔宁愿放弃自己擅长的剑道。
<br/>　　转而苦修火之一道。
<br/>　　要练成这门功法，首先，便是拥有世间最强悍的火焰。
<br/>　　裂天剑魔的‘霸炎剑’武魂，赋予的霸绝之火。
<br/>　　只算得上是堪堪达到要求，勉强可列为第一种。
<br/>　　裂天剑魔来火焰圣山闭关的第二个原因，便是要找寻第二种火焰。
<br/>　　这第二种火焰，就在这火焰圣山中。
<br/>　　名为地脉金火。
<br/>　　地脉金火，传言乃是游走于地底岩浆的一种神火。
<br/>　　存在的岁月，无比漫长。
<br/>　　似有灵智，顺着岩浆，自动游走于地脉。
<br/>　　裂天剑魔，花了极大的功夫，才查到它的踪迹。
<br/>　　萧逸皱眉思索了一下。
<br/>　　九阳轮回诀，如此神妙的功法，没有人不想修炼。
<br/>　　所谓富贵险中求。
<br/>　　萧逸打算拼一把，去寻找一番。
<br/>　　他如今的敌人，太多了，他迫切需要实力。
<br/>　　当然，他只是想尝试一番。
<br/>　　若是没有把握，他也不会拿自己的性命开玩笑。
<br/>　　想罢，萧逸转过身，离开了禁制。
<br/>　　果然，一离开禁制，那密布的熊熊大火，瞬间朝他袭来。
<br/>　　“嘭”的一声。
<br/>　　萧逸大手一挥，一股紫炎，笼罩了自己的身躯。
<br/>　　若是之前重伤状态，他自然无法抵抗这些火焰。
<br/>　　但如今，伤势已恢复。
<br/>　　且修为大增。
<br/>　　加之，紫晶灵炎，焚毁万物，形成的屏障，自然可以抵挡这些火焰。
<br/>　　当然了，萧逸体内的元力，也在快速消耗着。
<br/>　　按萧逸的估计，自己大概有1个时辰的时间。
<br/>　　算上要回到禁制内。
<br/>　　自己便只有半个时辰去寻找。
<br/>　　若再加上稳妥起见，得将更多的元力留在回归的路上。
<br/>　　那么，自己的时间，绝对不足半个时辰。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，飞速疾驰。
<br/>　　20分钟后。
<br/>　　他来到了圣山顶峰。
<br/>　　顶峰，原来竟是一个巨大的火山口。
<br/>　　而且是活火山。
<br/>　　里头，岩浆涌动，极为骇人。
<br/>　　那层层热浪，哪怕萧逸站在火山口边缘，也是一阵心慌。
<br/>　　据裂天剑魔留下的信息，他当初就是在这里找到了地脉金火。
<br/>　　并将其禁锢了起来。
<br/>　　如今，数百年过去了。
<br/>　　也不知那地脉金火破了禁制没有，又离开了没有。
<br/>　　要知道，整条地脉，贯穿整个大陆地面最深处。
<br/>　　地脉金火，四处游走，几乎无法再找到它的踪迹。
<br/>　　然而，萧逸的运气，似乎不错。
<br/>　　当年裂天剑魔的手段，也确实厉害。
<br/>　　数百年过去了，他的禁制，并没有被地脉金火冲破。
<br/>　　当然，禁制已经很弱了。
<br/>　　估计自己晚来几年，它就冲破禁制跑了。
<br/>　　......
<br/>　　第三更。
<br/>　　我记得昨天欠一更，但暂时无法补。
<br/>　　只能说一声抱歉，迟些天补回来。
