　　这武者突然出手，向着楚风眠发难而来。
<br/>　　周围的其他武者，都是站在原地，没有任何一人有着想要出手的心思，不过他们的目光，也都是在看着楚风眠。
<br/>　　楚风眠的名字，他们其中的一些人，也都已经听说过了。
<br/>　　敢以一人之力，斩杀七大宗门弟子，这样疯狂的行为，在中域之中已经是广为流传了。
<br/>　　这些人的目光，都在注视着楚风眠，似乎也是想要看着楚风眠的一举一动，看出楚风眠的真正实力来。
<br/>　　就连那坐在王座上的洛汐，如今都是平静的看着这一切，似乎是这一切，都跟她没有什么关系一般。
<br/>　　“小子，听说你在归墟天中杀了一堆七大宗门的弟子？我告诉你，七大宗门的弟子，在我们六道魔门的面前，也都是废物，今日我倒是要看你到底有什么本事！”
<br/>　　这武者看着楚风眠，不由分说，立刻出手。
<br/>　　这一拳向着楚风眠的胸口，猛然轰杀了过去。
<br/>　　“死！”
<br/>　　楚风眠站在原地，脸色甚至都没有因为这男子出手而产生任何一点的变化，他的嘴角露出一抹轻蔑笑容，一股灵力猛然爆发而起。
<br/>　　那武者这一拳的拳风，在即将接近到楚风眠的身躯的一刻，却是瞬息之间，突然停了下来。
<br/>　　在距离楚风眠身边三尺左右的地方，这拳风便是停了下来，甚至是在往前一点都无法靠近。
<br/>　　楚风眠就这样平静的站在了原地，任由这男子出手，楚风眠都毫发无伤。
<br/>　　“这是什么力量！”
<br/>　　那武者也感受到，在靠近楚风眠的身躯一刻，便是有着一种巨大无比的压力。
<br/>　　在这压力的面前，他几乎便是寸步难行，无法靠近，这让这武者的眼神中，也露出了几分认真。
<br/>　　楚风眠的实力，比其他想象的还要强大许多。
<br/>　　“小子，你现在挣扎的在多，也没有任何意义！”
<br/>　　这武者看着楚风眠，脸上露出几分狰狞，他的拳风突然变化这，在这宫殿之中的魔意，突然之间全部是被这武者所吸引过来，所有的魔意，都是涌入到了他的这拳风之上。
<br/>　　一拳打出，群魔显现，无数魔头的力量，都是加持在了上面。
<br/>　　在这宫殿之中，到处都是魔意，这种魔意，正好是最为完美的补品，来增添这武者的力量。
<br/>　　在这一拳打出的刹那之间，楚风眠也是突然叹息了一声，出手了。
<br/>　　楚风眠这一出手，一股灵力便是猛然挥了过去，这灵力之中，蕴含远古战龙之力，这男子打出一拳，其中蕴含的魔意，在这一股灵力面前，全然破碎。
<br/>　　就是随意出手，一股灵力挥出，这男子的一身魔意，全部都是被打的粉碎，甚至他的身躯，都在开始消亡。
<br/>　　“啊！”
<br/>　　随着一声惨叫，这武者的身躯，开始节节破碎，甚至都没有来得及如何挣扎，身躯便是彻底泯灭。
<br/>　　就算是生死境的境界，这样身躯泯灭，也是已经死的不能在死了。
<br/>　　“这是什么实力？一招之间，就将吕狼轻易轰杀了？”
<br/>　　“好强的实力！轻描淡写之间便是做到了这一切，此子的实力，比起传闻之中的甚至还要可怕。”
<br/>　　“一个生死境巅峰的武者，哪里来的如此可怕的实力！”
<br/>　　“若是此人超越了生死，那他的实力又该到一种什么地步！”
<br/>　　宫殿之中的那些六道魔门的弟子，看着楚风眠，都是一个个惊声起来。
<br/>　　这武者在他们这一群人中，虽然只能算是中下等的，可是依然是有着足以斩杀寻常老祖的实力。
<br/>　　在他们这些人看来，就算是不能够足以将楚风眠击败，也足以是将楚风眠的真正实力逼出来的，可以跟楚风眠抗衡一二才是。
<br/>　　可没想到，居然是如此轻易之间，这武者就被楚风眠随意轰杀了，甚至是楚风眠身上的灵力都似乎没有什么波动，如同是刚刚踩死了一只蝼蚁一般的感觉。
<br/>　　甚至是在场的一些超越生死的老祖，如今看向楚风眠，眼神中都是深深的忌惮。
<br/>　　楚风眠现在表现出来的实力，可以说仅仅只是他实力的冰山一角，就算是那些超越生死的老祖，现在都没有人愿意去面对楚风眠。
<br/>　　宫殿之中，一片寂静。
<br/>　　“厉害，楚公子的实力，果然比起传闻中的还要可怕。”
<br/>　　就在这寂静之中，坐在最高处王座上的洛汐，缓缓的开口道。
<br/>　　“吕狼这个废物，挑衅楚公子，被楚公子杀了，也是他死有余辜，还请楚公子入座。”
<br/>　　洛汐轻描淡写的开口道，根本不去管这吕狼的死活。
<br/>　　她手一挥，在楚风眠的面前，便是一方王座便是凝聚出来哦，跟这洛汐的王座，正在对面。
<br/>　　楚风眠看了一眼，便是身躯一动，一下子坐在了这王座之上。
<br/>　　“洛汐姑娘今日来，可不是为了试探楚某的实力而来的吧。”
<br/>　　楚风眠坐在王座上，看着洛汐，缓缓的开口道。
<br/>　　“六道魔门从新出世，不知道洛汐姑娘是这六道魔门的哪一位魔主呢？”

<br/>　　魅魔尊主。
<br/>　　这洛汐，如今便是六道魔门之中，魅魔宗的尊主了。

<br/>　　楚风眠随意笑了笑，这洛汐的身份，他本来也已经猜测的八九不离十了，如今只是确认一下罢了。
<br/>　　“说吧，你们六道魔门这一次卷土重来，到底是为的身份。”

<br/>　　就在楚风眠开口之际，站在一旁的一尊老祖，猛然怒喝道。
