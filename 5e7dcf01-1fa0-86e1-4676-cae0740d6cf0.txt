　　“破碎！”
<br/>　　楚风眠缓缓起身，轻喝一声，他的身躯就凌空突然破碎了，十二万九千六百枚血肉晶体，都是四散飞去，就像是被一股巨力所凌空打爆一样。
<br/>　　寻常九劫古帝，甚至是半步天人境武者，一旦是被打爆身躯，最轻也需要耗费大量的资源，闭关修炼数年，才可以恢复伤势。
<br/>　　如果一连被打爆数次身躯，都有着境界跌落的可能。
<br/>　　不过楚风眠现在凝练了不朽真身，纵然是被打爆数十次身躯，都不会有任何影响。
<br/>　　这就是不朽真身最为逆天的一点。
<br/>　　楚风眠的目光，仔细观察着一枚枚的血肉晶体，旋即楚风眠又是开口轻喝一声。
<br/>　　“重塑！”
<br/>　　十二万九千六百枚血肉晶体，在这一刻开始纷纷凝聚起来，瞬息之间，便是凝聚为了一体，重新化为了楚风眠的模样。
<br/>　　这十二万九千六百枚血肉晶体，就是现在楚风眠的根本，只要是这十二万九千六百枚血肉晶体没有被摧毁，楚风眠纵然是被打爆千百次肉身都不会陨落。
<br/>　　“重新凝聚的速度，的确是快了不少。”
<br/>　　楚风眠看着重新重塑的肉身，心中低声自语道。
<br/>　　现在楚风眠身上的十二万九千六百枚血肉晶体，在融入了藏金精跟六翼蜈蚣王之后，果然是有着极大的提升。
<br/>　　起码是比起之前，楚风眠重塑身形的速度，要变快了数倍。
<br/>　　这重塑身形的速度越快，楚风眠也就越是可以在面对强敌的时候，占据优势。
<br/>　　“不朽真身的极致，还从未有人达到过，不知道这真正的极致，是否可以天地不朽？”
<br/>　　楚风眠不由的想到。
<br/>　　楚风眠感觉得到，就算是青铜天魔这样传说之中的万魔之祖，他的肉身，不朽真身，都依然是没有达到最极致的地步。
<br/>　　天下之间，只怕是还无人达到过这个极致，楚风眠现在的不朽真身，距离青铜天魔那个层次，都相差不少。
<br/>　　不过楚风眠还无比年轻，在古籍记载之中天人境之下的武者，能够凝炼不朽真身的，是少之又少。
<br/>　　但是一旦是在天人境之下的境界，凝炼出不朽真身来，那未来的成就便是都极高。
<br/>　　踏入天人境，是一种蜕凡化仙的蜕变，如果在半步天人境，甚至是帝尊境的时候，就凝炼出不朽真身来，便是足以在踏入天人境的一刹那间，得到无比巨大的好处。
<br/>　　这种好处，比起从生死台阶第九重，不朽之境直接踏入古帝境界，成就绝世古帝带来的好处还要巨大。
<br/>　　所以一些历史之中妖孽的天才，许多都是尝试再半步天人境的时候，凝炼不朽真身。
<br/>　　像是少帝，也是如此。
<br/>　　以少帝的资质，如果说一心是想要突破的话，只怕是早就已经踏入到了天人境了。
<br/>　　可是少帝为了凝炼不朽真身，却是压制着自己实力的增长，所以现在还只是半步天人境的境界。
<br/>　　这样的举动，在天才妖孽之中，并不少见。
<br/>　　凝炼不朽真身的难度，可是要比起真正度过天人大劫，踏入天人境，还要更加的困难。
<br/>　　起码楚风眠知道的，那荒古门的上一任门主，荒天王，他也是想要有着凝炼不朽真身，在踏入天人境的想法。
<br/>　　不然的话，在这万年之间内，荒天王只怕也足以踏入天人境了。
<br/>　　虽然哪怕是踏入到了天人境之中，也依然有着凝炼不朽真身的机会。
<br/>　　可失去了一次蜕变的话，在天人境之中凝炼不朽真身的武者，实力却是远不如那些在半步天人境凝炼了不朽之身，然后在踏入天人境的武者。
<br/>　　楚风眠现在的不朽真身，凝炼的越强，等到楚风眠真正踏入到了天人境的一刻，得到的好处也就越是巨大。
<br/>　　“可惜琉璃净水，都是几乎消耗完了，如果可以将那太初剑锋之中的魔意消除，就可以重新让太初剑变的完整起来，倒时候一件极道仙兵相助，也许不用踏入天人境，我就足以斩杀天人了。”
<br/>　　楚风眠看着那剩下的一滴琉璃净水，感叹了一声。
<br/>　　楚风眠毕竟是一尊剑修，以剑道为主，太初剑，乃是曾经第一剑帝的佩剑，要是可以让楚风眠得到的话，楚风眠凭借太初剑，斩杀天人境，都有可能。
<br/>　　毕竟极道仙兵的力量，楚风眠从人皇罐之上，就已经是见识过了。
<br/>　　而且人皇罐，还只是一件残破的极道仙兵，真正的力量，能够发挥出的力量，十不存一。
<br/>　　要是得到完整的一件极道仙兵，尤其还是灵剑的话，足以是让楚风眠的实力，提升无数。
<br/>　　不过在那之前，楚风眠必须是还要找到大量的琉璃净水，来炼化太初剑锋之上的魔意才行。
<br/>　　不然的话，这一切可都是空谈，不过琉璃净水，楚风眠能够从沧海都这里，得到四滴，已经是极大的幸运了。
<br/>　　除了沧海都，现在楚风眠的脑子里都有些无法想象，还有着在那一个地方，还存在琉璃净水。
<br/>　　楚风眠一边想着，一边便是离开了这宝库之中。
<br/>　　对于其他琳琅满目的宝物，楚风眠都没有放在眼中，也没有去取。
<br/>　　他从五行天，齐林山之中的收获已经够多了。
<br/>　　既然已经是从这沧海都的宝库之中得到了琉璃净水，那其他的东西，就都留给沧海都吧。
<br/>　　楚风眠离开了宝库之中，海公主正在宫殿外站着，等待着楚风眠。
<br/>　　看着楚风眠出来表情有些凝重，海公主有些紧张的问道。
<br/>　　“难道没有琉璃净水？”
<br/>　　沧海都的宝库之中，拥有琉璃净水，其实也只是一个传闻，海公主都不敢确定。
<br/>　　“有，不过不够，太少了。”
<br/>　　楚风眠摇了摇头，脸上露出几分失望之色，旋即看向了海公主开口道。
<br/>　　“我答应你的事，无妨，说吧，我答应你一个条件，你想要我做什么？”
<br/>　　虽然这宝库之中的琉璃净水不多，但是既然是有，楚风眠答应海公主的条件就不会变，可以帮海公主做一件事。
<br/>　　这既是为了琉璃净水，也算是回报海公主的恩情了。
