　　如果你不念，我就弄死他们。
<br/>　　假如没有这一番背叛与救赎，叶少阳对他们也不会用什么深刻的感情——他自己也不是圣母婊，在生死攸关的瞬间，他最先考虑的肯定还是自己和身边亲人。
<br/>　　但这些人之前的行为，让他感动，不说什么生死患难的兄弟，至少……自己不可能看着自己这些虔诚的追随者们死在自己面前。
<br/>　　如果是大战一场，有死伤也是正常的。但偏偏他们的性命全捏在自己手中——自己一个选择，就能决定他们的生死。
<br/>　　这才是最最可怕的地方的。
<br/>　　影魅是极聪明也是极可怕的一个人。叶少阳猜的都没错，他费心布置了诛仙剑阵，但一直磨磨唧唧地没有真正动手，正是因为，他压根没想过杀叶少阳。
<br/>　　他想让他屈服——这比杀了他要难十倍，但效果也好十倍。
<br/>　　他本来是想用叶少阳身边的人——小马、吴嘉伟他们的性命来要挟叶少阳，但是想来没用，叶少阳身边这些人都是亡命之徒，是不怕死的，就算叶少阳要为他们妥协，这些人也肯定不会干。
<br/>　　所以，只有用这些陌生的生灵。
<br/>　　叶少阳也能看透这一切，却无能为力——他不可能眼看这上百号人因为自己而死，就算自己冷血到可以这么做，这件事在将来也一定会被自己牢牢记住，污染自己的道心，自己虽然已经是上仙了，但还不够啊，他的目标可是三界第一，虽然很遥远，但他绝对不希望眼下就停步的。
<br/>　　然而，假如跟着念咒，在旁人看来也没什么，无非就是认怂，以叶少阳的性格，认怂是不怕的，就算影魅让他叫爸爸，逼到杠子上那也是能叫的，关键是……那几个盘古僧念的是接引咒，是接引信徒用的咒语，凡是念了这个咒语，就相当于皈依了。
<br/>　　更不要说，自己是道士，假如皈依佛门……不光是当了道门的叛徒，自己内心这一关也过不去。这跟学习佛门法术还不一样，性质要恶劣得多，相当于叛教。
<br/>　　况且那几个盘古僧肯定在咒语中夹带了私货，自己念咒之后，神念中会发生什么变故，都是不可预料的。
<br/>　　只是……这两样必须得选择一样了。
<br/>　　在他思考做选择的工夫，影魅带着玉玑子等人，开启了诛仙剑阵的第三式：上上下下的电弧都连在了一起，形成了一片剑影，但阴阳罡风没停，将剑影裹在中间，从两边不断滚动，看上去就像是一个巨大的绞肉机，而那两股旋风便是绞肉机的叶片，对被困在中间的生灵，展开了全方位无死角的进攻。
<br/>　　这些剑影，并不是真的剑，是直接针对神识的幻象攻击，不断切割着人的神识，身在阵中的所有生灵都不能避免。
<br/>　　这种感觉，就像是一把钝刀，在脑子里慢慢地来回切割……
<br/>　　“啊……”
<br/>　　一时间惨叫连连，许多生灵都抱着脑袋，痛苦地在水中翻腾起来，有些意志力强一点的，还在苦苦坚持，脸上表情也十分难看。
<br/>　　叶少阳几人意志力都相当顽强，一时间还能支撑，但这种切割神识的痛苦是持续性的，就算再坚强的人，也早晚有挺不住的那一刻。
<br/>　　“啊哈哈哈……呜呜……”
<br/>　　有生灵神识崩溃，彻底陷入疯狂，在水中翻滚着，一会痛哭一会狂笑。
<br/>　　发疯的生灵越来越多。
<br/>　　叶少阳一咬牙，冲影魅说道：“若我念咒，你能放了他们？”
<br/>　　“他们于我，只是要挟你的手段，我杀他们做什么？”影魅倒也坦白，对几个盘古僧又扫了一眼，点了一下头，几个盘古僧又从头开始念起接引咒：“彼时，八荒之内，鬼魅魍魉横行，万物有灵，生而化之，是为佛念，万物之善，是为佛心……”
<br/>　　叶少阳本来想跟着念的，但听到这几句，立刻犹豫起来，这最前面几句经文，就已经跟道教的观念大相庭径，自己要是念这个，就等于是自我否定了。
<br/>　　又是一阵怪叫声响起，又一个人疯了。
<br/>　　也属于一咬牙，朗声念道：“彼时，八荒——”
<br/>　　“不能念！！”
<br/>　　小马突然扑上来，双手一下捂住他的嘴里，骂道：“你傻啦，你知道念咒代表什么，你想自己整个人都毁了是不是……”
<br/>　　我特么总不能看他们死啊！
<br/>　　叶少阳心中大骂，一把推开小马，继续往下念，但只念了两个字，突然天空一道炸雷，震得人两耳轰鸣，在这一瞬间，所有人都怔住了。
<br/>　　这是幻象空间，不是人间，连太阳月亮都没有，哪里来的什么雷声？
<br/>　　回过神来之后，所有人一起抬头望去，在遥远的西边，遥远的天空中，突然出现了一块巨大的黑斑，就好像一个漩涡，不断扩大，不断有雷声传来，之后漩涡逐渐缩小，彻底消失了。
<br/>　　“有人闯进来了！来的人很多！”鲲鹏望着恢复原状的天边说道。
<br/>　　“快，快去调查一下！”
<br/>　　影魅沉声说道。
<br/>　　身后树丛之中，立刻有几道人影腾空而起，都是跟影魅一样的黑影，速度极快，消失在远处山野间。
<br/>　　“接着念，快，叶少阳，接着念！”鲲鹏大声催促道。
<br/>　　叶少阳心中燃起了希望，提醒大家再坚持一会。
<br/>　　经过这次变故，鲲鹏等人作法的节奏也被打乱，法阵之内，飞剑产生的速度也慢了许多，大伙多少也算是喘了一口气，在叶少阳的鼓舞之下，咬牙支撑着。
<br/>　　而那些之前已经疯掉的生灵，逃出莲叶的覆盖范围，堕入水中，又被巨浪带到空中去，立刻就被撕碎了身躯……
<br/>　　一道黑影从远山飞来，顷刻间就到了影魅身边，耳语了几句，影魅大惊失色，“什么！”
<br/>　　鲲鹏忙问：“怎么了师伯？”
<br/>　　影魅目光闪动，似乎在犹豫，对鲲鹏沉声说道：“速度杀光所有人，擒住叶少阳！”
<br/>　　鲲鹏大惊，在神识中说道：“师伯，只要再撑一会，那叶少阳便屈服了，只要他跟着念经，那便只有做我们的人……何故放弃？”
