　　“倒也不必这么急。”
<br/>　　老者见萧逸点头，松了口气。
<br/>　　“前往总殿，路途不近，你养足精神，明日一早出发便是。”
<br/>　　“也行。”萧逸点点头。
<br/>　　前往总殿参加传承试炼，这是前些天来风刹殿时，答应这位主殿主的事。
<br/>　　风刹殿，自上古以来，便是司职搜索情报的一殿。
<br/>　　这里是天藏地域。
<br/>　　自然的，这座主殿内，有天藏学宫的情报资料。
<br/>　　不过，天藏学宫即便放眼中域，也是超然势力。
<br/>　　故萧逸即便是副殿主头衔，却无权限查阅其内长老以上的情报。
<br/>　　还是多得多主殿主借他权限，才调阅了里头卷宗资料。
<br/>　　而作为条件，便是尽快前往风刹殿总殿，参加传承试炼。
<br/>　　至于这传承试炼，原本上次在四方域时，萧逸只以为是什么考核。
<br/>　　原来是一份传承上的考核。
<br/>　　这种传承上的考核，萧逸再熟悉不过了。
<br/>　　当初在东域猎妖殿时，萧逸便获得了天火圣者的传承，也得了一门领域武技，天火印。
<br/>　　“我说萧逸小子。”老者瞥了眼萧逸。
<br/>　　“让你去总殿参加传承考核，那是莫大好处，又不是让你去闯什么刀山火海。”
<br/>　　“别人求都求不来。”
<br/>　　“你倒好，一拖再拖。”
<br/>　　风刹殿的令牌，自有禁制在内。
<br/>　　除了记录着殿内武者的身份、气息等等之外，还与各殿的亚圣器有联系。
<br/>　　这也是为什么任何一个殿内武者，出示令牌后都能在各座分殿证明自己身份的原因。
<br/>　　而萧逸的身份记录等等，皆是总殿那边处理。
<br/>　　故他前些天刚来风刹殿出示令牌后，总殿便已有显示，也发了传令过来，让他尽早前往总殿。
<br/>　　其它七殿也一样。
<br/>　　如果萧逸拿着他修罗殿的令牌前往修罗殿分殿的话，修罗殿总殿要寻他，亦是一道传令的事。
<br/>　　这时，萧逸摇了摇头，脸色平淡，“原本是有些事，所以未能确定时间。”
<br/>　　“现在无事，便能确定时间了。”
<br/>　　前些天，他一拖再拖，自然是因为百院之争的事。
<br/>　　他本打算百院之争后，若无其它事宜，再抽空去总殿参加考核的。
<br/>　　但现今他已不必参加百院之争。
<br/>　　不过，他还有一个疑惑。
<br/>　　“参加总殿考核的，应该不止我一个吧？”萧逸疑惑问道。
<br/>　　他记得当初在四方域时那几位副殿主所说，一般各主殿内出现的武道天骄，若得总殿认可，都会前往总殿参加考核。
<br/>　　这是总殿对殿内年轻一辈武者的厚待。
<br/>　　“是。”老者点点头，回答道，“算上你，此次传承考核，共十人。”
<br/>　　“这十人，来自于中域各地，但他们任何一人，都皆是近几年风刹殿最出色的年轻一辈。”
<br/>　　“共十人。”萧逸点了点头，问道，“那为何就急着传我？”
<br/>　　“其余九人都到达总殿了？”
<br/>　　“还没。”老者摇了摇头，“你一日未给出准确答复，何时前往总殿，传承考核的时间便一日定不下来何时开启。”
<br/>　　“自然的，其余九人也不会那么闲前去总殿干等。”
<br/>　　“至于先急传你。”老者脸色怪异地盯着萧逸。
<br/>　　“那是因为十人中，唯你一人最是麻烦。”
<br/>　　“有些时候，殿内甚至追踪不到你的踪迹，也压根找不到人。”
<br/>　　“你那般销声匿迹的本事，连殿内的风使都毫无办法，更别说通知你了。”
<br/>　　老者顿了顿，继续道，“你确定下时间了，其余九人会赶往总殿的。”
<br/>　　“我稍后便发消息回总殿，等你达到总殿，传承考核也该开始了。”
<br/>　　“原来如此。”萧逸微微尴尬。
<br/>　　“好了，没什么事便离去吧。”老者摆摆手，道。
<br/>　　“明日一早便出发。”
<br/>　　“好，便不打扰殿主处理事务了。”萧逸点点头，转身离去。
<br/>　　萧逸并未去主殿给他安排的房间，而是像以往那般，在闭关室内度过了一晚。
<br/>　　……
<br/>　　翌日清晨，天蒙蒙亮，萧逸便离开了闭关室。
<br/>　　与殿主告辞一番，萧逸正式出发。
<br/>　　前往总殿，路途不近，少说也得数月时间。
<br/>　　这还是他现今实力下，全速赶路的时间。
<br/>　　前往总殿的路程，比四方域还要遥远得多。
<br/>　　……
<br/>　　萧逸一路御空飞行，不多时，彻底出了天藏地域。
<br/>　　远方天际，一眼看不到尽头。
<br/>　　不得不说，中域实在太大了，地域众多。
<br/>　　即使现今是一路飞行，一路横跨，可也不知要横跨多少个地域才能到达总殿。
<br/>　　按萧逸的估计是数月时间。
<br/>　　但，对他而言，数月已经是不短的时间了。
<br/>　　毕竟，他最缺的，便是世间。
<br/>　　萧逸双眼眯了眯，一缕剑意从眼眸中迸发。
<br/>　　指尖一凝，一道剑气破空而出。
<br/>　　嗖…
<br/>　　萧逸的身影，一瞬间跃前数千米。
<br/>　　再看真切些，萧逸的速度，忽然比之前快了不少。
<br/>　　一缕缕剑气，此刻正环绕萧逸周身。
<br/>　　剑气破风，御空而行。
<br/>　　这便是他此刻速度增快的原因。
<br/>　　按这般速度下去，应该两月左右就能达到总殿了。
<br/>　　萧逸暗暗想着，也暗暗点了点头。
<br/>　　在他看来，前往总殿参加这份传承考核，未尝不是一次历练。
<br/>　　……
<br/>　　十日后，萧逸已横跨多个地域。
<br/>　　一路上，倒也见识了不少地域的精彩。
<br/>　　一个地域，本身就极大，动辄过百万里范围。
<br/>　　各地域间，自有其属于本身的独特之处。
<br/>　　如运河地域，有着踏云兽嬉戏其中的云海险地。
<br/>　　如四方域，有着闻名四方，内含各种险地的四方险地。
<br/>　　等等。
<br/>　　如果是正常历练，独自修行的话，闯荡各个地域，倒也算一份不错的经历。
<br/>　　不过，萧逸并无这些闲暇时间罢了。
<br/>　　或许，日后所有事都解决了，他才可能有闲暇。
<br/>　　萧逸这般想着，蓦地一笑，又摇了摇头。
<br/>　　忽然，萧逸嘴角笑意，陡然凝固。
<br/>　　眼眸，亦霎时一冷。
<br/>　　“谁？”萧逸冷喝一声，一道剑气凌空打出。
<br/>　　下方，一道道身影破空而起。
<br/>　　“萧逸。”为首的一个老者，冷眼盯着萧逸。
<br/>　　“我们可是在此等候多时了。”
<br/>　　“你们是？”萧逸皱了皱眉。
<br/>　　忽然，一道凛冽掌风，陡然轰来。
<br/>　　萧逸指尖一道剑气打出，轰散了掌风。
<br/>　　“北隐元劲？你们是北隐宗的人？”萧逸脸色一冷。
