        落以柔未注意到傅夜沉正在签字，而是自顾自地从挂在臂弯上的袋子里，拿出了一个粉色的四层餐盒，轻轻地放在了傅夜沉的办公桌上。
<br/>
<br/>        “这是我昨晚辛辛苦苦自制的牛肉干、樱桃干、雪花酥、肉松小贝，噢，对了，这里还有一瓶杨枝甘露！”落以柔一边说着，一边将餐盒每层挪开，然后又从袋子里拿出了一个玻璃瓶，立在了傅夜沉的办公桌上。
<br/>
<br/>        而这所谓的“杨枝甘露”，其实就是“百香果、芒果、西柚、椰汁、牛奶、西米露”。
<br/>
<br/>        这些食材，全是昨天他陪她在超市里买的，她自然要做一些好吃好喝地回馈给他。
<br/>
<br/>        除了牛肉干，其余全是甜食……
<br/>
<br/>        傅夜沉微微皱了下眉头。
<br/>
<br/>        为了做这些小零食，落以柔熬到昨晚两点才睡。
<br/>
<br/>        “大哥，你一定要吃噢！我先下去工作了，就不打扰你工作啦！”落以柔微笑着接着说，然后转身离开。
<br/>
<br/>        其实，她也不知道自己为什么要对这个男人这么好。
<br/>
<br/>        与其说对他好，倒不如说是对他上了心。
<br/>
<br/>        而落以柔做的这些小零食，傅夜沉一口未动，全被过来交接资料的连华生一一纳入了腹中。
<br/>
<br/>        连华生现在有一家自己的制药厂，因为是傅夜沉投资的，但傅夜沉分了一半的股份给连华生。
<br/>
<br/>        所以，连华生和他自己带的医学团队，就无需他傅夜沉再额外支付工资。
<br/>
<br/>        只不过，制药厂的实权，还是握在傅夜沉的手中，所有的交易合同都必须他傅夜沉过目签字了才生效。
<br/>
<br/>        既然吃了人家的东西，那么就得替人家办事。
<br/>
<br/>        连华生打算帮那个落以柔一把，教她如何去俘获一个高冷又失恋了的男人的心。
<br/>
<br/>        下班后，如果不是连华生自己主动找上落以柔家的门去说事，落以柔还真不知道，她声声叫唤的那位“大哥”竟然就是傅氏集团的大总裁傅夜沉。
<br/>
<br/>        当然，连华生也不知道她落以柔不知道这事。
<br/>
<br/>        落以柔泡了茶，又拿来了一些自己做的健康小零食，热情地招待了连华生。
<br/>
<br/>        她记得连华生，和“大哥”关系很好，看起来像是铁哥们。
<br/>
<br/>        连华生坐在落以柔的小沙发上，翘着二郎腿，一对贼溜的黑眸，将这里的四周打量了个遍。
<br/>
<br/>        这一进门就是卧室客厅厨房连体的出租屋，看起来也太寒碜了点。
<br/>
<br/>        不过，这小屋里倒是明窗净几，看起来舒适而温馨。
<br/>
<br/>        “傅氏集团旗下有员工宿舍，是滨江城里最好的公寓，我建议你，明天就去申请一下，提升一下你的居住档次。”连华生一手搭在沙发扶手上，一手食指敲击着膝盖，若有所思地说。
<br/>
<br/>        落以柔不明白连华生的来意，但是对于他的好意，她还是微笑着点头接受了。
<br/>
<br/>        “还有，我得带你去换个发型，买一些衣服。身为傅氏集团的一名员工，总是一副学生打扮可不行！”连华生打了个响指，指向落以柔，问道，“会开车吗？”
<br/>
<br/>        落以柔微微点了点头，当初她在读大学时期就去考了驾照，纯粹是为了课余时间有空兼职代驾赚点外快。
<br/>
<br/>        “很好！”连华生勾唇一笑。
<br/>
<br/>        傍晚时分，澳洲悉尼。
<br/>
<br/>        童筱颖代傅昼景签收了一个大件的跨国次日达快递。
<br/>
<br/>        因为上面写的寄件人是爷爷傅炎廷的名字，所以，童筱颖便直接将这个大件快递给拆了。
<br/>
<br/>        箱子里，有用气泡膜包得扎扎实实的奶粉、幼儿衣物、幼儿玩具、幼儿图书等等。
<br/>
<br/>        很明显，这些东西是送给季小景的。
<br/>
<br/>        童筱颖将这些东西一一整理出来后，心里突然间很不是滋味。
<br/>
<br/>        就算季小景是小卫的私生子，那也是流着傅家血脉的孩子，爷爷傅炎廷自然不会不管他的这个小曾孙。
<br/>
<br/>        只是，爷爷为什么突然快递这么多东西过来？
<br/>
<br/>        童筱颖心里莫名其妙地胡思乱想起来。
<br/>
<br/>        直到她要给季小景洗澡，到季思妍给季小景留下的行李箱前，打开行李箱拿衣服的时候，里面的那台数码相机，给了她一个确切的答案。
<br/>
<br/>        童筱颖鬼使神差地按了数码相机的开关，直接进入主界面的库存里，点开了第一个视频录像。
<br/>
<br/>        而这录像的显示日期，是季思妍来找她的第三天，正是傅昼景临时回来想跟她生孩子的那天。
<br/>
<br/>        视频里，傅昼景抱着季小景，面带微笑地坐在了旋转木马上。
<br/>
<br/>        季思妍的人虽未出现在视频里，但是她的声音，却源源不断地从视频里传来：“小景，和爸爸靠拢点，看着妈妈这边，笑一个！”
