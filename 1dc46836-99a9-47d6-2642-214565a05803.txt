豪门最看重的就是自己的子嗣和血脉，帝都四大豪门除了厉家，其他都有儿子来继承，而夜滢只生了一个女儿，这一直是她的一大憾事。
<br/>
<br/>        都说母凭子贵，生不出儿子的豪门太太都坐不稳这个位置的，夜滢也想给厉君墨再生个儿子，但是她连厉君墨的面都见不到了，跟谁生儿子去？
<br/>
<br/>        夜滢很要强，跟她的父亲夜老一样很看中自己的面子，她不想让别人看穿她现在的窘境，所以迅速笑道，“再生一个儿子一直在我和君墨的计划里，谢谢大家的关心。”
<br/>
<br/>        这时厉嫣然拉着陆寒霆走过来了，“妈咪，你们在这里聊些什么呢，聊得这么开心？”
<br/>
<br/>        “嫣然小姐，我们正在聊你很快就要有一个弟弟了。”有人打趣道。
<br/>
<br/>        “真的吗？”厉嫣然双眼一亮，“妈咪，那你和爹地快点给我生一个弟弟吧，我就喜欢弟弟！”
<br/>
<br/>        厉嫣然跟别的孩子不同，别的孩子可能不喜欢弟弟，怕弟弟分去自己的爱，但是厉嫣然很想要一个弟弟。
<br/>
<br/>        她的爹地妈咪早就分居了，她很想爹地妈咪能和好，这些年奶奶和爹地对她都不热情，妈咪虽然顶着厉夫人的名号，但是妈咪并没有管家，没有任何的实权，厉嫣然很有危机感，她想爹地妈咪和好然后生下一个弟弟，这样就可以巩固她的位置和利益了。
<br/>
<br/>        夜滢疼爱的点了一下厉嫣然的额头，“嫣然，你还是想想你自己吧，你什么时候跟寒霆完婚，早点给寒霆生个儿子，给陆家添一个长子嫡孙才是正事。”
<br/>
<br/>        厉嫣然漂亮的小脸迅速一红，就连陆寒霆身体有毛病这种事都抛诸脑后了，她害羞的撒娇道，“我知道了妈咪~”
<br/>
<br/>        现场真是其乐融融，夜滢和厉嫣然母女也是赚足了眼球，一直是全场瞩目的焦点。
<br/>
<br/>        这时一直没出声的厉老夫人指了指厉嫣然身上那件深蓝色的星空吊带长裙诧异道，“嫣然，这件裙子你在哪里拿到的？”
<br/>
<br/>        “这件裙子啊，”厉嫣然提着裙摆像小公主一样在现场转了两圈，笑靥如花道，“奶奶，谢谢你给我准备的这个，我真的好喜欢这件裙子。”
<br/>
<br/>        厉老夫人，“…”
<br/>
<br/>        这时厉老夫人身边的李嬷嬷出声道，“嫣然小姐，你搞错了，这件裙子不是为你准备的，而且你身上穿的这件裙子是…试衣样品。”
<br/>
<br/>        试衣样品？
<br/>
<br/>        大家都知道量身定制一件长裙是需要很多繁琐步骤的，其中就会反复的试验样品，这些试衣样品都是有瑕疵的，等瑕疵逐一改善了，才能制作出最完美的成品。
<br/>
<br/>        现在厉嫣然身上这件星空长裙不过就是一件试衣样品！
<br/>
<br/>        厉嫣然直接僵在了原地，她不可置信的看着厉老夫人，“什么，奶奶，你是不是搞错了，我穿的怎么会是试衣样品呢？而且…这件星空长裙不是为了我准备的，那是为了谁准备的啊？”
<br/>
<br/>        夜滢的脸色也是猝然惊变，她也看向了厉老夫人，“妈，这…这究竟是怎么回事啊？”
