　　晚上，
<br/>　　生物学院新生联谊会，在天海大学南门的君悦酒店举行。
<br/>　　事实上，不光是生物学院，其他各大学院，也基本上都这里举办了新生联谊会。
<br/>　　叶尘原本不打算参加的，被李虎三人的一再劝说下，这才勉强答应前往。
<br/>　　等四人来到酒店的时候，其余众人几乎已经到齐，连大二、大三的学生，也来了不少。
<br/>　　看到他们四人出现，众人顿时开始指指点点，议论纷纷起来。
<br/>　　“快看！那个就是叶尘，果然很帅呢！”
<br/>　　“难怪连咱们学校四大女神之一的萧若曦，都被他给迷住了！”
<br/>　　“这下总算是见到了！”
<br/>　　“哼！一个大一新生，不知道天高地厚，那萧若曦可不是什么人都能染指的！”
<br/>　　……
<br/>　　叶尘神念一扫，已经将众人的议论声，全部听在了耳中。
<br/>　　有人羡慕，有人嫉妒，有人好奇，不过他也懒得理会。
<br/>　　因为四人来的比较晚，所以只能在角落里，找了几个座位坐下。
<br/>　　不过，因为叶尘如今已经成为了天海大学的名人，虽然坐在角落里，却依旧备受瞩目，不少人时不时朝他这边望来。
<br/>　　对此，叶尘淡然如故，自顾自的端起一杯红酒，翘起二郎腿，慢悠悠的品了起来。
<br/>　　又过了一会，
<br/>　　忽然，整个宴厅再次躁动起来，
<br/>　　“快看快看！是杨之凡和陶婉清来了！”
<br/>　　“真是一对金童玉女啊！”
<br/>　　“哇！杨之凡好帅啊！就像是从童话故事里走出来的王子！”
<br/>　　“好像比叶尘还要帅上一些呢！”
<br/>　　……
<br/>　　听到众人的议论声，叶尘也不由得抬头望去，
<br/>　　只见门口处，
<br/>　　一名身穿黑色晚礼服的英俊少年，和一名穿着白色礼裙的漂亮女孩，两人谈笑风生之间，并排走入了宴厅之中。
<br/>　　“杨之凡，陶婉清……”
<br/>　　看到这两个人，叶尘也不由得微微有些晃神，前世的一些记忆，不由自主的就涌了上来。
<br/>　　这杨之凡，不但人长得帅，同时又聪明绝顶，是一个智商爆表的天才，而且情商很高，家庭条件也很好，可以说是一个完美人物，说是天之骄子，也一点不为过！
<br/>　　叶尘记得在前世，杨之凡是令所有生物学院男生，都为之嫉妒和羡慕的存在，几乎所有女生都把他当做梦中情人。
<br/>　　就连叶尘当时，也对他有几分嫉妒。
<br/>　　当然，
<br/>　　前世在叶尘眼中，杨之凡是高不可攀的存在，但对于今生，今时今日的叶尘来说，就不值一提了。
<br/>　　叶尘只淡淡瞥了一眼，便不再理会，继续低头品酒。
<br/>　　一旁的李虎，忍不住嘀咕道：
<br/>　　“这家伙还真是下血本啊！一个小小的新生联谊会而已，居然整了一套阿玛尼！手表也是江诗丹顿的，这一身行头下来，少说也值一百万！”
<br/>　　一旁的陈东和唐威，听的直咋舌。
<br/>　　……
<br/>　　“之凡学弟，婉清学妹，快到这边来坐！”
<br/>　　杨之凡和陶婉清，在众人的瞩目下，才刚刚踏入宴会厅。
<br/>　　宴厅正中央的主桌上，一名相貌颇具威严的青年男子，忽然站了起来，冲着杨之凡和陶婉清热情的高喊了一声。
<br/>　　看到这一幕，众人顿时再次一阵羡叹。
<br/>　　李虎则一脸愤愤不平，生怕叶尘不知道，低声道：
<br/>　　“这人是咱们学院的院学生会主席，叫张连鹏！听说他就是靠溜须拍马，才爬到今天这个位置！”
<br/>　　“听说这家伙一向趋炎附势，对他有用的，就拼命的讨好，对他没用的，他鸟都不鸟！”
<br/>　　“如今肯定是看中了杨之凡的家世背景，这才刻意讨好，真是恶心啊！”
<br/>　　叶尘见李虎一副愤慨的样子，不由得为之莞尔，忍不住调侃道：
<br/>　　“老四，没想到这才开学第一天，你倒是知道的挺多嘛！”
<br/>　　李虎顿时大为得意，
<br/>　　“那当然！我老爸常常告诫我，无论是战场还是商场，信息永远都是第一位的！”
<br/>　　“再说现在都互联网时代了，想查个什么信息，还不是小菜一碟？”
<br/>　　一旁的唐威忍不住笑道：
<br/>　　“说你胖，你还喘起来了！那你倒是说说，这杨之凡到底是什么背景啊？”
<br/>　　李虎嘿嘿一笑，
<br/>　　“这个我还真知道！”
<br/>　　说到这里，李虎喝了一口眼前的啤酒，压低声音道：
<br/>　　“据说，这杨之凡的叔叔，是天海市的副市长！而杨之凡的老子，则是我们东江省，有名的富商，身家好几百亿！他们杨家在我们东江省，那可是排名十的大家族！”
<br/>　　唐威和陈东听到这话，顿时纷纷露出一抹深深的忌惮之色。
<br/>　　横跨政商两界，而且叔叔又是天海市的高官，别说张连鹏了，怕是院长都要给他几分面子！
<br/>　　如果李虎所言不假，那这个杨之凡，的确称得上是天之骄子，和此人相比，他们这些普通人，都太过遥远了！
<br/>　　杨之凡落座之后，不少人顿时便开始拿杨之凡和叶尘对比起来，
<br/>　　“果然还是杨之凡更帅一些！”
<br/>　　“而且连咱们学生会的张主席，都主动给他让座，背景肯定不简单！”
<br/>　　“相比之下，那叶尘虽然容貌不必杨之凡差，可是无论仪态，还是气质，两人都不在一个档次啊！”
<br/>　　……
<br/>　　听到众人的议论声，叶尘倒没觉得有什么，倒是把一旁的李虎气得不行，
<br/>　　“不就是会装逼吗？有什么好得意的！”
<br/>　　“尘哥，你也别气馁，以你的条件，要是好好打扮一下，肯定能完爆这小子！”
<br/>　　在李虎看来，叶尘身上的衣服，虽然也都是大牌，但都是平平无奇的休闲服，和杨之凡的一身名牌正装相比，终究还是差了不少档次。
<br/>　　叶尘淡淡一笑，没有多说什么。
<br/>　　以他如今的境界，杨之凡在他眼中，连蝼蚁都不算上，他又岂会将这些世俗的眼光放在心上？
<br/>　　那杨之凡似乎也听到了众人的议论声，也扭头朝叶尘的方向望了过来，不过只是看了一眼，便收回了目光，眼底闪过一抹轻视之意。
<br/>　　一旁的陶婉清，也悄悄打量了叶尘片刻，同样摇了摇头，
<br/>　　在她看来，这位才刚进校门，就惹出这么大动静的同学，虽然皮囊确实不错，但不过只是一个哗众取宠之辈，和杨之凡根本不在一个层次。
<br/>　　一念及此，陶婉清望向一旁的杨之凡，目光也变得越发热切。
<br/>(本章完)
