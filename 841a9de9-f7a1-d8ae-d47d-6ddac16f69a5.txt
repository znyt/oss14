　　四人听了这话，也是明白了他的立场，那和尚说道：“祖师，人间法术界，也不是你一个人的，何苦因这无意义的坚持，得罪我法术公会？”
<br/>　　李浩然面不改色，带着一种他人无法理解的骄傲，说道：“当年，我本是乡下一耕牛，万幸遇到我主，点化于我。我主为道教之祖，西行化胡，创立佛宗，人间法术界，莫不尊其为大道之主。我为其弟子，何其荣幸……”
<br/>　　说到这，神情蓦然变冷，语速也陡然加快，声色俱厉地喝道：“你们轩辕氏虽为上古神祇后人，但若想图谋人间，奴役我法术界，我自当第一个杀上不周山，灭了昊天和盘古老僧！更不要说什么星月奴之辈！”
<br/>　　一番慷慨的陈词，令四人震惊无比。
<br/>　　一道士苦笑道：“祖师若是如此，我等也没甚么好说的，只是……祖师一己之力，就想力挽狂澜，也太小看我轩辕氏了。”
<br/>　　“或许，只是，你却是看不上了。”李浩然淡淡说道。
<br/>　　“什么！”四人震惊。和尚随后冷笑道，“以一敌四，祖师也太小看我等了。”
<br/>　　结果话音刚落，突然觉得身下有异样，低头一看，顿时大吃一惊，一股黑气从地面上升起来，缠住了他们的双脚。
<br/>　　四人想要拔脚的瞬间，黑气蔓延，将四人困在中间，李浩然人影一动，冲了进去，一只手中神光乍起，形成了一把光剑，劈头斩去。
<br/>　　四人作法抵挡，苦苦支撑。
<br/>　　一时半刻之后，黑气散去，四人皆被枭首，连魂魄也没能逃出去，被李浩然直接斩碎，化作精魄，破开虚空，自行飞往不周山去了……
<br/>　　李浩然面色冷峻，在山冈上站了片刻，低头沉吟道：“却是错了。这毕竟不是我的战场……”
<br/>　　“师兄没错。”一个俏生生的声音，从身后不远处响起来。
<br/>　　李浩然转过头，看到了从丛林中走出来的碧清。
<br/>　　“几百年不见，师兄，你一向可好。”
<br/>　　李浩然低头，说道：“就连这一面，我们本来也不该相见的……”
<br/>　　碧清道：“师兄，你之前的一切，我全看到了，你是从后世轮回中来的，我不知道你为什么会来，但你我的兄妹之缘，却并非一世而已。你能找我，我很高兴。”
<br/>　　李浩然隐匿在轮回之中，为了不让碧清干扰他每一世的轮回，并没有给她留下什么线索，当初的约定是，他拿了碧清的一缕头发，自己去轮回，碧清去找地方修炼，两不相见，等李浩然证道之后，再去找她，共同进退。
<br/>　　就在不久之前，她神念之中，收到了李浩然召唤自己的信息……
<br/>　　“师兄，我不知道你为什么来到当世。我很想你，你召唤我，是否是遇到了什么棘手的事情？”
<br/>　　李浩然朝她走过去，一边说道：“其实，并不是我要来这里，我是无意中被徐福送来的……”
<br/>　　李浩然有点略尴尬，想起自己来到当世的经历，不光是愤慨，还有点丢脸。
<br/>　　“我本不想在当世留下太多痕迹，但恰逢乱世，轩辕氏欲吞并法术界，我不得不出手……我需要帮手。”
<br/>　　碧清开心地笑起来，这个亦正亦邪、睥睨一切生灵的邪神，只有在李浩然面前，才会表现出自己天真烂漫的一面。
<br/>　　“师兄能想到找我，这就对了，我才不管什么轩辕氏，一起杀过去便是了。”
<br/>　　“没有这么简单，凭我们两个的力量，是做不到的。此事需要从长计议……”
<br/>　　“哦，我听师兄的。”碧清很自然地说道，随后想到一件事，皱起眉头，说道：“说来也是巧，其实我也不是因为师兄你的召唤而醒，我在一座古墓里修行，不久前有一群法师闯进去……有一个家伙极其讨厌，不过他说他认识你，而且说我跟一个叫什么的……忘了叫什么，说是跟我长得一样，嗯，就是师兄之前从我身上拿走的神念吧，跟你一起进入轮回证道的。”
<br/>　　李浩然心中咯噔了一下，忙问道：“是什么人，长什么样子！”
<br/>　　“这……二十出头，模样……还行吧，是个很让人讨厌的家伙！”
<br/>　　“很让人讨厌？”
<br/>　　“就是……很狡猾，有点下流。说话也怪里怪气，不过……我是先天灵体，而且身手敏捷，不得不说，很聪明的一个人。师兄你认识他吧？”
<br/>　　果然是……李浩然听了描述，立刻就想到了他的名字，缓缓点头，道：“认识，当然认识，这样的人，整个法术界就一个。茅山掌教叶少阳！”
<br/>　　“对对，是叫叶少阳。”碧清怔了一下，“他是茅山掌教，不会吧？”
<br/>　　“百年之后的。我能被困在这里，全是拜他所赐！他在什么地方，快带我去找他！”
<br/>　　碧清看他急切，两人立刻启程，这里离那古墓有几百里，两人当然不可能一直飞过去，破开虚空裂缝，进入鬼域，然后碧清带路，寻找古墓附近的虚空裂缝，钻进去……
<br/>　　叶少阳当然不知道李浩然在搜寻自己，他甚至都不知道李浩然跟自己一样来了这个时代，在客栈呆了一晚上之后，道渊真人回来了。
<br/>　　他刚从杭州城外的桃花山回来。
<br/>　　他本来是帮叶少阳打听桃花山上那个神秘来客的身份，结果那人的身份没打听到，却带过来一个惊人的消息：
<br/>　　桃花山上那个家伙，打败了法术公会派去的使者，并且废了他的肉身。法术公会震怒不已，目前已经召回了各大弟子，让他们去串联各大门派，征讨桃花山……
<br/>　　“不会吧，法术公会这么重视这个人？他再厉害，也就是一个人而已。桃花山也就是个中等门派，我就不信能跟法术公会抗衡？”毛小方表示出了强烈的质疑。
<br/>　　道渊真人道：“据说被那人打败的，是星月奴非常重视的一个弟子。那神秘人物废了他，表达了对抗法术公会的态度。法术公会本来是要一统法术界，布局都差不多了，这节骨眼上突然跳出来一个反对者，你们知道这意味什么。”
