<br/>    一线天峡谷。
<br/>    
<br/>        一线天峡谷是一条深渊，至于名字的来源，有两个原因，一个便是这一条峡谷极为狭长，故而名为一线天。
<br/>    
<br/>        而另外一个原因便是，这一条峡谷便是仙魔的分界线。
<br/>    
<br/>        而现在一线天之上，已经是诸王战成了一片，魔王与人王在一线天之上浴血搏杀，王级战争全面解禁，带来的便是一场血祸。
<br/>    
<br/>        无数诸王在天空之中喋血，鲜血挥洒，如同一场血雨落下，一具具诸王的尸首，从天空之中落下。
<br/>    
<br/>        大雨滂沱，和鲜血汇聚在一处，整个峡谷之中流动的河流化作一条血河，蔓延流向远方。
<br/>    
<br/>        整片大陆，所有的凡人惶惶不可终日，那些在他们眼中如同仙人的存在，整日在苍穹之上搏杀，毁灭一切，本是与世无争的地方，都可以看见诸王的搏杀。
<br/>    
<br/>        诸王的搏杀，对于凡人来说无疑是一场灾难，他们根本没有任何反抗的能力。
<br/>    
<br/>        不过还好的是，诸王们并没有直接对凡人出手，修行者对凡人出手，造成杀孽是大忌。
<br/>    
<br/>        大雨之中，一个魔王快速来到一具血棺之前，“报，哑魔大人，我们已经尽全力了，但是还是无法夺下一线天峡谷！”
<br/>    
<br/>        棺材缓缓推开，鲜血从棺材之中溢出，李素衣从血棺之中走出，暗红色的眸子打量着身前的魔王，看得魔王两腿打颤，李素衣并没有说话，只是看向远处的一线天峡谷。
<br/>    
<br/>        李素衣淡淡道:“从今以后，这个地方，不再叫做一线天，这个地方，应该叫做十字岭。”
<br/>    
<br/>        李素衣将血棺背在背后，伸手入血棺，拔出了血海魔刀，朝着一线天峡谷便是一刀劈去，刀光弥漫数十丈，直接在一线天峡谷之上劈开一道新的峡谷，将一线天峡谷变成了十字形。
<br/>    
<br/>        众多人王面露惊色，“糟了，是哑魔到了！”
<br/>    
<br/>        “我方悟道榜上的诸王翘楚呢，他们到了没有！”
<br/>    
<br/>        “若是他们没到，我们也只能撤了！”
<br/>    
<br/>        然而他们还没有反应过来，一抹刀光横掠数十丈，血海弥漫，将周围的一切都染红，将十多位人王直接轰得重伤。
<br/>    
<br/>        背着血棺的李素衣站在悬崖之上，看向对面，一言不发。
<br/>    
<br/>        潇潇大雨之下，魔气纵横四方，震慑着周围的一切，他，李素衣，是悟道榜第一，是诸王之首，所过之处，诸王退避！
<br/>    
<br/>        雨水化作白雪，雪自悟从悬崖的对面走出，青眸玄牛踏步，朝着李素衣缓缓走去，刘芒提着腾龙枪看向李素衣。
<br/>    
<br/>        李素衣并没有后退半步，即便是对面有着两位悟道榜上排名前十的存在，他也懒得后退，因为他，是悟道榜第一。
<br/>    
<br/>        李素衣横刀，血海魔刀血气蔓延，雪自悟大喊道:“李素衣，你本是名门正派，为何要叛变，远走魔宗？”
<br/>    
<br/>        李素衣的回答，是一抹血红色的刀光，刀光蔓延数十丈，将天地都染成血红色，刘芒提枪迎上，挥枪便是一条长龙呼啸而出，李素衣眉角轻佻，根本没有怎么在意，一跃而起，手中血海魔刀朝着长龙一刀劈去，直接将长龙劈成两半，李素衣乃是陈明门下的弟子，此时可不会听一番诸王的赞叹，直接飞身而起，便是冲着刘芒攻去。
<br/>    
<br/>        长刀和长枪碰撞在一处，刹那间已经是交手十多招，李素衣穆然一刀劈去，直接将刘芒轰飞，落在地面上，撞出一个大坑来，刘芒捂着胸口，他的内脏在这次的交手之中已经被震伤，口中也是一口鲜血喷出！
<br/>    
<br/>        雪自悟看得两眼露出焦急之色，周围的人王们也是心如死灰，“这才刚刚交手，刘芒就已经受伤了，哑魔，竟然是强大到了这种程度！”
<br/>    
<br/>        雪自悟也不再迟疑，直接朝着李素衣飞去，“寒冰掌！”
<br/>    
<br/>        雪自悟一掌挥出，天空之上，落下一道足足三十丈长宽的寒冰巨掌，李素衣挥手一刀便直接将寒冰掌轰得粉碎！
<br/>    
<br/>        而后直接朝着雪自悟冲去，雪自悟的手中出现一柄雪白长剑，朝着李素衣刺去，李素衣收刀入血棺，再次拔刀，拔刀术！
<br/>    
<br/>        轰！
<br/>    
<br/>        血海魔刀直接将雪自悟轰退十步，李素衣继续欺身而上，刀刀拔刀术！
<br/>    
<br/>        周围的人王和魔王们看得惊诧不已，“他的刀法境界，竟然达到了如此的境地！”
<br/>    
<br/>        “这刀刀拔刀术，可是只有大君才能施展出来的！”
<br/>    
<br/>        雪自悟连连挥剑，却是不敌李素衣的快刀，不过交手二十多招，就已经是被李素衣一刀劈飞，在地面上砸出一个大坑来！
<br/>    
<br/>        魔王们看得两眼发亮，“哑魔神功盖世，诸王之首！”
<br/>    
<br/>        “天下诸王，无一人是哑魔对手！”
<br/>    
<br/>        “君下无敌！”
<br/>    
<br/>        “大君不出，谁与争锋？”
<br/>    
<br/>        李素衣朝着重伤的雪自悟走去，漂泊的大雨之中，人王们已经丧失了信心，李素衣的强大，让他们只能感受到畏惧。
<br/>    
<br/>        大雨之中，李素衣想要挥刀斩了人王。
<br/>    
<br/>        大雨之中，有人掀开了斗笠，斗笠落在雨水之中，同样露出了他的脸，那是一张李素衣忘不掉的脸。
<br/>    
<br/>        凌仙抬头，看向李素衣，“三师弟，只要有我在，你就一个正道都杀不掉！”
<br/>    
<br/>        李素衣看向凌仙，挥刀直指凌仙，二师兄啊，我前世，可是一直听着你的传说的啊！你是仙门正道的天之骄子，而我只是魔道余孽，你万千荣宠，而我，孑然一身，父母双亡，妹妹也流落不知去处，被药王谷炼制成了丹药！
<br/>    
<br/>        没有想到，这一世，我居然有机会和你这个天之骄子一战，还真是:
<br/>    
<br/>        与君一战，三生有幸啊！
<br/>    
<br/>        雪自悟扫了一眼凌仙的修为，“凌仙，你只是道种境，你不是李素衣的对手的！”
<br/>    
<br/>        李素衣感受了一番自己的法力，还剩下两成，他的强大，也是有代价的，刀刀拔刀术，让他的法力消耗的很快。
<br/>    
<br/>        刘芒也是道:“李素衣已经入魔了，你即便是他的师兄，他也会对你下杀手的！”
<br/>    
<br/>        凌仙径直朝着李素衣走去，没有半步后退。
<br/>    
<br/>        李素衣纵身一跃，直接朝着凌仙冲去，血海魔刀再次挥动，斑驳血海迷茫四周。
<br/>    
<br/>        周围的人王被震得后退数十步，只能远远地看着燕山的这对师兄弟互相厮杀。
<br/>    
<br/>        凌仙抬手，双袖鼓动，方圆十里灵气爆涌，汇聚成漩涡，归于凌仙的双手，他左手穷奇飞舞，右手恶蛟出渊！
<br/>    
<br/>        搏仙术！
<br/>    
<br/>        这一击对拼，血海魔刀对搏仙术！
<br/>    
<br/>        轰！
<br/>    
<br/>        恶蛟与穷奇飞舞，十丈身上的两道飞影与血海魔刀的刀光撞在一处，发出雷鸣般的炸响，无数的山石崩塌，将整个大地轰得一片狼藉。
<br/>    
<br/>        李素衣看向凌仙，这是搏仙术！
<br/>    
<br/>        当年，你可是花了足足十年才学会的啊，这才过了多久，你居然就已经学会了？
<br/>    
<br/>        哦，对了，师父。
<br/>    
<br/>        有师父在，你才能学会的这么快。
<br/>    
<br/>        那就看看，到底是师父教给我的血海魔刀厉害，还是师父教给你的搏仙术厉害吧！
<br/>    
<br/>        周围的人王还是魔王，都在这一击对拼之下惊叹不已，“哑魔大人的血海魔刀，可是燕山之主教的！”
<br/>    
<br/>        “凌仙刚刚的那一招，也是燕山之主教的吗？”
<br/>    
<br/>        “燕山之主，到底是一位怎么样诡异的存在啊！”
<br/>    
<br/>        “一人门下，居然出了两位风华绝代的弟子，一人是魔宗君下无敌，另外一位，只是道种境，竟然可以和法相境对战！”
<br/>    
<br/>        李素衣再次横刀，凌仙眼中露出一抹决绝之意，“师父给了你血海魔刀，同样也给了我神木印，就看看，谁才真正学会了师父的手段吧！”
<br/>    
<br/>        凌仙张开手，神木印出现在手中，先天大道之气飞舞，汇聚在凌仙的双袖之间！
<br/>    
<br/>        李素衣飞身而上，背后黑色的披风在狂风中猎猎作响，血海魔刀淡然挥下，凌仙手持神木印，再次打出穷奇与恶蛟。
<br/>    
<br/>        血海魔刀将穷奇与恶蛟直接劈飞，余威将凌仙轰得口吐鲜血。
<br/>    
<br/>        凌仙擦了擦嘴角的鲜血，再次朝着李素衣飞去，再次施展搏仙术，穷奇和恶蛟飞去。
<br/>    
<br/>        李素衣再次挥刀，将凌仙劈的咳血。
<br/>    
<br/>        李素衣一次次挥刀，凌仙一次次咳血。
<br/>    
<br/>        足足一刻钟之后，浑身鲜血的凌仙再次站了起来，看向李素衣，“无疆在我的手上将你带走，我就要将你带回来，无论如何，我都要把你带回来！”
<br/>    
<br/>        李素衣终于是收刀，他感受了一番自己的法力，只剩下一成了，“二师兄，只是道种境的你，根本不是我的对手，你这又是何苦呢？”
<br/>    
<br/>        凌仙听到李素衣的回答，叫道:“三师弟，回头吧！”
<br/>    
<br/>        李素衣哈哈大笑道:“我这一身的罪孽，如何回头，若能回头，何来无边苦海？”
<br/>    
<br/>        李素衣看向凌仙，“今日一战，我法力不济，已经快要用尽，你的修为不够，你我都不在巅峰，不够尽兴，我杀人的时候不会说话，既然我已经开口，我便不会杀这里一人，今日，我不杀你！”
<br/>    
<br/>        李素衣道完，直接转身走了，待得李素衣来到魔宗魔王之中，有一个魔王道:“哑魔大人，你刚刚明明可以直接斩了那凌仙，只是道种境，就有这般实力，可是一个大威胁，哑魔大人，你刚刚可是手软了，对于魔修来说，手软可是魔修大忌！”
<br/>    
<br/>        李素衣拔刀，一刀斩下这个魔王的头颅，然后看向其他魔王，“还有人有什么疑问吗？”
<br/>    
<br/>        众多魔王再不敢说半句，寂静无声。
<br/>    
<br/>
