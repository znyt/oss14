　　操控天地灵气的能力，与武者自身小世界的大小有关。
<br/>　　小世界越大，操控的范围就越大，强度也越强。
<br/>　　任何武者，刚突破洞玄境时，小世界的大小都是气泉的一倍。
<br/>　　往后的修炼，气泉是定数，永远不变。
<br/>　　而小世界，却会随着武者修为增高而变大。
<br/>　　换然之，随着武者修为提升，小世界越大，日后的能力也会越来越强。
<br/>　　萧逸刚刚突破，并没有急着离开。
<br/>　　而是继续回到山洞，刻苦修炼，稳固修为。
<br/>　　数天后，他才再次离开山洞。
<br/>　　此时，他的修为彻底稳固，体内原本干涸的气泉，已然填满一成。
<br/>　　证明，现在他的修为乃是洞玄一重。
<br/>　　待得气泉完全填满，他便是洞玄九重。
<br/>　　当然，他很清楚，以自己气泉的庞大度，要填满是一件相当有难度的事。
<br/>　　长宽高各三千三百三十三丈，这可不是加法，而是乘法。
<br/>　　对比那些气泉不过两百丈、三百丈的武者，大了足有百多倍。
<br/>　　可想而知，这个气泉大到何等恐怖的地步。
<br/>　　要想填满，绝非一朝一夕。
<br/>　　若是按正常修炼，很可能其他武者已经洞玄九重了，他还在苦苦为突破洞玄二重而忧心。
<br/>　　但，他并不为此而担心。
<br/>　　气泉里面的真气，太过庞大，是武者的身体远远无法同时承受的。
<br/>　　这也是为什么要在身体内开辟出小世界，让这些真气储存在气泉里。
<br/>　　正常来说，武者一次性，顶多能释放气泉内一半左右的真气。
<br/>　　只有拥有道体的人，才能打破这个限制。
<br/>　　道体，乃是最适合修炼的身体。
<br/>　　道体，对真气的亲和度，以及真气在体内游走得顺畅度，都远非普通武者能比。
<br/>　　像先天道体，能一次洗释放气泉七成的真气。
<br/>　　别小看这仅仅多出的两成。
<br/>　　这是先天道体武者越级战斗的资本。
<br/>　　当然，一次性释放这么多真气，必是武者使用杀招的时候。
<br/>　　比如，同为两百丈气泉，且填满的情况下。
<br/>　　普通武者，只能一瞬间调动一百丈真气来施展强大的地阶武技。
<br/>　　而拥有先天道体的武者，则能一瞬间调动一百四十丈的真气来施展。
<br/>　　两者的威力，显然是不同的。
<br/>　　而萧逸自己试过，他的承受能力，在强大的肉体，以及冰火道体的辅助下。
<br/>　　气泉内的真气，可做到一次性百分百释放。
<br/>　　如今他只是洞玄一重，气泉填满一成。
<br/>　　气泉内的真气，长宽各三千三百三十三丈，高度，则只来到三百三十三丈这个高度。
<br/>　　他一瞬间就能将这个数量的真气全部释放出来。
<br/>　　那些超级天才，气泉两百丈，就算达到洞玄九重，气泉全部填满；但在实力上，也远远比不上现在只有洞玄一重的他。
<br/>　　只要有足够强大的实力，修为提升方面，时间长一些也没有关系。
<br/>　　这是萧逸的打算以及自信。
<br/>　　再者，他还有修罗战体。
<br/>　　数万瓶妖兽精血，他还未服用。
<br/>　　回到洞中，他盘膝坐下，立刻服用。
<br/>　　他现在手上的妖兽精血，是16支猎妖队手上换来的。
<br/>　　至于以前的，早就吞服完了。
<br/>　　四级精血，3200瓶；三级精血，32000瓶。
<br/>　　事实上，三级妖兽精血，效果已经不大了。
<br/>　　不过，量变，可以引起质变。
<br/>　　当三万多瓶三级妖兽精血完全吞服，他的肉体力量，从洞玄七重，达到了洞玄九重。
<br/>　　剩下的3200瓶四级精血，他服用了1500瓶。
<br/>　　肉体力量，达到半步破玄。
<br/>　　剩下的1700瓶，他没有服用。
<br/>　　因为，修罗战体再继续修炼下去，便是突破第二层，来到第三层了。
<br/>　　层与层之间，是一个大境界。
<br/>　　需要的力量，再次成倍增加。
<br/>　　1700瓶，暂时还不够突破突破到第三层，便暂时不服用了。
<br/>　　根据当初狂血玄君留下的书籍记载。
<br/>　　第三层，对实力的提升，有一个质的飞跃。
<br/>　　也是因此，导致了突破所需要的力量极其庞大。
<br/>　　萧逸摇了摇头，心中已经有了日后修炼的大致打算。
<br/>　　嗖的一声，他离开了山洞，来到了高空中。
<br/>　　“凝。”萧逸大喝一声。
<br/>　　方圆万米内的灵气，排山倒海般向他涌来。
<br/>　　在冰鸾剑和控火兽两大武魂的吸收下，顷刻间化为真气，储存于小世界的气泉中。
<br/>　　直接大幅度吸收天地灵气，这是洞玄境武者加快修炼的手段之一。
<br/>　　不过，其他武者的武魂品阶不高，吸收速度有限。
<br/>　　就算是数百米天地灵气，往往也需要吸收很久。
<br/>　　所以这个手段，就算能加快修炼速度，但也没有太过大的效果。
<br/>　　可萧逸不同，以冰鸾剑的吸收速度，再庞大的天地灵气，都是一瞬间就能吸收完毕。
<br/>　　所以这个手段，倒是能在萧逸手上发挥出恐怖的效果。
<br/>　　“轰”
<br/>　　方圆万米之内的天地灵气被吸收殆尽，导致这个范围内，成了一片灵气真空地带。
<br/>　　万米之外的天地灵气，会自动补充过来。
<br/>　　当然，这个时间不慢，起码要数个时辰。
<br/>　　萧逸没有在原地等待，而是直接飞到更远的地方。
<br/>　　再次‘掠夺’天地灵气。
<br/>　　“轰”
<br/>　　又是天地灵气被吸收殆尽，出现了一个万米大小的灵气真空地带。
<br/>　　轰…轰…轰…，高空中的天地灵气，不断被吸收殆尽。
<br/>　　为了加快飞行速度，萧逸幻化出紫炎火翼。
<br/>　　虽说洞玄境已经能御空飞行。
<br/>　　但有紫炎火翼的增幅，飞行速度会更快，快了数倍。
<br/>　　不多时，萧逸已经横跨数城。
<br/>　　这数城的高空中，灵气都被吸收一空。
<br/>　　当然，这不是什么严重的事，不过几个时辰，天地灵气就会从其他地方补充过来。
<br/>　　虽说吸收了这么大范围的天地灵气。
<br/>　　不过，天地灵气很是稀薄。转化为真气后，也并非太多。
<br/>　　“嗯？”半晌，萧逸忽然皱起了眉头。
<br/>　　在他横跨数城的时候，明显可以感觉到，城内的武者数量，比以往少了许多。
<br/>　　“怎么回事？”萧逸面露疑惑之色。
<br/>　　半个时辰后，他飞到疾风猎妖队总部，并停了下来。
<br/>　　他本想找宁浩商议以后的交易。
<br/>　　没想到，眼前的景象，却让他大吃一惊。
<br/>　　偌大个疾风猎妖队，队员逾千人，此时竟如人去楼空，只有寥寥十数个守门之人。
<br/>　　萧逸降落，立马有几个猎妖师前来行礼。
<br/>　　“易霄大师，你怎么还在这里？”几个猎妖师面露疑惑。
<br/>　　“我不该在这里吗？”萧逸皱眉问道。
<br/>　　“额，不是，易霄大师息怒。”几个猎妖师见萧逸皱眉，连忙赔罪。
<br/>　　“怎么回事？其他人呢，怎么就只有你们几个？”萧逸问道。
<br/>　　“额，易霄大师不知吗？”几个猎妖师脸色更加疑惑了。
<br/>　　“我知道还需要问你们？”萧逸没好气地说道。
<br/>　　几人狐疑地看着萧逸，回答道，“三天前，兽潮不期而来，妖兽攻城，所有武者都去守城了。”
<br/>　　“兽潮？”萧逸眉头皱着。
<br/>　　东荒十八城，每数年都会遭遇妖兽攻城。
<br/>　　妖兽森林内的妖兽大规模涌出，数以十万计。
<br/>　　每一次，这边的武者都要付出极大的伤亡，才能将这些妖兽赶回妖兽森林。
<br/>　　那些巨大城墙上的痕迹，就是这无数年来人类武者与妖兽一次次交锋所留下的。
<br/>　　那几人继续道，“无数年来，每次的兽潮，都是差不多的时间。”
<br/>　　“但这次，却毫无预兆，来势汹汹，绿光城那边，差点失守。”
<br/>　　“十八城城主，立刻调动士兵前往守护。”
<br/>　　“猎妖殿，则紧急召集十八城猎妖师，火速驰援。”
<br/>　　“我们疾风猎妖队，九成以上的猎妖师都去了。”
<br/>　　“原来如此。”萧逸点点头。
<br/>　　那几人疑惑问道，“易霄大师你这些天去哪了？这么大的事你竟然不知道。”
<br/>　　“额。”萧逸没有解释。
<br/>　　那几人面带愁容，道，“听说，绿光城那边，形势很不乐观。”
<br/>　　“这次兽潮，妖兽数量之多，实力之强，远非以往能比。这边的猎妖师和武者，也一时没有准备，被打了个戳手不及，死伤惨重呢。”
<br/>　　“我还听说，裂天剑派有一群弟子恰好在那边历练，现在都困在那里，生死未卜。”
<br/>　　“裂天剑派弟子？”萧逸一惊。
<br/>　　“嗯。”那几人回答道，“好像是内堂首席叶铭带队，还有好些个内门弟子。”
<br/>　　“其中有一人是我们东荒十八城以前有名的猎妖师，雷战。”
<br/>　　“还有一人也是这一带曾经颇有名气的猎妖师，名叫林劲。”
<br/>　　“其他人我就不知道了。”
<br/>　　“林劲？”萧逸脸色一变，身影瞬间消失在原地。
