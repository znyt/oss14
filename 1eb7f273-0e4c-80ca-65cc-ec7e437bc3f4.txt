叶翎竖起了中指。
<br/>
<br/>        不知道是对他，还是对着后面的夏妍妍和霍璇。
<br/>
<br/>        夏妍妍和霍璇脸色变了又变，顾夜瑾舌尖顶了一下右腮，薄唇勾起了一道暗色弧线，他抬手用力扯了一下脖间的纽扣。
<br/>
<br/>        ……
<br/>
<br/>        送完了夏妍妍和霍璇，顾夜瑾将迈巴赫停在了醉玉欢的楼下，他抬头看着上面亮起的明黄灯光。
<br/>
<br/>        这时“叮”一声，陆寒霆的短信来了今天叶翎带我的陆太太去哪里玩了？
<br/>
<br/>        陆寒霆今天在公司开会的时候就收到了消费短信，他给了夏夕绾黑金卡，不过这么久了她都没有动过，他的陆太太是有自己的坚持和小骄傲的。
<br/>
<br/>        所以今天收到消费短信他还是挺意外的，当时就勾了一下薄唇笑了。
<br/>
<br/>        开会的公司高层不明所以，还以为他又进账了多少个零了。
<br/>
<br/>        顾夜瑾回了一条你怎么不去问你的陆太太？
<br/>
<br/>        陆寒霆她刚刷我的卡，如果我问了，会吓跑她的，以后她不刷了怎么办？
<br/>
<br/>        顾夜瑾…
<br/>
<br/>        陆寒霆叶翎不是被你吓跑了么？
<br/>
<br/>        顾夜瑾觉得这天是没有必要聊下去了，他想扔手机，但是想了一下，他又很有耐心的回了一条我好像见叶翎带你的陆太太买了真丝睡裙，还做了美甲，应该都是你喜欢的。
<br/>
<br/>        陆寒霆已经回到了幽兰苑，现在人在书房里，他将顾夜瑾这条短信来回看了几遍，他的陆太太买真丝睡裙做美甲去了。
<br/>
<br/>        陆寒霆让叶翎在海城多待些天，别着急走。
<br/>
<br/>        顾夜瑾将手机丢在了副驾驶座椅里。
<br/>
<br/>        陆寒霆放下手机，垂眸批阅文件，但是他一个字都看不进去，所以就起身出去了，找奶奶。
<br/>
<br/>        “奶奶，你想不想绾绾？”陆寒霆问。
<br/>
<br/>        陆老夫人点头，“想啊，绾绾这都两天没回来了。”
<br/>
<br/>        “那我们跟绾绾视频聊天吧？”
<br/>
<br/>        “我看行。”
<br/>
<br/>        ……
<br/>
<br/>        醉玉欢。
<br/>
<br/>        夏夕绾刚洗了澡，穿了今天买回来的裸色真丝裙，刚用毛巾将潮湿的长发擦擦干净，这时她的微信就响了。
<br/>
<br/>        陆寒霆发来的奶奶想你了，想跟你视频。
<br/>
<br/>        视频聊天…
<br/>
<br/>        如果是陆寒霆想跟她视频，夏夕绾是绝对不会接的，但是奶奶想她了…
<br/>
<br/>        这时陆寒霆的视频聊天已经快速的拨了过来。
<br/>
<br/>        夏夕绾接通了，陆老夫人慈祥的脸庞迅速出现在了手机上，“呀绾绾，你今天没有戴面纱啊，啧啧，我家绾绾真美啊，像天上下来的瑶池仙女。”
<br/>
<br/>        夏夕绾摸了一下自己的脸，她刚洗过澡正在擦头发，没有戴面纱，而且这个公寓里只有她和翎翎，没有戴面纱的必要。
<br/>
<br/>        现在被老夫人看到了，夏夕绾绝丽的小脸慢慢一红，“奶奶，你就取笑我。”
<br/>
<br/>        “奶奶想绾绾了，绾绾想奶奶么？”
<br/>
<br/>        “恩，”夏夕绾小鸡啄米般点头，“绾绾也想奶奶。”
<br/>
<br/>        “虽然奶奶很想绾绾了，但是绾绾不要为了奶奶着急回家，等你什么时候心情好了再回来。”
<br/>
<br/>        夏夕绾的心里流淌出一股暖流，都说家有一老如有一宝，虽然奶奶上年纪了，但是奶奶心如明镜，什么都知道。
<br/>
<br/>        夏夕绾是感动了，陆寒霆却一脸陌生的看着自家奶奶，奶奶，究竟谁是你亲生的？
