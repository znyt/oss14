        但是，后来他又觉得，这个童筱颖若真是“小谢”，“小谢”都不愿意屈服于他，童筱颖一定不会上他的鱼钩。
<br/>
<br/>        所以，季思妍才想了另一个法子，就是让他先匿名追求童筱颖，然后在伺机约童筱颖见面，想尽办法都要拍一些童筱颖的暧昧照回去。
<br/>
<br/>        季运恒本来想搞阴险的那一套，后来又觉得，还不如今天这套来得干脆利索。
<br/>
<br/>        他开辆豪华跑车，直接去学校里“劫人”，就算被人看到了，大家顶多误认为他俩之间只是情侣的小吵小闹。
<br/>
<br/>        季运恒将车停稳后，又将跑车的折叠车顶收了回去，然后直接脱了自己的上衣，光着膀子，从裤兜里掏出手机来。
<br/>
<br/>        “你到底想做什么？”童筱颖恼火地呵斥道，她一直在扭动着手腕，想从手铐里挣脱出来，但是手铐的洞口太小，将她的手腕勒出了红印。
<br/>
<br/>        “你放心，我不会对你怎样，只不过是想和你一起拍几张照片而已。”季运恒咧嘴道，突然从驾驶座上站起身来，直接跨坐在了童筱颖的大腿上，抑制住她的双腿不让她乱动。
<br/>
<br/>        童筱颖另一只手得空，一边捶打着他，一边将他的身子往外推，大喊大叫起来：“你混蛋！放开我！救命啊！救命！”
<br/>
<br/>        季运恒大手一挥，“啪”地一声，掴在了童筱颖的脸颊上。
<br/>
<br/>        他下手又狠又重，将童筱颖的嘴角掴出了血丝。
<br/>
<br/>        季运恒又从储物箱里拿出另一只手铐，利索地将童筱颖的另一只手拷在了车椅上。
<br/>
<br/>        他的储物箱里不仅仅只有手铐，还有其他更多的成人用品，只要是能用在女人身上的，他这里全都应有尽有。
<br/>
<br/>        由此可见，他的私生活是多么混乱。
<br/>
<br/>        童筱颖看着那些东西，觉得有些难以启齿。
<br/>
<br/>        “撕拉”一声，季运恒双手用力粗暴地撕开了童筱颖的衣襟。
<br/>
<br/>        季运恒还真没想到，这光天化日之下，童筱颖这脸蛋不仅漂亮、声音清甜，就连身段都如此娇美。
<br/>
<br/>        如果，他那晚能真正的要了她就好了。
<br/>
<br/>        只可惜，他现在已经没有机会了。
<br/>
<br/>        季运恒不敢再对童筱颖乱来，只好压抑着自己心里的欲火。
<br/>
<br/>        “咔嚓咔嚓咔嚓——”
<br/>
<br/>        季运恒拿着手机给童筱颖来了一套连拍，拍完后，他立即把照片发给了妹妹季思妍。
<br/>
<br/>        童筱颖恼羞成怒地破口大骂：“你是禽兽！混蛋！猪狗不如！”
<br/>
<br/>        “骂！你使劲骂！我还有更刺激的东西！”季运恒咧嘴道，从储物箱里拿出一个粉色的软软的椭圆形的小东西和一只无线按钮，在童筱颖的眼前晃了晃，“你知道这是干什么的吗？”
<br/>
<br/>        季运恒咧嘴阴笑。
<br/>
<br/>        他暧昧地撩起了童筱颖的裙摆，将这个粉色的小东西放到了童筱颖的两腿间。
<br/>
<br/>        童筱颖恼火地怒斥道：“你混蛋！”
<br/>
<br/>        季运恒一只手按着手里的按钮，另一只手拿着手机对着童筱颖录像，勾唇一笑：“是不是很舒服？”
<br/>
<br/>        “你——变态！”童筱颖咬牙切齿地从嘴里挤出这句话，愤恨地瞪着季运恒。
<br/>
<br/>        一阵软软的冰凉感带着微微振动的频率从身下传来，刺激着童筱颖的敏感神经，没有痛觉，却令她浑身酥麻起来。
<br/>
<br/>        季运恒忽然间很期待，若是傅昼景看到这个视频后，会是一副什么样的表情？
<br/>
<br/>        “要不，再来点更劲爆的？”季运恒手里捏着按钮，将“魔爪”伸到了童筱颖的胸前。
<br/>
<br/>        当他想脱去她的胸衣，拍一些更露骨的视频的时候，身后突出传来一阵急促的刹车声。
<br/>
<br/>        季运恒下意识回过头去，看到一辆黑色越野车，而且还是比较独特的车牌号码，顿时就知道对方来头不小。
<br/>
<br/>        他想都没想，立即从童筱颖的大腿上跨回自己的驾驶座。
<br/>
<br/>        正当他启动了车子，想带着童筱颖一起逃走的时候，那辆霸气的黑色越野车直接横冲直撞了过来，在与他的车头仅仅只差几厘米的距离下稳稳停住，连倒车逃离的机会都不给他。
<br/>
<br/>        “Shit！”季运恒急得猛拍了一下方向盘，下意识地朝那辆黑色越野车望去。
<br/>
<br/>        只见黑色越野车的驾驶座上的车门被打开，一个男人从车上下来。
<br/>
<br/>        他一米九的高个子和宽肩厚胸的完美身材，光洁的脸庞，透着棱角分明的冷峻，似是一件名家砖雕的艺术品，没有一丝不符，不怒而威，还有眉宇间那股帝王般的威煞之气。
