
<br/>　　<br/>　　    “就算完美，也没有可能下手针对恒晟地产的机会……”<br/>　　<br/>　　    王惊蛰话音刚落，那个一直没有开口的首席女律师顾西凤，就突然开口了：“没办法，太干净了，没有瑕疵就没有见缝插针的机会”<br/>　　<br/>　　    王惊蛰好奇的问道：“正常来讲，搬倒一家公司需要多久，或者哪些步骤？”<br/>　　<br/>　　    这一点，之前郭茜茜就说过，需要三个因素，生产事故，官方调查和资金链断裂，但这个说法太笼统了简单了点，王惊蛰很想听听这些因素后面的细节。<br/>　　<br/>　　    女大状皱眉说道：“你还想着从这方面下手？”<br/>　　<br/>　　    王惊蛰皱眉，反问道：“你们不是应该听我的么？”<br/>　　<br/>　　    顾西凤一滞，随即就平淡的解释道：“如果恒晟地产是一家上市公司的话，还有可能操作，做空他们的股票就行了，这样可以导致一连串的连锁问题，可是徐少中没有把公司上市，而且他的产业都是实体，你就很难做空他了”<br/>　　<br/>　　    王惊蛰稍微加重了自己的语气，说道：“重新说下我刚才的问题”<br/>　　<br/>　　    顾西凤挑了挑两条好像毛毛虫一样漂亮的眉毛，说道：“从我们律师的角度来看，如果一家很优质的企业濒临倒塌的话，是需要一个周期和很多细节的，首先就是一点，丑闻！现在是互联网传播非常发达的时代，假如有一家公司忽然爆出丑闻，并且在速度极快的情况下传播出去，那就会给它带来很大的负面影响，这么说吧，明天恒晟地产正好在上饶和鹰潭都有两个楼盘的住户要入户和新盘发售是一起进行的，如果此时有丑闻爆出来，那么入户的人将会选择拒绝入户，同样，购买楼盘的人也不会购买，就像去年的碧绿园一样，因为质量问题最后公司高层不得不出来道歉，认错，整顿，于此同时恒晟地产要是爆发质量问题和建筑事故的话，那么官方就会介入了，这还得回到丑闻传播这一点上，因为官方不可能坐视民众有意见，所以他们对恒晟肯定得要下达文件，比如停业整顿，安全检查等等，当这个阶段到达高潮时，如果恒晟地产的资金链断裂的话，那他们就危险了，领导人势必要急的成为热锅上的蚂蚁团团转了，因为一家地产公司会涉及到很多的资金运转，像工人的工资，材料商的供应，企业人员的薪酬还有各种设施的维护和运转等等，方方面面非常的多，钱出了问题，公司就兜不住了”<br/>　　<br/>　　    王惊蛰听的略微有点明白了，脸上写出两个字的表情，那就是“了解”。<br/>　　<br/>　　    “这么看来，也还是有迹可循的啊？”王惊蛰语气轻快的说道。<br/>　　<br/>　　    顾西凤翘了下嘴角，说道：“我说的这些是因素，但并不会有构成的可能，刚才已经说过了，恒盛地产的风评非常好，企业优良，而且没有不良负债，哪怕是他们在银行的贷款，在偿还能力上都是有结余的，所以因素有了，但成不成就难说了”<br/>　　<br/>　　    “等一下，有个话我刚才没有插进来”董良忽然摆了下手，说道：“有个问题，我觉得挺有可能的，按照常理来讲恒晟地产如此业绩良好状态优秀的公司，都是应该早就上市了的，可是令人惊奇的是徐少中一直都说在运作上市，特别是现在还有这个传闻，这就很不寻常了，一个可以绰绰有余上市的公司却迟迟没有上，我觉得只有一个可能”<br/>　　<br/>　　    顾西凤顿时接了一嘴：“为人洗钱？”<br/>　　<br/>　　    谭明哲也点头说道：“我也想说这个来的，可能性实在太大了，不然没有道理啊？”<br/>　　<br/>　　    王惊蛰不解的问道：“洗什么，什么洗钱？”<br/>　　<br/>　　    银行的财务官董良解释道：“这么说吧，比如你捡到一个亿的钱，家里肯定放不下啊，你就得存到银行里去，但是如此大笔的资金一旦进入银行户头，监管系统就会发现了，因为这些钱和你的收入不符，那就是来历不明的钱，监管方和警方就得问你了，这些钱是从哪里来的，你没有合理的解释是不行的，你哪怕说你是捡的，也得必须上交，洗钱就是这个意思，有人把来历不明的钱送进徐少中的公司，然后通过他的操作走一圈，这钱以各种方式在出来，交回给洗钱的人，这就干净了，不然真没办法解释他这么好底子的公司，为啥一直都不上市了”<br/>　　<br/>　　    这回，王惊蛰听的很认真，就是插话也是很详细问的，几个问题之后他渐渐的就把一些认识明了了。<br/>　　<br/>　　    看了下时间，此时已经晚上了，王惊蛰从沙发上起来，跟郭茜茜说道：“你跟他们晚上吃吃饭，我出去一趟办点事，有可能出门的时间有点长，我不回来你们也不用找我”<br/>　　<br/>　　    郭茜茜诧异的问道：“嘛去？”<br/>　　<br/>　　    “给徐少中下绊子去”王惊蛰回了他一句，然后扭头看着顾西凤他们四个，淡淡的说道：“你们说的东西我有概念了，从今天晚上开始，你们就研究下如何操作，操作的切入点就是刚才说的那些问题”<br/>　　<br/>　　    “什么意思？”张泽雨，董良等人都不解的问道。<br/>　　<br/>　　    王惊蛰说道：“我的意思是，刚才说的那些问题，将会在未来几天内一一变成现实，你们要做的就是在变成现实以后，推波助澜，火上浇油，在最快的时间内，把徐少中往深渊里推，最后逼得他自己都处理不了，然后寻求向上面求助，这个时候他求助谁，谁帮他压着，那就是徐少中幕后的人了，明白？”<br/>　　<br/>　　    顾西凤直接摇头说道：“不明白，因为你说的完全不可能发生”<br/>　　<br/>　　    “那你就拭目以待吧……”王惊蛰扔下一句话，转身就走了。<br/>　　<br/>　　    顾西凤极其无奈和不解的冲着郭茜茜问道：“他今天吃药了么？”
