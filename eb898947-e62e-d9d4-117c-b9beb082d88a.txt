　　田不易看完聚元功，转头惊奇的看着苏重：“这真是你自创的功法？”
<br/>　　“是的。”苏重坦然承认。
<br/>　　田不易盯着苏重出神半晌，最后叹了口气：“老天对你真是不公啊。”
<br/>　　他看懂了聚元功运行方式，但草木之精是什么，为什么会有强化身体的效果，田不易却看不明白。
<br/>　　苏重小小年纪能创出如此功法，让他惊叹不已，他不由想到青叶祖师。那是一人带领青云崛起的绝世天才。
<br/>　　可想到苏重体质，田不易顿时遗憾摇头。即便悟性惊人却依旧无法修炼，前途渺茫啊。
<br/>　　“徒儿，聚元功吸来的灵气不能用，原因在于不纯粹。”他看不懂聚元功原理，却明白苏重的问题。
<br/>　　“太极玄清道是借助天地灵气修炼，让我们更加贴合天地。聚元内功吸引的灵气已经沾染了聚元功性质，所以不能修炼玄清道。”
<br/>　　苏重恍然大悟。太极玄清道之所以神奇，是因为它充分借用了灵气奥秘。或者说这个世界的修仙力量，都是在借助灵气改变人身本质。
<br/>　　如果灵气不在纯粹就无法达到原本效果。
<br/>　　“不对！”苏重突然想到：“其他门派功法可能并不像玄清道，并不一定需要纯粹灵气。为了某种效果，他们反而会改变灵气性质！不然也就不会有千奇百怪的功法能力。”
<br/>　　苏重恍然，不是所有功法都需要纯粹灵气，只是太极玄清道太过特殊。
<br/>　　田不易显然也知道玄清道的本质：“徒弟，你可以想办法提纯聚元功，让它无限接近灵气，这样就能修炼玄清道。”
<br/>　　沉吟片刻，田不易又开口道：“或者继续改进聚元功，沿着这条路走下去。”
<br/>　　说出这句话对田不易来说挺困难。让苏重放弃玄清道，继续修炼聚元功。相当于放弃青云根本。那些邪魔外道之所以是邪魔外道，不只因为他们行事残暴恶劣，根本上他们的修炼路子就不一样。
<br/>　　苏重若有所思，摆在他面前有两条路，一个是纯化聚元功，一个是继续开发聚元功。
<br/>　　几乎没怎么犹豫，苏重就选了第二条。
<br/>　　第一条看似简单，只要聚元内力足够纯粹，就不会熏染灵气，就能修炼玄清道。
<br/>　　可苏重知道，只要他体内还有聚元内功，灵气始终都会被污染。
<br/>　　“即便是很小的一部分，甚至不影响玄清道运行的一部分，日后也会产生巨大变化。”
<br/>　　灵气不纯，和别人相比就差了一层，日积月累，他只能在别人后面吃灰，这不是苏重想要的。
<br/>　　弄明白玄清道本质，苏重突然发现。聚元功实际上已经不算世俗武功。
<br/>　　他已经可以看做初期的修仙功法。只是和青云不同，运用的不是纯粹天地灵气。反而和那帮邪魔外道一样，用其他东西熏染天地灵气，然后达到特殊效果。
<br/>　　“徒弟，你这聚元功能交给你其他师兄吗？”田不易不好意思的问道。
<br/>　　苏重思绪停止，连忙笑着点头：“当然没问题。”
<br/>　　想到之前田不易看到他徒手折断黑竹时的惊讶，苏重知道田不易这是看上了聚元功强身健体的功效。
<br/>　　苏重也没想到聚元功强化身体功效那么好。他创造聚元功是，下意识的把营卫二气概念融入其中。结果练出来的聚元内力格外滋养身体。
<br/>　　“师傅，您是想让师兄们用聚元功滋养身体吗？”苏重问道。
<br/>　　“不错，我青云秘法神剑御雷真诀，需要强横身体素质。有聚元功辅助修炼，将会大大缩短修炼时间。”田不易道。
<br/>　　“那修炼聚元功就不太合适了。聚元功最大特点是内力量大，有利于突破先天。如果只是想运用内力滋养身体，需要做些修改。”苏重解释道。
<br/>　　田不易点头理解，然后又摇头劝道：“修改功法可不是那班容易，现在就挺好。”
<br/>　　然后苏重就笑着看田不易，他也不说话，就这么呵呵笑。
<br/>　　田不易一愣，然后反应过来，一巴掌轻轻拍在苏重后脑勺上：“为师忘了，我们大竹峰还有你这个天才在呢！”
<br/>　　对于一个月创造功法的苏重来说，稍微修改一下聚元功还真不是问题。
<br/>　　“那找个时间，你给你的几个师兄弟上一堂课吧。”田不易拍板。
<br/>　　“修炼的事情也不要着急，稍后我给你找些功法参考，对你有好处。”田不易修炼数百年，手里着实攒了不少存货。有些东西太敏感，自然不能放在藏书楼。现在苏重需要继续改造聚元功，需要开阔视野，他当然不会吝啬。
<br/>　　第二天，一起吃完早饭后，田不易把几个徒弟留住。
<br/>　　“你们小师弟这些天勤学苦读，创造出了一套聚元功。能快速强化身体，我打算让你们都学一学。今天就让你们师弟教给你们。”田不易道。
<br/>　　见钟徒弟楞楞发神，顿时一拍桌子：“愣什么呢，还不道谢！”
<br/>　　几个徒弟这才从惊讶中回过神，一个个连忙向苏重道谢。田不易这才满意点头，然后和苏茹一起离开。
<br/>　　两人一走，众人顿时炸了锅。
<br/>　　“二蛋，你可以啊！竟然创造了功法，连师傅都惊动啦！”杜必书满脸惊讶。
<br/>　　“二蛋师弟的聚元功真有那么好？”田灵儿满脸的不相信。
<br/>　　苏重和她差不多年纪，之前她一直修炼玄清道领先，现在突然被苏重超过，顿时不乐意起来。而且，她实在不相信苏重能创造什么神奇功法。
<br/>　　几个师兄弟都好奇的看着苏重，但也紧紧是惊奇自家小师弟能创造功法，对功法本身显然也没怎么在意。
<br/>　　苏重看着众位师弟眼中惊奇，笑着摇了摇头。他也不说话，大步走出房间，来到祠堂门口。
<br/>　　青云祠堂内供奉历代大竹峰弟子牌位，门口放着一个青铜大鼎，用于特殊时间拜祭燃香。
<br/>　　走到大鼎前，一手抓住通鼎一脚猛然乏力，一下就把铜鼎举了起来。
<br/>　　众位师兄顿时目瞪口呆。瘦瘦小小的师弟，竟有这么大力气？！
<br/>　　“这是聚元功的威力？”杜必书顿时惊喜起来。
<br/>　　宋大仁早知道苏重身强力大，可看到苏重轻松举鼎，顿时也被吓到。
<br/>　　昨天徒手折断黑竹，宋大仁就已经够吃惊。现在看来，小师弟还是藏拙啦！
<br/>　　但旋即就心情热烈起来，只要练好聚元功，让身体素质提升。他就能修炼神剑御雷真诀，而且还不怕反噬伤身！
