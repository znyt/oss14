<br/>				    关习凛和冷华相视了一眼，两人来到他面前，一人坐下，一人则站着。<br/>　　<br/>　　    “你说，小九到底出什么事了？”关习凛沉着声音问着，心想，莫不是还真让陌尘给说中了？小九又遇到什么劫难了不成？<br/>　　<br/>　　    “我们在陆河城遇到了魔族魔主手底下的十大魔修之一的血罗，他……”段夜将事情原原本本的跟他们仔细说了一遍。<br/>　　<br/>　　    最后，他才道：“就是因为这样，她被吸进了那黑暗之门，而血罗也被后来赶到的几大仙宗的仙尊所杀，我询问过几位仙尊，他们对那黑暗之门之事也知之甚少，所以我才赶过来跟你们说一下这事，想跟你们们商量一下，看看你们有没什么办法。”<br/>　　<br/>　　    听到这话，两人皆沉思着，他们本就非这上游大陆的人，对这边的一些事情也并不了解，就更别说那什么黑暗之门的事情了，此时听到段夜这么说，两人只知事情很是严重，却是无计可施。<br/>　　<br/>　　    “这样吧！你先在这里住下，将伤养好，我去找陌尘，他是天机老人的弟子，我想他应该知道这黑暗之门的事情，兴许，他会有什么办法也不一定。”关习凛站了起来说着。<br/>　　<br/>　　    闻言，段夜点了点头：“眼下也只有这样了。”<br/>　　<br/>　　    “冷华，他受了伤，你让范霖给他看一下，然后看要服用什么样的丹药，从楼中取了给他服下。”关习凛交待着。<br/>　　<br/>　　    “好，我知道了。”冷华点了点头，看着他出了院后，这才道：“段公子，你先坐会，我去叫范霖过来。”<br/>　　<br/>　　    “不用麻烦了，我这内伤过几天就会好。”段夜说着，又道：“当初凤九给了不少丹药给我，过来时我也已经服下内丹了。”<br/>　　<br/>　　    “待范霖看过后，若说不用那便不用，若是内伤还没好，那就再服下一枚丹药也不迟。”冷华说着，对他道：“你且坐着，我去去就来。”<br/>　　<br/>　　    他往前面去，不多时，将范霖带了过来给段夜把脉……<br/>　　<br/>　　    而在另一边，关习凛出了天丹楼就朝纳兰府而去，由下人领着来到陌尘的院子后，一见到陌尘，他还没开口，倒是陌尘便问出声了。<br/>　　<br/>　　    “可是出什么事了？”<br/>　　<br/>　　    一听这话，关习凛大步来到院中的石桌坐下，瞥了他一眼，道：“你这张嘴，真是说什么准什么，还真让你说中了，小九出事了。”<br/>　　<br/>　　    听到这话，陌尘并不意外，他负着手缓步走着，来到桌边坐下，问：“出什么事了？如今她人在哪里？你如何得到的消息？”<br/>　　<br/>　　    “就是在陆河城，她和段夜在城主府里遇到了十大魔修之一的血罗，被那血罗打开的黑暗之门给吸进去了，现在也不知去了哪里。”<br/>　　<br/>　　    关习凛没好气的说着，问：“这黑暗之门是怎么回事？被吸了进去会被传送到哪里去？怎么那段夜说连仙宗的仙尊也不知道那黑暗之门通往何处？你可知道？”<br/>　　<br/>　　    陌尘眸色微动：“黑暗之门？”他若有所思的道：“这对她来说，是祸，也是福。”<br/>　　<br/>
