
<br/>　　<br/>　　    鬼面掐住王惊蛰的脖子，他根本都没啥反应，任由对方直接上了他的身，所谓的鬼上身无非就是鬼占据了人的三魂七魄才能达到操控人的目的。<br/>　　<br/>　　    就拿北方的出马请阴来讲，出马仙从阴间请魂上身附在了自己的魂魄上，就是暂时放弃了对自身的掌控。<br/>　　<br/>　　    但王惊蛰肯定不怕这个，他的三魂七魄连阴帅都勾不出来，何惧一道小鬼？<br/>　　<br/>　　    鬼面上身之后，从王惊蛰体内就冒出一声惊恐而又绝望的尖啸，他的体表蔓延出一层黑气，挣扎着似乎想要脱离开来，但王惊蛰忽然划破指尖在自己的眉心上快速的写了一个封字，字一成型就隐没在了他的印堂下方，随即他身上黑气渐渐收敛“噗”的一下全都钻进了他的体内。<br/>　　<br/>　　    王惊蛰两眼短暂的失神过后就恢复了清明，他轻轻的吐了口气，自言自语的说道：“在里面老实的呆着，想来你也知道是怎么回事了，给我眯着吧，过会我在找你问话······”<br/>　　<br/>　　    王惊蛰说完就来到陈臣身边坐下，刚才那一瞬间她根本就不知道发生了什么事，唯一的感觉就是自己的身体忽的一轻，仿佛卸下了千斤重担，整个人都轻松了不少。<br/>　　<br/>　　    “喝点水吧”王惊蛰递给她个杯子，等陈臣稍微稳定过后，他就接着说道：“毫无疑问的是，你肯定被人下手段了，有人在刻意针对你”<br/>　　<br/>　　    陈臣茫然的问道：“为什么？”<br/>　　<br/>　　    “那就得问你自己是不是得罪什么人了，你被这东西缠上，短时间内是要不了你命的，这就像是一种慢性病，会熬的你身子发虚，有气无力，做什么都不顺，直到一年半载后你被生生的给熬死了，干这种事的人无疑是很缺德的，并且这么做也会损耗自己的功德，说白了就是损人不利己的勾当，所以一个人要是跟你无冤无仇的话，是绝对不会这么干的”<br/>　　<br/>　　    “我不记得自己什么时候和人发生过争执了，我的工作也很难和谁产生什么冲突，得没得罪过人，我也真的不记得了······”<br/>　　<br/>　　    王惊蛰沉默半晌，说道：“如果不是你，那就有可能是你的家人，或者你的男朋友什么的了”<br/>　　<br/>　　    菜刀文无语的跟丁武说道：“看见没有？这是套话呢，这厮真骚气，么的，还能这么问人家有没有对象啊？”<br/>　　<br/>　　    “嗯，转性了！”丁武点头说道。<br/>　　<br/>　　    “我没有男友”<br/>　　<br/>　　    王惊蛰“哦”了一声，点头说道：“那就是你的家人了”<br/>　　<br/>　　    陈臣喝了口谁，叹着气说道：“那就只能是我爸爸了，他这辈子不知道得罪过多少人了，不过他们为什么不直接朝我爸爸下手，拐到我身上干什么？”<br/>　　<br/>　　    “可能是有忌讳或者不方便的地方，也可能是他的身份·······”<br/>　　<br/>　　    菜刀文忽然在旁边插了一句：“如果你祖上有人身份太敏感也不行，你是女儿身不承载家族功德，但你爸爸就不同了”<br/>　　<br/>　　    这个年代正是立国后几十年的时候，如今还有多位开国功臣在世呢，菜刀文的意思是说如果陈臣祖上有人在此行列的话，那她的父亲就承受祖辈的余荫了，她是女儿身则就不受了，有句话说的很难听但也很残忍，就是女儿是嫁出去的人泼出去的水，大概也是这个意思。<br/>　　<br/>　　    在古代，为大官者享皇朝气运乃是共知的事实。<br/>　　<br/>　　    那些所谓的几品大员们，魑魅魍魉妖魔鬼怪，都是不能近身的。<br/>　　<br/>　　    陈臣无语的说道：“这也要区别对待么？”<br/>　　<br/>　　    菜刀文摊着手说道：“你嫁出去之后，随夫姓自然不受祖宗余荫了，以前称呼妇人家不都是在名字前面加上夫姓的么，比如你嫁给个姓王的，就叫王陈氏了”<br/>　　<br/>　　    王惊蛰瞥了他一眼，心道这货真是能乱打比方，你就不能换个人么。<br/>　　<br/>　　    陈臣叹了口气，说道：“那我爸，就算是个红色子弟吧！”<br/>　　<br/>　　    “所以啊，对你爸下不了手，就得转嫁到你身上来了，不过我估计对方用这种手段也没想着把事情做绝了，应该是要跟你爸谈谈，胁迫他什么的，不然你就不只是鬼上身这么简单了······”<br/>　　<br/>　　    “我爸的事情我向来不管，也从来都不问”陈臣摇了摇头后，然后用一种充满了希望和渴求的目光看着王惊蛰。<br/>　　<br/>　　    王惊蛰皱了皱眉也没吭声，菜刀文无语的说道：“她是让你好人做到底，既然管了就别中途而废，把善后也给解决得了”<br/>　　<br/>　　    “嗯，嗯”陈臣连忙点着小脑袋。<br/>　　<br/>　　    王惊蛰说道：“你不说我也得管，我这么干已经和对方有了过节，打蛇不死反受其害······斩草就得除根啊”<br/>　　<br/>　　    菜刀文指了指王惊蛰，说道：“问它，人在哪？”<br/>　　<br/>　　    几分钟之后，王惊蛰和陈臣还有丁武，菜刀文从家中出来，上了车后他跟陈臣说道：“我指路，你照着开，快一点过去，拖的久了对方就有可能察觉到会跑了”<br/>　　<br/>　　    于此同时，某个偏僻小区里，一长得枯瘦的男子盘膝坐在床上，他的身前放着两个被扎起来的草人，上面压着符纸写着两个人的生辰八字。<br/>　　<br/>　　    其中一个草人的眉心上插着一根针，针上颜色漆黑，正在微微的颤动着。<br/>　　<br/>　　    “唰”枯瘦男子这时又拿起一根长针，照着另外一个草人的头顶刺了过去。<br/>　　<br/>　　    长针的针尖刚刚碰到草人的头顶上，就没能在往下插得进去了，顿时就被挡住了。<br/>　　<br/>　　    “咦？”这人脸色一变。<br/>　　<br/>　　    同时，车里的王惊蛰也瞬间皱起了眉头：“他下手了，快点开······前面路口左转”<br/>　　<br/>　　    “嗡”陈臣一脚油门到底，车子顿时提速冲了出去。<br/>　　<br/>　　    枯瘦男子一针没有扎进草人中后，从身上掏出一个瓷瓶，打开后一股刺鼻的恶臭味瞬间就蔓延开了，他将长针伸进瓶里沾了两下，再拿出来的时候针上沾满了黑色的尸油。<br/>　　<br/>　　    “噗”这一次，长针刺进了草人的眉心，但扎入了针尖下端就没能再进去了，另外一头王惊蛰感觉眉心一阵刺痛迅速狂跳起来。<br/>　　<br/>　　    陈臣身上的鬼面已经转接到了王惊蛰的身上，如果此时还附在她身后的话，陈臣恐怕当场就得失去知觉昏过去了，和她母亲基本一样。
