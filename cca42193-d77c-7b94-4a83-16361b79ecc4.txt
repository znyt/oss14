　　闭关室内，萧逸盘膝坐下。
<br/>　　闭关修炼，对武者来说，除了吸收天地灵气，增幅修为外。
<br/>　　还有着明神静心之效，对参悟武道，提升心境，大有裨益。
<br/>　　说起来，这些天一直四处奔波，自己已经许久没真正闭关过了。
<br/>　　就算难得进入闭关室，也只是匆匆闭关数日。
<br/>　　此时，他静静地思索着。
<br/>　　闭关的主要目的，自然是为了修炼的。
<br/>　　而关于修炼方面，自然考虑到他现今修炼的两大功法。
<br/>　　冰尊圣纹诀，乃是以武道为基，庞大力量为源，烙印下冰纹。
<br/>　　日后的修炼，可通过不断吸收武道力量，加诸到冰纹上。
<br/>　　以此来增加冰纹基石的凝聚度与‘深度’，让得冰纹可以容纳更庞大的力量。
<br/>　　由此，冰纹将愈加强大。
<br/>　　可是，自己如今在武道力量上面的吸收，已达瓶颈。
<br/>　　修为境界增加之前，武道力量的吸收，在现今层次已是上限。
<br/>　　而自己的小世界又如此庞大，修为境界想突破，难度不小。
<br/>　　故，冰尊圣纹诀的修炼，暂时无法修炼了。
<br/>　　九阳轮回诀也一样。
<br/>　　在得到第五种世间强悍火焰之前，这门功法的修炼，也暂时停滞着。
<br/>　　萧逸微微皱眉，思索了一番阵法上的事。
<br/>　　毫无疑问，阵法师这门已经失传的职业，战力相当强悍。
<br/>　　可是，先不说阵法师的系统修炼，已经失传。
<br/>　　以阵法的玄奥程度，自己独自领悟或者专研，太过麻烦，也太过费时间。
<br/>　　当然，这一点不是萧逸所担心的。
<br/>　　以他的参悟力，若想真正参悟阵法，难度也不至于太过大。
<br/>　　可问题在于，哪怕是一些大势力中，流传并保留下来的阵法中，并没有什么太过强大的阵法。
<br/>　　皆是最低级的阵法。
<br/>　　也就是说，就算他参悟了一些阵法师知识，也无阵可布。
<br/>　　风雪大阵，算是这些流传下来的阵法中，较强的一种了。
<br/>　　而风雪大阵，又已烙印在天机圣盘上，他一个念头即可触发，如臂指使。
<br/>　　既是如此，现今并没有必要浪费时间去参悟阵法师的内容和知识。
<br/>　　日后若是有机缘，得到更强大的阵法。
<br/>　　那时，再作参悟也不迟。
<br/>　　思来想去，萧逸率先开始了魂师的修炼。
<br/>　　十天后。
<br/>　　萧逸猛地睁开了眼睛，眼中，精光暴现。
<br/>　　“呼…”一口浊气，微微吐出。
<br/>　　但他的眉头，却紧皱着。
<br/>　　小世界中，两大武魂。
<br/>　　控火兽武魂，被一股火属性魂力所包裹，甚是奇异。
<br/>　　而冰鸾剑武魂周遭，则空空如也。
<br/>　　这种情况，他之前就已经遇到过了。
<br/>　　修炼而出的冰属性魂力，会悉数被冰鸾剑的裂痕所吸收，且吸得一干二净。
<br/>　　萧逸的脑海中，闪过‘冰暴噬魂’这门魂技的内容。
<br/>　　半晌，他摇了摇头，有些无奈。
<br/>　　冰暴噬魂，是一门冰属性魂技；若无冰属性魂力支撑，根本施展不出。
<br/>　　不同于真气、元力等力量。
<br/>　　武者吸收外在力量，如天地灵气、丹药等等，可随意在体内转化成不同属性的真气或元力。
<br/>　　一般来说，元力属性，与武者的武魂契合。
<br/>　　当然了，也有不契合的。
<br/>　　举个例子，比如雷属性元力。
<br/>　　若萧逸愿意，大可吞服一些蕴含雷属性精纯力量的天材地宝。
<br/>　　在体内留下雷属性‘种子’，便可自由转换成这种属性的元力。
<br/>　　但，由于没有相契合的武魂，他对这雷属性元力的控制，会相当无力。
<br/>　　这就造成这种元力哪怕转换了，也威力不会太强，对战斗和实力提升，效果不大。
<br/>　　故，一般武者，若无特殊情况，不会做这种无用之事。
<br/>　　而魂力方面。
<br/>　　魂师的各种手段，因武魂而来，也必需由武魂施展。
<br/>　　故，魂力，只与武魂挂钩。
<br/>　　且，魂力，并非吸收转化而来，而是通过武魂凝聚修炼而来。
<br/>　　火属性武魂，只能凝聚出火属性魂力。
<br/>　　冰属性武魂，只能凝聚出冰属性魂力。
<br/>　　…
<br/>　　简单来说，萧逸如今修炼凝聚出的冰属性魂力，都会被冰鸾剑吸收。
<br/>　　他根本没有可用于施展‘冰暴噬魂’的冰属性魂力。
<br/>　　这一点让他相当无奈。
<br/>　　而控火兽凝聚出来的火属性魂力，由于没有这种属性的魂技去施展，威力也不强。
<br/>　　这些火属性魂力，萧逸同样一个念头，便可无形无影地释放而出。
<br/>　　但，它们就像一盘‘散沙’，无法凝聚成一股强大的力量。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　苦修十天，并无太大的实力增长。
<br/>　　魂师手段，暂时只能用作奇招。
<br/>　　萧逸摇了摇头，将心思放到最后一项修炼上，修罗战体的肉体修炼。
<br/>　　说起来，他已经有许久没有修炼修罗战体了。
<br/>　　一来，他的武道修炼速度太快，本身的实力，已远远超出了修罗战体赋予的战力。
<br/>　　二来，他一直没有存够足够的妖兽精血。
<br/>　　若无法一次性大幅度增加肉体力量，修炼来也是无用。
<br/>　　故他一直卡在修罗战体第四层，赋予的肉体力量，也仅在地元九重巅峰。
<br/>　　虽说经受过天雷炼体后，肉体力量，达到天元层次。
<br/>　　但修罗战体的层次，始终只在第四层。
<br/>　　这近一年来，他倒是存够了不少妖兽精血，自然要再次开展修炼。
<br/>　　在极寒之地时，他斩杀了大量妖兽。
<br/>　　前些天，还击杀了冰承天、寒冰殿主、不夜宫主等极境强者，得了他们的修炼之物。
<br/>　　他们的乾坤戒中，有不少七级以上，乃至八级初阶的妖兽精血。
<br/>　　最重要的是，上次回主殿，殿主将剩下的一半奖励，给了他。
<br/>　　五万颗妖兽内丹精血，四万瓶七级巅峰阶品，一万瓶八级初阶阶品。
<br/>　　不得不说，猎妖殿不愧是最古老的势力，底蕴丰厚得惊人。
<br/>　　言归正传。
<br/>　　萧逸彻底开始吸收妖兽精血。
<br/>　　足足数天后，才将所有妖兽精血吸收一空。
<br/>　　而此时，修罗战体第四层，早已突破，达到第五层巅峰。
<br/>　　这代表着，他的肉体力量，已达天元境巅峰。
<br/>　　同时，修罗之力，达到80滴。
<br/>　　当然，这80滴修罗之力，也比以前升华了许多。
<br/>　　现在的一滴，怕是就比以前的20滴，蕴含之力量，更为强大。
<br/>　　萧逸猜测着，待得修罗之力积蓄到100滴时，便是突破第六层之时。
<br/>　　那时，肉体力量，便是极境实力。
<br/>　　地极境武者，因武道力量，而拥有着强悍实力。
<br/>　　可修罗战体，踏入第六层时，却仅仅靠肉体力量，便能拉平武道力量的差距。
<br/>　　修罗战体之强大，自不必多说。
<br/>　　“咔咔咔。”萧逸握了握拳头，感受着自己爆炸性的肉体力量。
<br/>　　锵的一声，暴雪剑凭空而现。
<br/>　　依稀记得，以前自己手持暴雪剑时，觉得相当重，拿起来极其费劲。
<br/>　　只有开启了‘冰纹’时，才达到举重若轻的效果。
<br/>　　可现在，单凭肉体力量，手持暴雪剑，却如若无物，轻松至极。
<br/>　　“现今，单凭肉体力量，手持暴雪剑，便有极境战力。”萧逸喃喃自语。
<br/>　　“若是此时再开启冰纹，战力又会达到何种层次？”
<br/>　　萧逸眼中充满了期待之色。
