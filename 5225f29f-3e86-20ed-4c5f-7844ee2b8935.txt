　　以正常速度御空而行，半日的时间，就隐约看到一座望不到边际的黄土城市，周围的温度也提升许多。
<br/>　　塔戈尔大沙漠，加玛帝国最大的沙漠，位于东部省份，望着自己的目的地，慕清霄搂在小医仙在距离城市百米处停了下来。
<br/>　　或许是由于接近沙漠的原因，这里的天气颇为干燥与炎热。
<br/>　　炽热的阳光从天空中挥洒而下，将脚下的大地烘烤得不断散发着熏人的热气，那股热气缓缓升腾而起，竟然也使得人的视线，出现了许些扭曲与模糊的视感。
<br/>　　对于周围的热气，慕清霄倒是没什么感觉，小医仙却是极其的不适应。
<br/>　　在这个地方，体质娇弱的她明显熬不下去，总不能让自己的女人跟在身后受苦吧，于是慕清霄左手一挥，就将升腾而起的热气隔绝在一米外。
<br/>　　感受到空气清新许多，小医仙甜甜一笑，内心甜蜜极了，温柔道：“我们现在去哪？”
<br/>　　“先进城吧，找一个人顺便买张地图。”
<br/>　　慕清霄望着一眼看不到边际的黄土城市，嘴角勾起一丝笑容，若是自己去找‘青莲地心火’，绝对会像头苍蝇似得，到处乱窜，但只要找到那个人，事情就好办了。
<br/>　　小医仙若有所思的点了点头，感受到一股热气袭来，快步跟上慕清霄的脚步，乖巧的跟在他身后。
<br/>　　距离黄土城市越来越近，路上的行人也开始多了起来。
<br/>　　望着路上的行人，小医仙快步走了上来，抓着他的衣袖，慕清霄则反过来，将她的玉手抓在手中。
<br/>　　由于周围气候炎热，男子的光着上半身，浑身皮肤显得黝黑，但看上去却显得精干豪爽，偶尔也有女子路过，皮肤并不想男子，却是呈古铜色。
<br/>　　此处的女子，并不像帝国内部女子含蓄娇羞，一套紧身的皮衣，刚好将****遮掩起来，纤细的腰肢却是大胆的暴露在外，下半身同样只穿着一些短裙或者短裤。
<br/>　　行走在路上，水蛇一般的腰肢扭动，看上去别有一番迷人的韵味与风情。
<br/>　　慕清霄只是瞥了一眼就失去了兴趣，比起小医仙犯规的纤腰，路上的简直弱爆了。
<br/>　　一路走到城门口，迎来了无数异样的目光，不光是因为两人的容貌，更是因为穿着与众不同，小医仙穿着白色长裙，慕清霄穿着白衫，在这种热死人不偿命的地带，简直就是异类。
<br/>　　黄土城市近在眼前，只见城门上方，两个硕大的淡红字体，被雕刻在城墙上，慕清霄拉着小医仙向城内走去，道：“沙漠中的城市，漠城。”
<br/>　　城门口，十几名身穿铠甲的士兵，正手持长枪，不顾炎热吆喝着进程的路人缴纳入城费用。
<br/>　　或许是因为天气太过炎热的关系，守卫在此的士兵，也是被这种鬼气候搞的极其烦躁，一道道毫不客气的喝骂声，不断的催促城门口的行人。
<br/>　　慕清霄拉着小医仙向城内走去，守卫的士兵看向他们的目光就像是在看珍惜动物，连催促声都给忘了，待两人的背影消失，叫骂声又再次响起。
<br/>　　走出有些昏暗的城墙通道，视线再次亮起，别具一格的城市建筑映入眼中，一幢幢模样古怪的房屋建筑，倒是与电影阿拉丁神灯中的有些相似。
<br/>　　慕清霄在大街上随意找了一位行人，后者脸色顿时沉了下来，见他拿出几枚金币后，瞬间眉开眼笑，道：“公子，有什么有小的能够帮忙的？”
<br/>　　“此处可有专门售卖地图的商店？”
<br/>　　“有的，有的......”
<br/>　　得知专门售卖地图的商店后，慕清霄就拉着小医仙转过街角，随后向着行人告知的商店位置走去。
<br/>　　不急不缓的走在路上，两人边聊边走，不知何时已经站在了一家名为‘古图’的地图店铺前。
<br/>　　店铺看上去，与买水源食物的店铺不同，没有一丝的豪华，却透露着一股古朴的气息，小医仙美眸眨了眨，道：“这个店铺有些奇怪呢。”
<br/>　　慕清霄微微点头，径直向店铺中走去，店铺内并不是太宽敞，墙壁上镶嵌着两枚月光石，将店铺照的颇为明亮。
<br/>　　视线在店铺中环视一圈，来此购买地图的人并不是很多，甚至可以说极为冷清，但他知道，确实是这里没错，缓缓走进店铺，视线停留在柜台后面的一位老者身上。
<br/>　　此时，老者正低着头仔细绘制着地图，年龄显得颇大，虽然已经满头白发，但绘制地图的干枯手掌，却显得强而有力。
<br/>　　老者抬起头来，与他年龄不符的是，眼眸没有一丝浑浊，反而透露出一丝沧桑。
<br/>　　视线落在小医仙身上，只是略微的打量了一眼，随后又定格在慕清霄身上，眼中闪过一丝惊异，因为面前的青年，他居然看不透！
<br/>　　海波东，加玛帝国上一代十大强者之一，曾在万宗大会上和一名斗皇一名斗王比试而不落下风，艺惊全场，米特尔家族太上长老，性子孤僻自傲，极其擅长冰系斗气。
<br/>　　因为得罪美杜莎女王被封印实力骤降至斗灵，被多数人遗忘，估计任谁都想不到，一个普通地图店铺的老板，居然会是曾经的斗皇强者。
<br/>　　慕清霄朝着海波东微微点头，也不在乎他的目光，视线在周围地图上扫过，饶有兴致的看了起来。
<br/>　　于是，店铺内就出现了诡异的一幕，慕清霄走到何处，老者的目光就跟到何处，以至于小医仙都有些害怕，紧紧拉着他的手臂，总觉得老者有些诡异。
<br/>　　最终，慕清霄停留在一个古朴木架前。
<br/>　　木架上的痕迹明显年代颇为久远，上面随意的堆放放着一些泛黄的地图，看这些地图表面的一些残破痕迹，似乎是制作地图时的失败品。
<br/>　　也不介意脏乱，慕清霄伸手在木架上翻找起来，片刻后，果然找到一张残破的地图，上面绘制着模糊的神秘路线。
<br/>　　小医仙望着他手中的地图，不禁想到了在青山镇魔兽山脉中所获得的藏宝地图，上面的纹路简直太相似了。
