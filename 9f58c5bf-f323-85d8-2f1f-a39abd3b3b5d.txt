<br/>　　要是可以得到人皇罐，只怕是现在楚风眠就可以轻而易举的灭杀生死境强者。

<br/>　　想要催动完整的极道仙兵，生死境强者都做不到。

<br/>　　其他的碎片，根本是不知去向，诺大九域，想要寻找齐，根本便是难如登天。
<br/>　　“不过就算是能够凑齐一半，也是天大的机缘啊。”
<br/>　　楚风眠看了看手中的两块人皇罐碎片，一股灵力便是打出。
<br/>　　在这一股灵力之下，这两道人皇罐碎片，便是开始融合，凝聚。
<br/>　　人皇罐在远古时代，不只因为什么原因被打碎，不过毕竟属于同源，随着楚风眠的力量催动，这两块碎片，已经是开始融合了起来。
<br/>　　其中的灵力，也是在节节攀升。
<br/>　　“幸好这赵无敌，没有催动人皇罐的力量，不然他突然拿出人皇罐来偷袭，虽然不至于可以击败我，却也可也有着全身而退的机会。”
<br/>　　楚风眠一边控制着灵力，将这两道人皇罐碎片合二为一，一边心中便是暗自想到。
<br/>　　这人皇罐，虽然只是其中一块碎片，但是却也足以将一尊半步生死境的强者一身灵力抽干。
<br/>　　楚风眠要是没有神脉，也根本不敢催动，但是就算是有着神脉的加持，楚风眠催动一刻，现在还要有着一息时间，失去力量。
<br/>　　而对赵无敌来说，更是不可能催动的了，催动的人皇罐，凭借他的灵力，还远远做不到。
<br/>　　“这两道人皇罐，属于同源，融合起来，没有任何的排斥，反倒是一种水到渠成的感觉。”
<br/>　　楚风眠感觉到，他手心中的人皇罐，如今已经合二为一，凝聚完成。
<br/>　　虽然看起来外观变化不大，还是破破烂烂的一个瓦罐，但是其中的力量，却是比起刚刚两者之和相加，都要强大了数倍。
<br/>　　“比起曾经的人皇罐，力量倒是强大了至少四五倍。”
<br/>　　楚风眠感觉了一下，脸上便是露出会心笑意。
<br/>　　这才是真正的大机缘，赵无敌在九溪国度的古迹中疯狂寻找奇物，还得到了这人皇罐，如今却是全部为楚风眠做了嫁衣，都被楚风眠落入到了手中。
<br/>　　这人皇罐的力量强大，楚风眠也就有了新的底牌，就算是真正生死境的强者来了，楚风眠也丝毫不惧。

<br/>　　她知晓楚风眠有着丰饶碑，这件天级灵器，如今这人皇罐上的力量，却是要比起丰饶碑，还要强大。

<br/>　　这四个字，在寒月漓的心中不由一震。
<br/>　　极道仙兵，这可是传说中的存在，如今居然就这样出现在了她的眼前。
<br/>　　虽然说只是残骸，但是的确是真正的极道仙兵。
