　　“这二人，跟燕皇有些关系，既然遇到了，就不能让他们白白死去！”
<br/>　　燕皇是楚风眠如今在大罗天星域之中少有交好的强者，如今居然是碰到了燕州大陆的人，被追杀。
<br/>　　楚风眠也没有坐视不理的道理。
<br/>　　“怎么办？难道我们真的要将这渡厄圣叶交出去？”
<br/>　　那其中名为燕莫的年轻古帝，盯着身后追逐而来的血虎五子，脸色惨白的说道。
<br/>　　他现在也知晓，他们现在这样在逃下去，也没有任何的意义，迟早一日会将这血虎五子给追逐到。
<br/>　　这血虎五子的实力，本来是要超越他们二人，更何况现在他们二人的身上，还有着不小的伤势。

<br/>　　令一人，那燕无峰的眼神之中看着血虎五子，眼神之中闪过几分坚决之意。
<br/>　　“我们就算是将这渡厄圣叶毁了，也不能让这渡厄圣叶落入到这些畜生的手中！”
<br/>　　“渡厄圣叶？这二人的机缘倒是不错，可惜这机缘到手，也是祸端。”
<br/>　　听到渡厄圣叶这个名字，楚风眠也明白过来，这血虎五子为何要一直追杀这燕州大陆二人了。
<br/>　　渡厄圣叶，是曾经历史之中赫赫有名的古圣药之一，位列上古圣药级别，但是论起价值，要比起远古级别的圣药，还要珍贵。
<br/>　　相传天地之中，有着一株神树，名为渡厄神树，这渡厄圣叶，正是渡厄神树枝叶上的嫩叶，百年才出三枚。
<br/>　　每一枚渡厄圣叶的出现，都足以引起大量的九劫古帝，甚至是半步天人境强者的争夺。
<br/>　　这渡厄圣叶，不能提升力量，也无法恢复伤势，但是却有着唯一的一种独特效果，也是天下间，唯有这渡厄圣叶才拥有的一种效果。
<br/>　　那便是足以减弱天劫之力！
<br/>　　渡厄二字，便是由此而来。
<br/>　　这渡厄圣叶，一旦是服下，便是足以让下一次天劫的威力减弱。
<br/>　　对冲击九劫古帝的天劫，足以减弱半数的威力，可以说一名八劫古帝，只要是得到了一枚这渡厄圣叶，冲击九劫古帝便是没有任何的难度。
<br/>　　传说中帝尊境最强的一道劫雷，在这一枚渡厄圣叶之下，就会被轻易的化解。
<br/>　　当然只是能够削弱九劫古帝天劫之力，还不足以如此的珍贵，这渡厄圣叶最为可怕的，是就连冲击天人境，号称死劫的天劫，都可以削弱。
<br/>　　虽然最多削弱其中一成的威力，但是对冲击天人境的存在来说，提升一成的实力，简直是比起登天还难，就算是吞服大量的古圣药，只怕都无法让这个级别的强者，提升一成的实力。
<br/>　　而这一枚渡厄圣叶，就足以做到让天人境死劫之力，削弱一成，这已经是无比逆天了。
<br/>　　所以对任何一位想要冲击天人境的强者来说，这渡厄圣叶都是绝对的至宝，没有不争夺的必要。
<br/>　　在如今这个时代之中，知晓渡厄圣叶的人物，却是不多了，不然的话，只怕是这围观的不少人，都要不顾一切的向着燕州大陆二人出手，来抢夺这渡厄圣叶。
<br/>　　这血虎五子，倒是认出了渡厄圣叶的价值，如今一路追杀燕州大陆二人，就是为了得到一枚渡厄圣叶。
<br/>　　就算是这渡厄圣叶对他们无用，一旦是卖出去，也是天价的财富，足以是让他们五人得到大量的资源，纷纷晋升到八劫古帝的地步。

<br/>　　看着逼近的血虎五子，那燕无峰的眼神之中越发的冰冷。
<br/>　　“燃烧天地法则，跟他们拼了！实在不行就自爆，绝不能让这些畜生得到一点的好处！”
<br/>　　这燕无峰的话中露出了决断之意。
<br/>　　“此子心智坚定。”
<br/>　　楚风眠远远的看到二人的交谈，也不由的点了点头，燕无峰这样的性格，才是成就一方强者的性格。
<br/>　　宁死不屈，立刻决断。

<br/>　　将这燕莫，燕无峰二人，围困在其中。

<br/>　　燕无峰冰冷的声音，几乎是从牙缝之中挤出来的。

<br/>　　那血虎五子之中，年纪最小的一名红袍人，突然一步上前，猛然一掌对着燕无峰轰击了过去。
