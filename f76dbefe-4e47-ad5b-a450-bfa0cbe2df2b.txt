　　一天，一月，一年，洪荒之中，岁月无声无息，飞速流逝。眨眼间，便已过数万年。
<br/>　　这一天，娲皇宫中。
<br/>　　“哥哥，这些年来，我并没有什么感悟，想要出去走走，这样或许可以找到一些感觉。”
<br/>　　女娲满脸苦涩地对着伏羲说道，她心中极为苦恼，这些年来努力修炼，通过鸿蒙紫气感悟圣人之道，却没有丝毫尽斩。眼见元始都已经成圣多年了，而她还在原地踏步，感觉很是沮丧。
<br/>　　但最近，仿佛有一股神秘的力量在驱使着她，让她赶紧晋升，这种感觉很玄妙，非常奇怪。
<br/>　　伏羲听后，心中也知道这些年来自己这个妹妹一直在努力，特别是元始成圣后，她更加刻苦了。想必是她那好胜心在作祟。不过，对此他也没有什么办法，谁让自己什么也不知道呢，他也不好劝说。最后只能对她说道：
<br/>　　“好吧，妹妹你可以出去，不过却是要小心一点，最近巫族在蠢蠢欲动，虽说还不会开战，但咱们终究是妖族，你说不定会遇到危险。”
<br/>　　“哥哥，你放心好了，我知道该怎么做，我又不是手无缚鸡之力的人，放心好了。”
<br/>　　女娲知道自己该怎么做，对着伏羲安慰道，叫他不要担心。虽然上一次巫妖大战之中，巫族中人死在她手中的不少，但不到逼不得已，她还是不想与巫族开战。
<br/>　　说服了哥哥伏羲后，女娲就走出娲皇宫，向着洪荒大地走去。一步一步地走在洪荒大地之上，看着各个种族在大战，在争斗，感悟自己的机缘。
<br/>　　走走停停，不知不觉中，一千年就这样过去了。这期间，女娲走过无数山水，见过无数种族的兴亡争斗，见过巫妖两族的大战，她遇到最多的就是争斗，洪荒之中，除了争斗，也没别的什么了。见多了，她也就心烦了。
<br/>　　不仅烦巫族，就连自己本族妖族她也有些心烦，洪荒为何要这么斗下去，难道就没有一个渴望和谐的种族吗？
<br/>　　不知不觉中，女娲再次回到不周山，这里，是她出生的地方，也算是重回故里了。看着不周山上那熟悉的环境，想起昔日还没有加入妖族前，与自己的哥哥伏羲那无忧无虑的生活，昔日的点点滴滴浮上心头，她感觉到一种淡淡温馨的感觉。
<br/>　　女娲站在一处山谷中，这山谷中，一潭清泉，倒映出她那绝美的容颜，女娲站在岸边，四周一片寂静，她心中也变得异常宁静，她想起了哥哥昔日抚琴时的场景，不由的痴痴笑了起来。
<br/>　　这时，她又想起巫妖两族之间的不断争斗，不断地杀戮，不断地死亡，洪荒之中，无尽苍生遭受战乱的痛苦。
<br/>　　“为什么他们一定要斗下去呢？有什么种族能够平平和和，不喜争斗呢。”水潭边，女娲似愁肠百结，眉头紧皱，她有些厌倦了打打杀杀的日子。
<br/>　　这一刻，女娲元神中的鸿蒙紫气前所未有的悸动，元神跳动，不断有感悟在她心中浮现。这些年游走洪荒的经历，变成了她内心深处的感悟。她站在池塘边，默默无语，静心感悟。
<br/>　　不知不觉中，时光流逝，六十年再次过去了。突然，女娲娇躯一震，全身上下散发出玄妙的造化神光，元神之中，鸿蒙紫气在跳动，一阵阵天地道韵涌上心头。这个时候，女娲似乎对自己的道有了一丝明悟。
<br/>　　“我的道，乃造化之道。造化，不仅能够造，也能化。造，有创造之意，化，有化去，毁灭之意。我之前只有化，而没有造，现在，是要我创造生命么？”水潭边，女娲低声呢喃，眼睛瞬间变得前所未有的明亮起来，犹如九天之上的星辰般耀眼，一时之间，万花失色，这一刻，就仿佛天地之间只剩下了她，她就是天地间最美的存在。
<br/>　　“洪荒之中没有和谐的种族，既然如此，那我便创造一个。”
<br/>　　女娲霸气的宣言。她元神之中，鸿蒙紫气在剧烈抖动，显然，这再一次证明了，她的圣路是对的。
<br/>　　看着水潭倒影中的女娲，她若有所思，捡起水潭边上的黄泥，用手按照着自己的模样捏了起来。
<br/>　　造化相生，玄妙无比，不一会儿，女娲便捏出一个与她一模一样的泥人，只见女娲随手打出一道造化之光，没入泥人身上，玄光绽放，泥人大涨，瞬间便变得与女娲一般大小，不过，只是持续瞬间，那泥人便散落一地，神光消散。
<br/>　　“怎么回事？还是不行吗，到底是哪里出错了？”女娲看着散落一地的泥人，眉头紧皱，暗暗思索道。
<br/>　　她不甘心，再一次蹲下，用泥巴捏起人来。不过，结局依旧不变，待泥人长大后，便散落一地，显然，这是不行的。
<br/>　　“怎么回事？我感觉自己的圣道就是如此，怎么会不行呢？”女娲眉头紧皱，娇俏的小脸之上，脸色变得有些难看，低声自语道。
<br/>　　“哈哈！何事竟让道友如此焦灼？”
<br/>　　就在此时，一阵爽朗的大笑声，惊醒了沉思中的女娲。
<br/>　　她心中一惊，连忙收敛心神，暗道自己大意了，若是别人在此刻偷袭她，那她能躲开吗？她心中暗恼，抬头看去，却见一个身穿黑袍的道人，踏步而来。
<br/>　　“道友乃何人，到此所为何事？”女娲暗自警惕，因为她发现，此人的修为一点也不比她弱，甚至她有点隐隐看不透的感觉。
<br/>　　“本尊是谁，并不重要，重要的是，本尊为道友而来？”黑衣道人肆无忌惮地打量着女娲，脸上突然一笑道。
<br/>　　女娲眉头一皱，有些不悦，但她并没有发作，而是反问道：“为我而来，道友此言何意？”
<br/>　　“若是本尊没有看错的话，道友如今正处在悟道的关键时刻，但道友又进入不了那种状态，想必是缺少某种关键的东西吧。”黑衣道人满脸笑意的看着女娲。
<br/>　　“你说得不错。但，那又如何？”女娲有些不喜道。
<br/>　　“呵呵，那有如何？不错，不错，道友还真是……有性格。本尊是不能如何，但正好本尊知道道友你现在正缺少的东西。”黑衣道人呵呵一笑道。
<br/>　　女娲突然安静下来，沉默了片刻后，说道：
<br/>　　“虽然我并不知道你是谁，但看你一身修为，想必也不是什么无名之辈吧。说吧，你想要什么？”
<br/>　　“我想要什么？呵呵！”
<br/>　　黑衣道人呵呵一笑，反问道：“你有什么值得我在意的？”
<br/>　　“你——”
<br/>　　女娲有些生气，但紧接着，就平缓了自己的心情，看了一眼黑衣道人，转身就走。
<br/>　　呃？这情况有些不对呀，接下来不是要留在此次继续讨价还价么？自己最后心情一好，就告诉她，她所缺少的东西。但如今怎么一言不发地就走了？
<br/>　　黑衣道人有些懵了。不过，到底是得道高人，心情转瞬间便调整过来。他的声音幽幽的传到远去的女娲耳边，让她身体一滞：
<br/>　　“道友，难道你不想成圣吗？你难道还想看着三清一个个在你之前成圣，将你压在身下？”
<br/>(本章完)
